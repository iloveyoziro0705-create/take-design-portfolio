## 使い方

LP制作を開始する前に、このチェックリストを業種に合わせて使用する。
「必要素材一覧」を確認し、調達方法を確定してからコーディングに入ること。

## Phase 2: 素材受取後チェック

クライアントから素材を受け取った後に確認する。

```markdown
### 受取素材チェック

#### 解像度・品質
- [ ] 最低解像度クリア（幅2000px以上推奨）
- [ ] JPEG品質が低くないか（ブロックノイズなし）
- [ ] ピンボケ・手ブレがないか

#### ライセンス・権利
- [ ] 人物写真: 肖像権・使用許諾が確認済みか
- [ ] ロゴ: 使用権限があるか（クライアント自社ロゴ以外）
- [ ] フリー素材: ライセンス確認済みか

#### 内容確認
- [ ] 薬機法・景表法に抵触する表現が含まれていないか
- [ ] 個人情報が写り込んでいないか（名刺・書類等）
- [ ] 競合他社のロゴ・商標が写り込んでいないか
- [ ] 古い情報（閉店した店名・旧ロゴ等）ではないか
```

## Phase 3: コーディング後チェック

```markdown
### 実装後 素材チェック

#### 表示品質
- [ ] 画像が正しく表示されているか（壊れていない）
- [ ] モバイルで画像が切れていないか
- [ ] 画像の読み込み速度が遅くないか（WebP変換推奨）
- [ ] alt属性が適切に設定されているか

#### デザイン整合性
- [ ] ブランドカラーと画像の色調が合っているか
- [ ] 全ページで画像スタイルが統一されているか
- [ ] 写真とイラストが同一セクション内で混在していないか

#### OGP確認
- [ ] OGP画像が1200×630pxになっているか
- [ ] OGPタイトル・ディスクリプションが設定されているか
- [ ] SNSシェア時に正しく表示されるか（OGPテスター確認）
```

## 素材ファイル命名規則

```
[案件名]/images/
├── fv-main.jpg          ← FVメイン画像
├── fv-sub.jpg           ← FVサブ画像（ある場合）
├── service-01.jpg       ← サービス画像（番号付き）
├── service-02.jpg
├── staff-01.jpg         ← スタッフ写真
├── staff-02.jpg
├── review-01.jpg        ← 口コミ者写真（使用許諾必須）
├── before.jpg           ← ビフォー（美容・医療のみ）
├── after.jpg            ← アフター（美容・医療のみ）
├── cta-bg.jpg           ← CTA背景（使用する場合）
└── ogp.jpg              ← OGP画像（1200×630px）
```

# Asset Checklist — 素材チェックリスト

**バージョン**: 1.0
**作成日**: 2026-06-27
**管理ファイル**: knowledge/assets/asset-checklist.md

---

## このファイルの目的

LP制作時に必要な素材を漏れなく収集・確認するためのチェックリスト。
制作開始前・コーディング前・納品前の3段階で使用する。

---

## 使い方

1. **制作開始前**: 「収集計画チェック」でクライアントへの依頼リストを作成
2. **コーディング前**: 「素材準備チェック」で全素材の準備完了を確認
3. **納品前**: 「品質チェック」で素材の品質・ライセンスを最終確認

---

## Phase 1: 収集計画チェック（制作開始前）

### 基本確認

- [ ] `industry-asset-patterns.md` で業種別必要素材一覧を確認した
- [ ] クライアントに提供を依頼する素材リストを作成した
- [ ] フリー素材で補完する素材を特定した
- [ ] ChatGPT生成が必要な素材を特定した（`chatgpt-image-prompt-library.md` 参照）
- [ ] アイコンセットを選定した（デフォルト: Lucide Icons）

---

## Phase 2: 素材準備チェック（コーディング前）

### FV（ファーストビュー）画像

- [ ] FV背景画像が準備できている
- [ ] 画像サイズが適切か（最低 1920×1080px 推奨）
- [ ] 横長・縦長・正方形など、FVレイアウトに合ったアスペクト比か
- [ ] テキストを重ねる場合、文字が読めるトーン・コントラストか
- [ ] スマホ表示時の見え方を確認した

### サービス・メニュー画像

- [ ] 各サービス・メニューの画像が揃っている
- [ ] 複数画像がある場合、トーン・スタイルが統一されている
- [ ] カードレイアウトに合ったサイズか（推奨: 600×400px〜）

### 実績・事例画像

- [ ] 実績紹介用の画像が準備できている（スクリーンショット・完成物・Before/After等）
- [ ] 使用許可を得た画像のみ使用している

### 口コミ・お客様の声 画像

- [ ] 顔写真がある場合は使用許可を確認した
- [ ] 顔写真がない場合はアイコン代替を設定した
- [ ] アイコン代替の場合は Lucide の `user-circle` 等を使用

### CTA画像・背景

- [ ] CTAセクションの背景画像またはグラデーションが準備できている
- [ ] CTAテキストとのコントラストが十分か（WCAG AA 基準: 4.5:1以上）

### 背景画像・テクスチャ

- [ ] セクション背景に使う画像・テクスチャが準備できている
- [ ] 繰り返し使用する背景はシームレスタイルになっているか

### アイコン

- [ ] 使用するアイコンセットを1種類に統一した（原則: Lucide Icons）
- [ ] 特徴・メリット紹介に使うアイコンが選定できている（6〜10個）
- [ ] ステップ・フロー説明のアイコンが選定できている
- [ ] 連絡先アイコン（電話・メール・LINE）が選定できている
- [ ] SNSアイコンが必要な場合は準備できている（Font Awesome推奨）

### OGP画像

- [ ] OGP画像が準備できている
- [ ] サイズが 1200×630px か確認した
- [ ] LPのタイトル・サービス名が含まれているか
- [ ] ブランドカラーに合っているか

### イラスト素材

- [ ] イラストが必要な場合（スクール・SaaS等）、サイトを1種類に統一した
- [ ] イラストの色をブランドカラーに合わせた（SVGの場合）
- [ ] 各セクションで1枚の原則を守っている

---

## Phase 3: 品質チェック（納品前）

### ライセンス確認

- [ ] すべての画像のライセンスを確認した
- [ ] 商用利用OKのもののみ使用している
- [ ] 帰属表示が必要な素材は表示を設定した
- [ ] クライアント提供素材の使用権利を確認した

### 画質・表示確認

- [ ] すべての画像がぼやけていない（適切な解像度）
- [ ] PC表示で画像が荒れていない
- [ ] スマホ表示で画像が正しく表示されている
- [ ] Retina/高解像度ディスプレイでの表示を確認した（2x対応推奨）

### パフォーマンス確認

- [ ] 各画像のファイルサイズが適切か
  - FV画像: 200KB以下推奨
  - その他: 100KB以下推奨
- [ ] 必要に応じてWebP形式に変換した
- [ ] `loading="lazy"` 属性を FV 以外の画像に設定した
- [ ] FV画像に `loading="eager"` または設定なし（優先読み込み）

### アクセシビリティ確認

- [ ] すべての `<img>` タグに `alt` 属性を設定した
- [ ] 装飾目的の画像は `alt=""` を設定した
- [ ] テキストと背景のコントラスト比が基準を満たしている
- [ ] アイコンのみのボタンに `aria-label` を設定した

## 素材依頼テンプレート（クライアント向け）

制作開始時にクライアントへ送るメッセージのテンプレート：

```
【LP制作 素材ご依頼リスト】

制作をスムーズに進めるため、以下の素材をご提供いただけますか？

■ 最優先（必須）
・トップページ用の写真（横長・1920px以上推奨）
  → [具体的な内容: 施術風景・スタッフ・商品など]
・スタッフ・代表者の写真
・[業種固有の必須素材]

■ あれば使いたい（なければフリー素材で対応）
・店内・オフィスの写真
・サービスの様子がわかる写真
・お客様・受講生の写真（使用許可済みのもの）

■ 会社ロゴ・ブランド素材
・ロゴデータ（PNG透過 または SVG 形式）
・ブランドカラーコード（わかれば）

■ 補足
・お持ちの素材はなんでもお送りください。
・スマートフォンで撮影した写真でも構いません。
・写真が準備できない場合はフリー素材で代用します。
```

## 素材が揃わない場合の対処

| 不足素材 | 対処方法 |
|---------|---------|
| FV画像なし | フリー素材 → ChatGPT生成 → プレースホルダーで制作開始 |
| スタッフ写真なし | アイコン代替（Lucide: `user-circle`）で対処 |
| Before/Afterなし | テキストのみの実績紹介に変更 |
| ロゴなし | テキストロゴで代替・後日差し替え前提 |
| 料理写真なし（飲食） | フリー素材を使用するが、納品後の差し替えを強く推奨 |
| 店内写真なし | フリー素材または地図・アクセスのみで対処 |

---

## Learning Reason

**Why**: 素材の漏れや品質不足が納品直前に発覚して手戻りが発生するパターンが多いため、制作の3段階でチェックすることで手戻りをゼロにする。
**応用先**: すべてのLP・HP・WordPress制作の素材管理フェーズ

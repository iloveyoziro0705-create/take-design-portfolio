# Illustration Source Library — イラスト素材ライブラリ

**バージョン**: 1.0
**作成日**: 2026-06-27
**管理ファイル**: knowledge/assets/illustration-source-library.md

---

## このファイルの目的

LP・ホームページ・WordPress制作時に適切なイラスト素材を選定できる状態を作る。
写真素材が使いにくい業種・シーンでのイラスト活用を標準化する。

---

## 基本ルール

- Claude Codeはイラスト生成を行わない
- イラスト生成が必要な場合はChatGPTのみ使用する（`chatgpt-image-prompt-library.md` 参照）
- SVGイラストは色変更・サイズ変更が可能なため積極的に活用する
- クライアント指定のブランドカラーに合わせて色変更して使用する

## 素材サイト一覧

### ILLUST-SRC-001 unDraw

**URL**: https://undraw.co/illustrations
**ライセンス**: 商用利用可・帰属表示不要・フォーク・改変可
**特徴**:
- SVG形式で色変更が自由自在
- サイトでカラーを指定してダウンロード可能
- BtoB・SaaS・テック系に最適
- シンプルでモダンなスタイル

**推奨業種**: SaaS・BtoB・スクール・教育・スタートアップ

**利用シーン別推奨イラスト**:

| 利用シーン | 推奨キーワード | 用途 |
|----------|-------------|------|
| データ分析・ダッシュボード | `analytics` `data` `dashboard` | SaaS FV・機能紹介 |
| チームワーク・コラボ | `team` `collaboration` `meeting` | 採用・コーポレート |
| 学習・教育 | `learning` `teaching` `studying` | スクール・教育 |
| セキュリティ・安心 | `security` `protection` `privacy` | SaaS・医療の信頼セクション |
| コミュニケーション | `messaging` `chat` `notification` | SaaS機能紹介 |
| 成功・達成 | `success` `achievement` `winner` | 実績・CV前セクション |

**注意点**:
- 全体的に海外テイスト（人物が白人・黒人モデル）
- BtoCの感情訴求には向きにくい
- 複数イラストを使う場合は同一サイトで統一して世界観を揃える

## イラスト使用時の注意点

1. **世界観の統一**: 同一ページで複数のイラストを使う場合、必ず同じサイト・同じスタイルパックで統一する
2. **色の統一**: SVGイラストはブランドカラーに合わせて色変更する（特にunDraw・ManyPixels）
3. **使いすぎ禁止**: イラストが多すぎると情報が埋もれる。1セクション1枚を基本とする
4. **写真との混在**: 写真とイラストを同一ページで混在させる場合は明確なゾーニングを行う
5. **サイズ統一**: 横並びのイラストはサイズ・余白を統一する

## Learning Reason

**Why**: 写真素材が使えない・使いにくい業種やシーンでのイラスト素材活用を標準化するため。業種別のイラスト選定基準を持つことで、デザイン品質と制作速度を向上させる。
**応用先**: 写真NGの業種（スタートアップ・教育・採用）・ステップ説明セクション・サービス概念説明

## 概要

LP・ホームページ・WordPress制作で使用するイラスト素材の選定基準と推奨サイト・利用ガイドラインを統一管理する。

**使用判断フロー:**
```
写真素材でリアリティを出したい → photo-source-library.md
抽象的・説明的な表現をしたい → illustration-source-library.md（本ファイル）
独自世界観が必要 → chatgpt-image-prompt-library.md（ChatGPT生成）
```

## ILLUS-SRC-001: unDraw

**URL:** https://undraw.co  
**用途:** SaaS / BtoB / スクール / 教育系LP・HP  
**特徴:** カラー変更可能（ブランドカラー一括変更）/ SVG商用利用可 / 帰属不要

### 利用業種・シーン

| 業種 | 推奨シーン | 検索ワード例 |
|------|----------|------------|
| SaaS | 機能説明・ダッシュボード紹介 | "dashboard" / "analytics" / "data" |
| スクール | 学習シーン・オンライン授業 | "online learning" / "education" / "studying" |
| 採用 | チームワーク・仕事の様子 | "teamwork" / "collaboration" / "office" |
| コーポレート | サービス説明・フロー図 | "business" / "success" / "process" |
| 医療 | ヘルスケア・アプリ説明 | "health" / "medicine" / "healthcare" |

### 推奨構図・配置

```
推奨構図:
- セクション左にイラスト / 右にテキスト（横並び）
- テキスト主体セクションのアクセント（右下に小さく）
- 機能紹介の各カードに1個ずつ

推奨サイズ:
- FV補助: 400〜600px幅
- セクション画像: 500〜700px幅
- カードアイコン: 80〜120px

注意点:
- カラー変更機能を使いブランドカラーに統一する
- 同じページで複数使う場合はスタイルを統一する
- 写真と混在させると世界観が崩れる（一方に統一）
```

### Learning Reason

業種別に検索ワードを整理することで、制作時の素材選定時間を削減。特にSaaS・スクール系LPはフォトよりイラストの方が訴求効果が高いケースが多い。

## ILLUS-SRC-002: Storyset

**URL:** https://storyset.com  
**用途:** SaaS / スタートアップ / 教育 / コーポレート  
**特徴:** アニメーション対応（GIF/SVG）/ 高品質 / カラー変更可 / 帰属表記必要（無料プラン）

### 利用業種・シーン

| 業種 | 推奨シーン | カテゴリ例 |
|------|----------|----------|
| SaaS | FVビジュアル・機能紹介 | Technology / Business |
| スタートアップ | サービス説明・ストーリー | Startup / Innovation |
| 教育 | 学習・成長ストーリー | Education / Learning |
| 採用 | チーム・成長・キャリア | Business / People |

### 推奨構図・配置

```
アニメーション活用:
- FVのメイン画像（動きで目を引く）
- スクロールトリガーで動き出す（AOS.js連携）

静止イラスト活用:
- セクション区切りのビジュアル
- 特徴紹介カードのアクセント

注意点:
- 無料プランは帰属表記必要（<a href="https://storyset.com">Storyset</a>）
- Paidプランで帰属不要
- アニメ版はファイルサイズに注意（遅延読込推奨）
```

## ILLUS-SRC-004: DrawKit

**URL:** https://www.drawkit.com  
**用途:** 高品質LP / スタートアップ / 高級感が必要な制作物  
**特徴:** デザイン品質が高い / アニメーション対応（一部）/ 商用利用可（要利用規約確認）

### 利用業種・シーン

| 業種 | 推奨シーン | 注意点 |
|------|----------|--------|
| スタートアップ | FVメインビジュアル | 有料素材あり・要確認 |
| コーポレート | 上質なサービス説明 | ライセンス種別確認 |
| SaaS | 機能紹介・成長グラフ | スタイル統一 |
| 美容（モダン系） | ライフスタイル訴求 | 抽象的な世界観に限る |

### 注意点

```
- 無料素材と有料素材が混在（ダウンロード前に確認）
- 商用利用条件を各素材ページで確認
- スタイル（色・タッチ）の統一感を重視
- 1LP内で複数スタイルを混在させない
```

## イラスト選定ルール

### 業種×スタイル 推奨マトリクス

| 業種 | 推奨スタイル | 推奨サイト | NG |
|------|------------|----------|-----|
| 美容 | 原則使わない（写真優先） | — | イラストはブランド毀損リスク |
| 採用 | フラット・人物系 | unDraw / ManyPixels | リアリティを損なうもの |
| 医療 | アウトライン・クリーン | unDraw | かわいすぎる・ポップ |
| 飲食 | 原則使わない（写真優先） | — | 料理はリアル写真必須 |
| スクール | フラット・教育系 | unDraw / Storyset | 子供向けすぎる |
| SaaS | アイソメトリック・フラット | ManyPixels / Storyset | ビジネス感のないもの |
| コーポレート | シンプル・クリーン | unDraw / DrawKit | カジュアルすぎる |

### 写真との混在禁止ルール

```
同一セクション内での写真とイラストの混在は原則禁止。

OK: FVは写真 / 特徴説明セクションはイラスト（セクション単位で分ける）
NG: 同じカード内に写真とイラストを並べる
NG: FVの写真の上にイラストを重ねる（アイコン除く）
```

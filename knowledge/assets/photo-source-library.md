## 概要

LP・ホームページ・WordPress制作で使用する写真素材の選定基準と推奨サイト・検索ワードを統一管理する。

**原則:** フリー素材 → クライアント提供 → ChatGPT生成 の優先順位で調達する。

---

## サイト別特性マップ

| サイト | 強み | 日本人モデル | 無料範囲 | 推奨業種 |
|--------|------|------------|---------|---------|
| Unsplash | 高品質・自然・雰囲気 | 少ない | 全て無料 | 美容・飲食・コーポレート |
| Pexels | 人物豊富・多様性 | 普通 | 全て無料 | 採用・医療・スクール |
| Pixabay | 素材数最多・汎用 | 少ない | 全て無料 | 背景・テクスチャ・汎用 |
| O-DAN | 複数サイト横断検索 | — | 各サイト依存 | 全業種の横断検索に |
| Burst (Shopify) | EC・商品・店舗 | 少ない | 全て無料 | 飲食・物販 |
| StockSnap | ビジネス系充実 | 少ない | 全て無料 | SaaS・コーポレート |

## PHOTO-SRC-001: Unsplash

**URL:** https://unsplash.com  
**用途:** 美容 / 飲食 / コーポレート / 店舗 / ライフスタイル  
**特徴:** 高品質・自然な雰囲気・商用利用可・帰属表記不要

### 利用業種別 推奨検索ワード

#### 美容サロン
```
英語検索（推奨）:
- "skincare woman natural light"
- "beauty salon treatment"
- "glowing skin portrait"
- "facial massage spa"
- "clean beauty minimal"
- "woman touching face"

日本語検索:
- スキンケア
- 美容 女性
- 自然光 ポートレート
```

#### 飲食・レストラン
```
英語検索（推奨）:
- "restaurant food photography"
- "japanese food plating"
- "cafe interior warm light"
- "chef cooking kitchen"
- "food flat lay overhead"
- "fine dining plate"
```

#### コーポレート・店舗
```
英語検索（推奨）:
- "modern office workspace"
- "business meeting professional"
- "minimal interior design"
- "retail store clean"
```

### 推奨利用シーン

| シーン | 検索ワード例 | 注意点 |
|--------|------------|--------|
| FV背景 | "minimal background texture" | 人物なしで使う場合はテクスチャ系 |
| コンセプト画像 | 業種別ワード（上記） | 日本人モデルが少ないため業種に注意 |
| 環境・空間 | "interior design" + 業種 | 国内感が出にくい場合あり |
| 背景ぼかし | "bokeh background" | FVのサブ素材として活用 |

## PHOTO-SRC-003: Pixabay

**URL:** https://pixabay.com  
**用途:** 汎用 / 背景 / テクスチャ / アイコン素材  
**特徴:** 写真数が最多・背景・テクスチャに強い・商用利用可

### 推奨検索ワード

#### 背景・テクスチャ
```
- "white background minimal"
- "marble texture"
- "wood texture"
- "gradient background"
- "abstract background"
- "paper texture"
- "fabric texture"
```

#### 汎用人物（補助的に）
```
- 業種名 + "japan"（日本語環境なら日本語も有効）
- "person" + シーン
```

### 推奨利用シーン

| シーン | 検索ワード例 | 注意点 |
|--------|------------|--------|
| セクション背景 | "texture background" + 色 | 画質チェック必須 |
| OGP背景 | "minimal background" | シンプル系が使いやすい |
| イラスト素材 | Illustrations タブ | ベクター素材も豊富 |

## PHOTO-SRC-004: O-DAN（横断検索）

**URL:** https://o-dan.net  
**用途:** 複数フォトサイト横断検索  
**特徴:** Unsplash・Pexels・Pixabay等を同時検索可能・日本語検索対応

### 使い方

```
1. O-DANでキーワード検索（日本語でもOK）
2. 各サイトの結果から最適画像を選定
3. 元サイトでダウンロード・ライセンス確認
```

| 採用 | オフィス / チーム / ビジネス / 求人 |
| 医療 | 医師 / クリニック / 健康 / 医療 |
| 飲食 | 料理 / レストラン / カフェ / グルメ |

## ライセンス確認チェックリスト

制作前に必ず確認すること。

- [ ] 商用利用が許可されているか
- [ ] 帰属表記（クレジット）が必要か
- [ ] 人物の肖像権が含まれる場合、モデルリリース取得済みか
- [ ] ロゴ・商標が写り込んでいないか
- [ ] 医療・美容系で「効果を保証する」ような表現として使えないか（薬機法）
- [ ] 利用規約の変更がないか（特にBeforeAfter画像）

## Learning Reason

### 保存理由
業種別フォト素材の検索効率を上げるため、推奨サイト・検索ワードを統一管理する。

### 再利用価値
全業種・全制作物（LP/HP/WP）で参照可能。

### 特に優れている点
サイト特性を業種別に整理することで、検索時間を削減しつつ品質を担保できる。

### 他案件への応用先
新業種対応時は本ファイルに検索ワードを追記して継続育成する。

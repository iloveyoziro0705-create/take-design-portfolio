# 業種別LP辞典

## 概要
業種ごとの最適LP構成・配色・心理学・CVポイントを体系化する。
「〇〇業種ならこの構成」を即座に提案できる状態を目指す。

**現在のステータス**: Phase 4（LP辞典150パターン到達後に本格育成）

---

## 業種管理ルール

- 業種ID: `IND-XXX`
- 各業種に「推奨パターン」「禁止パターン」を記録
- 実案件からの更新を優先

### Learning Reason（各業種パターン追加時に必須記録）

新規業種パターンを追加する際は、パターン末尾に必ず以下のセクションを記録すること。

```markdown
## Learning Reason
### 保存理由
[なぜこの業種パターンを独立して登録したか（業種特有の設計要件・発見経緯）]
### 再利用価値
[この業種パターンが有効な案件の範囲・規模]
### 特に優れている点
[業種固有のCV構造・ターゲット心理・禁止パターン回避の観点で優れている点]
### 他案件への応用先
[類似業種・周辺業種への展開可能性]
```

## IND-001: 美容・サロン系

**カテゴリ**: 美容院・ネイルサロン・エステ・脱毛・まつ毛

### 共通特性
- 意思決定者: 女性30〜50代（主力）
- 意思決定期間: 短〜中（1日〜1週間）
- 競合差別化軸: 技術力・雰囲気・価格・アクセス
- 主なCVアクション: LINE予約・電話予約・ネット予約

### 推奨構成（Pattern-LP-001参照）
```
FV（高級感） → 実績数字 → 特徴3点 → サービス → 
口コミ → Before/After → スタッフ → アクセス → FAQ → CTA
```

### 推奨カラーパレット
| ブランドポジション | カラー |
|----------------|--------|
| 高級・ラグジュアリー | `#faf7f3` + `#c8a56a` + `#3b2f2a` |
| ナチュラル・癒し | `#f9f5f0` + `#b5936c` + `#3d2b1f` |
| クール・モダン | `#f5f5f7` + `#8b7355` + `#1a1a1a` |
| ポップ・若者向け | `#fff0f3` + `#ff6b8a` + `#2d1b2e` |

### 禁止パターン
- ダークテーマ（信頼感・清潔感を損なう）
- テキスト過多のFV
- 料金表の曖昧表示

### CV向上ポイント
1. LINE予約ボタンをFVに必ず配置
2. Before/Afterは"劇的な変化"を数枚用意
3. スタッフ写真は笑顔・白衣or制服着用
4. 口コミは「具体的な数字+地域+年代」付き
5. 固定フッターにLINEボタン（スマホ）

## IND-002: 飲食・レストラン系

**カテゴリ**: レストラン・カフェ・居酒屋・ラーメン・寿司・ベーカリー

### 共通特性
- 意思決定者: 全年齢層
- 意思決定期間: 超短（当日〜数日）
- 競合差別化軸: 料理の質・雰囲気・価格帯・アクセス
- 主なCVアクション: 電話予約・ネット予約・来店

### 推奨構成（Pattern-LP-002参照）
```
FV（料理ビジュアル） → コンセプト → 人気メニュー → 
空間・雰囲気 → アクセス → 予約CTA
```

### 視覚的優先順位
1. 料理写真の品質（最重要）
2. 空間・照明の雰囲気
3. シェフ/スタッフの顔

### 禁止パターン
- フリー素材の料理写真（信頼性破壊）
- テキスト優位のFV
- 予約導線が分かりにくい構成

## IND-003: 医療・クリニック系

**カテゴリ**: 歯科・整形外科・皮膚科・美容クリニック・整骨院

### 共通特性
- 意思決定者: 全年齢（症状による）
- 意思決定期間: 短（痛みが強いほど即決）
- 競合差別化軸: 技術・痛みの少なさ・アクセス・待ち時間
- 主なCVアクション: ネット予約・電話予約

### 推奨構成
```
FV（安心感訴求） → こんな症状に → 当院の特徴 → 
治療の流れ → 実績・資格 → 患者の声 → 料金 → アクセス → 予約CTA
```

### 配色
- 歯科: `#f0f7ff` + `#2563eb` + `#1e3a5f`
- 整形・整骨: `#f0fff4` + `#16a34a` + `#14532d`
- 美容クリニック: `#faf7f3` + `#c8a56a` + `#3b2f2a`

### 法規制注意事項（必須）
- 効果の過剰表現禁止（医療広告ガイドライン）
- 「治る」「完治」等の表現不可
- 症例写真の使用条件（患者同意書必須）
- 費用の明示義務

## IND-004: 採用・求人系

**カテゴリ**: 保育士・介護士・看護師・建設・製造・IT

### 共通特性
- 意思決定者: 求職者（転職意向あり）
- 意思決定期間: 中〜長（1週間〜1ヶ月）
- 競合差別化軸: 給与・働き方・職場環境・成長機会
- 主なCVアクション: 応募フォーム・見学申込・説明会参加

### 推奨構成（Pattern-LP-003参照）
```
FV（悩み共感） → 求職者の悩み → 選ばれる理由 → 
待遇・給与 → 先輩の声 → 1日の流れ → 施設紹介 → 
応募の流れ → 応募CTA
```

### ターゲット別コピー差異
| ターゲット | FVコピー方向性 |
|-----------|-------------|
| 現役だが職場に不満 | 「今の職場のままでいいですか？」 |
| ブランク明けの復職 | 「久しぶりの復職、不安なあなたへ」 |
| 新卒・未経験 | 「未経験でも大丈夫。一から教えます」 |
| 経験者スカウト | 「あなたの経験を活かせる場所があります」 |

### 禁止パターン
- 給与の非表示（求職者は必ず確認する）
- 施設写真がない構成
- 応募フォームの項目数過多（5項目以上で離脱増加）

## IND-005: 金融・資産運用系

**カテゴリ**: 投資・保険・不動産投資・FP相談・節税

### 共通特性
- 意思決定者: 30〜50代 会社員・経営者
- 意思決定期間: 長（1週間〜1ヶ月以上）
- 競合差別化軸: 実績・信頼性・担当者の質
- 主なCVアクション: 無料相談予約・セミナー申込

### 推奨構成（Pattern-LP-004参照）
```
FV（数字インパクト） → ターゲット条件 → サービス特徴 → 
実績数字 → 信頼性証明 → 顧客の声 → 相談の流れ → CTA
```

### 法規制注意事項（必須）
- 金融庁登録番号の表示
- リスク説明の表示義務
- 「必ず儲かる」等の表現禁止
- 顧客の声は実際の顧客から取得（捏造禁止）

## IND-006: 教育・スクール系

**カテゴリ**: プログラミングスクール・英会話・資格・オンライン講座

### 共通特性
- 意思決定者: 20〜40代 自己成長意欲高い
- 意思決定期間: 中（数日〜2週間）
- 競合差別化軸: 卒業生の実績・カリキュラムの質・価格
- 主なCVアクション: 無料体験・説明会申込

### 推奨構成（Pattern-LP-005参照）
```
FV（成果訴求） → 悩み解決 → 得られること → カリキュラム → 
卒業生の声 → 講師紹介 → 料金プラン → FAQ → 無料体験CTA
```

## IND-007: 不動産系

**カテゴリ**: 賃貸・売買・リノベーション・建売・注文住宅

### 共通特性
- 意思決定者: 20〜50代（ライフステージ変化時）
- 意思決定期間: 超長（1ヶ月〜1年）
- 競合差別化軸: 物件の質・担当者の信頼感・アフターサービス
- 主なCVアクション: 物件問い合わせ・来場予約

### 推奨構成
```
FV（物件ビジュアル） → 物件一覧 → 特徴・こだわり → 
施工事例 → お客様の声 → スタッフ紹介 → 会社概要 → 問い合わせCTA
```

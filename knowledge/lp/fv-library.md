# ファーストビュー（FV）辞典

## 概要
LP制作時に即座に参照・適用できるFVパターンライブラリ。
Phase 2（50パターン到達後）から育成開始。

**現在のステータス**: Phase 1完了待ち（現在 5/50 パターン）

---

## FVパターン管理ルール

- パターンID: `FV-XXX`（3桁ゼロ埋め）
- 構成要素を必ず記録する
- 適用業種を明記する

### Learning Reason（各FVパターン追加時に必須記録）

新規パターンを追加する際は、パターン末尾に必ず以下のセクションを記録すること。

```markdown
## Learning Reason
### 保存理由
[なぜこのFVパターンをライブラリに追加したか（CVへの効果・発見の経緯）]
### 再利用価値
[適用できるFV場面の頻度・業種の幅広さ]
### 特に優れている点
[視線誘導・ベネフィット伝達・信頼要素配置の観点で優れている点]
### 他案件への応用先
[適用推奨の業種・LP目的・ビジュアル方針]
```

## FV-001: フルスクリーン背景画像型

**適用業種**: レストラン・ホテル・美容・旅行・不動産  
**視覚的効果**: 没入感・高級感・ブランド訴求

### 構成要素
```html
<!-- 構造 -->
<section class="hero" style="
  min-height: 100vh;
  background: linear-gradient(rgba(0,0,0,.45), rgba(0,0,0,.7)), url('hero.jpg') center/cover;
  display: flex;
  align-items: center;
">
  <!-- ラベル -->
  <span class="label">英字キャッチ / カテゴリ名</span>
  <!-- H1 -->
  <h1>メインキャッチコピー<br>（2行以内）</h1>
  <!-- サブコピー -->
  <p>ベネフィットを一文で</p>
  <!-- CTA -->
  <a href="#" class="btn">行動喚起テキスト →</a>
</section>
```

### デザイン仕様
- オーバーレイ: `rgba(0,0,0, 0.4〜0.65)`（写真の明度による）
- H1サイズ: `clamp(40px, 7vw, 88px)`
- CTA配置: H1から40px下
- ラベル色: アクセントカラー（金・ブランドカラー）

### 注意点
- 画像なし時はグラデーション背景で代替
- モバイルでは画像の中心部にフォーカスを当てる
- テキスト可読性確保のためオーバーレイ必須

## FV-003: テキスト中心型（ビジュアルレス）

**適用業種**: コンサル・士業・採用・B2B  
**視覚的効果**: 誠実さ・情報の明確さ・信頼性

### 構成要素
```
レイアウト:
  左カラム（60%）: コピー + CTA
  右カラム（40%）: 数字実績 or イラスト

コピー階層:
  ① 悩みラベル（12px 追跡あり）
  ② H1 キャッチ（48〜72px）
  ③ サブコピー（17〜18px 行間1.8）
  ④ CTAボタン
  ⑤ 安心訴求（「無料」「〇秒で申込」等）
```

## FV-004: 分割レイアウト型（左テキスト / 右画像）

**適用業種**: 美容・サロン・クリニック・物販  
**視覚的効果**: バランス・視線誘導・情報整理

### 構成要素
```
<div style="display: grid; grid-template-columns: 1fr 1fr; min-height: 100vh;">
  <div class="hero-text">  ← 左: コピー + CTA </div>
  <div class="hero-image"> ← 右: 画像（object-fit: cover）</div>
</div>
```

### モバイル対応
- 縦並び: 画像を上、テキストを下
- 画像高さ: `min(50vh, 400px)`

## FV-005: 動画背景型

**適用業種**: ブランド訴求・観光・イベント・高級サービス  
**視覚的効果**: 最高の没入感・プレミアム感

### 構成要素
```html
<video autoplay muted loop playsinline
  style="position:absolute; inset:0; width:100%; height:100%; object-fit:cover; z-index:-1;">
  <source src="hero.mp4" type="video/mp4">
</video>
```

### 注意点
- ファイルサイズ: 5MB以下推奨（WebM形式）
- 必ずmuted + autoplay属性
- Fallback画像を必ず設定
- 画像重視案件に限定使用

## FV-008: 縦書きFVコピー型（和の品格・老舗系）

**適用業種**: 美容・日本料理・旅館・茶道・和雑貨・伝統工芸  
**視覚的効果**: 和の品格・静けさ・上質感・温かみ

### 構成要素
```
レイアウト:
  背景: 施術シーン写真 or 和の空間写真（オーバーレイ薄め 0.2〜0.3）
  コピー: CSS writing-mode: vertical-rl で縦組み
  配置: 右上 or 中央右寄り

コピー階層:
  ① 縦書きメインコピー（16〜20文字・1〜2行）
  ② 横書きサブコピー（小文字・行間広め）
  ③ CTAボタン（左下）
```

### 縦書きCSS実装
```css
.hero-copy {
  writing-mode: vertical-rl;
  text-orientation: mixed;
  font-size: clamp(22px, 3.5vw, 42px);
  font-family: 'Noto Serif JP', serif;
  letter-spacing: .15em;
  line-height: 1.8;
  color: #fff;
}
/* 数字・英字を横に戻す場合 */
.hero-copy .tcy {
  text-combine-upright: all; /* -webkit-text-combine: horizontal; */
}
```

### コピー例
```
「毎日のために
  心地のよいスタイルを」（Hériter — B/70）

「お客様を想う心が
  身に付く仕事。」（吉帆楼採用LP）
```

### 注意点
- モバイルでは縦書きをやめて通常横書きに切り替えを推奨
- 縦書きの場合は右から左に読む日本語のリズムを意識する
- フランス語・英語ブランド名と縦書きの組み合わせで「洋×和」の独自世界観が生まれる

## FV-009: 英語哲学型FVコピー（全大文字・ミニマリスト）

**適用業種**: 美容・アパレル・フィットネス・ライフスタイル・インテリア  
**視覚的効果**: クール・スタイリッシュ・哲学的差別化・セルフスクリーニング

### 構成要素
```
レイアウト:
  背景: 施術シーン or モデル写真（ダークオーバーレイ 0.4〜0.5）
  コピー: 全大文字英語 + 日本語補足

コピー構造:
  ① 英語哲学コピー（全大文字 / 3〜5単語）← H1
  ② ピリオドで完結させる
  ③ 日本語補足コピー（18〜24px・細め）← p
  ④ CTAボタン（白 or ブランドカラー）
```

### コピー例
```
「BE YOURSELF.」 — MADA A/80
「FOR ALWAYS.」  — MADA
「STAY NATURAL.」
「FEEL THE CHANGE.」
「FIND YOUR STYLE.」
```

### スタイル実装

### セルフスクリーニング効果
- 業種説明・悩み共感型コピーと異なり「この世界観に共鳴するか」でターゲットを自然に絞り込む
- 共鳴したユーザーはページ滞在時間が長く、CVRが高い傾向がある
- **注意**: 英語コピーだけではFV品質が低く評価されることがある。Conceptセクションで日本語による哲学・価値観の補完を必ず設計する

## FV-010: 差別化施術方式型コピー（「完全〇〇」型）

**適用業種**: 美容・クリニック・整体院・パーソナルジム・個人レッスン  
**視覚的効果**: 明確な差別化・希少性の示唆・FAIL-004回避

### 構成要素
```
コピー構造:
  ① 「完全[施術方式]の[業種名]」形式のH1
  ② ベネフィット補足（サブコピー）
  ③ 実績バッジ（数字あれば）
  ④ CTAボタン
```

### コピーフォーマット
```
「完全マンツーマンスタイルの美容室」（Locco — D/56 CTA最高16/20）
「完全個室のネイルサロン」
「完全予約制の整骨院」
「少人数制パーソナルジム」
「1対1のオーダーメイドカット」
```

### 機能分析
| 要素 | 効果 |
|-----|------|
| 「完全」 | 例外なし・徹底という強調。信頼性訴求 |
| 施術方式（マンツーマン等） | 競合との明確な差別化点。FAQを事前に解消 |
| 「の〇〇サロン」 | 業種を明示してFAIL-004（FV訴求不明確）を回避 |
| 希少性の暗示 | 「完全マンツーマン = 枠が限られている」という稀少性を示唆 |

### スタイル実装
```css
.hero-title {
  font-family: 'Noto Serif JP', serif;
  font-size: clamp(28px, 4.5vw, 54px);
  font-weight: 700;
  color: #fff;
  letter-spacing: .05em;
  line-height: 1.4;
}
```

## FV-011: 飲食系ドアップ料理写真型（炭火・照り・湯気の三要素）

**適用業種**: 飲食全般（焼き鳥・カレー・ラーメン・焼肉・寿司・カフェ）  
**視覚的効果**: 食欲の即時刺激・来店動機の直接形成・CVR直結

### 構成要素
```
レイアウト:
  背景: 料理写真（画面の70〜100%占有 / 高解像度必須）
  オーバーレイ: 薄めのグラデーション（下部のみ 0.3〜0.6 — 上部は料理を最大表示）
  コピー: 左下 or 中央下寄り

コピー構造:
  ① 業態タグ（12px 小文字 / 上部 or ロゴ）
  ② 感情語コピー H1（「今夜も幸せ 焼き鳥時間」型）
  ③ 来店動機サブコピー（価格・素材・特徴の一文）
  ④ CTAボタン（「席を探す」「ネット予約」「地図を見る」等）
```

### 食欲を最大化する「三要素」

| 要素 | 効果 | 料理例 |
|-----|-----|------|
| 炭火の焦げ | 「香ばしさ」の視覚化 | 焼き鳥・焼肉・BBQ |
| 照り（つや）| 「旨み・ジューシー感」の視覚化 | 焼き鳥タレ・寿司・肉 |
| 湯気・蒸気 | 「熱さ・出来立て感」の視覚化 | ラーメン・鍋・焼きたて |

→ 三要素が揃うと「食欲×信頼（出来立て）×緊急性（今すぐ食べたい）」を同時に喚起

### FV料理写真の優先順位（GEN-003 10件から確立）

| 順位 | 写真タイプ | 効果 | 実証事例 |
|-----|---------|------|--------|
| 1位 | ドアップ料理写真 | 食欲直結・CVR最高 | SPICE HOLIC A/82・慶鳥 FV最高評価 |
| 2位 | 食材・素材写真 | 専門性・こだわりの証明 | とりすけ「朝びき丹波鶏」|
| 3位 | シェフ・職人写真 | 信頼性・人間味の付与 | 各分析推奨 |
| 4位 | 店内・空間写真 | 雰囲気の訴求のみ | — |
| 5位 | 内装写真のみ | 食欲刺激なし（最低）| A'unique失点の主因 |

### 実装メモ
```css
/* 下部からグラデーションオーバーレイ（料理を上部に最大表示）*/
.hero-food {
  position: relative;
  min-height: 100vh;
  background:
    linear-gradient(
      to top,
      rgba(0,0,0,.65) 0%,     /* 下: テキストエリアの可読性確保 */
      rgba(0,0,0,.15) 50%,    /* 中: 料理写真を見せる */
      rgba(0,0,0,.0)  100%    /* 上: 写真最大表示 */
    ),
    url('food-main.jpg') center/cover no-repeat;
}
/* コピーエリアは下部に配置 */
.hero-food-content {
  position: absolute;
  bottom: 10%;
  left: 5%;
  color: #fff;
}
/* 料理写真は中心部（皿・料理本体）にフォーカス */
.hero-img {
  width: 100%;
  height: 100%;
  object-fit: cover;
  object-position: center 40%; /* 料理が中〜上部に来るよう調整 */
}
```

## Learning Reason
### 保存理由
GEN-003（飲食系LP）10件の比較分析から、飲食LPのFV写真タイプが直接CVRに影響することが実証された。SPICE HOLICカレードアップ（A/82）vs A'unique内装写真（B-/65）の17点差、慶鳥の炭火串超アップ写真（FV最高評価）など複数の実証事例があり、「料理ドアップ写真 > 内装写真」を体系化する必要があったため。
### 再利用価値
飲食LP全般のFV設計において、写真選定の優先順位として毎回参照する。業種を問わず「主力商品のドアップ写真」という原則は物販・飲食・美容全般に転用できる。
### 特に優れている点
「炭火・照り・湯気の三要素」という具体的な撮影指示が、クライアントへの写真素材オーダー時に役立つ。「感情語コピー+ドアップ料理写真」の組み合わせ設計で食欲と来店動機を同時に形成できる点。
### 他案件への応用先
飲食系LP全般（焼き鳥・カレー・ラーメン・焼肉・寿司・カフェ・居酒屋）のFV設計。写真素材の撮影ディレクション指示書としても活用可。

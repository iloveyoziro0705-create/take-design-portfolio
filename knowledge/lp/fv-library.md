# ファーストビュー（FV）辞典

## 概要
LP制作時に即座に参照・適用できるFVパターンライブラリ。
Phase 2（50パターン到達後）から育成開始。

**現在のステータス**: Phase 1完了待ち（現在 5/50 パターン）

## FVパターン管理ルール

- パターンID: `FV-XXX`（3桁ゼロ埋め）
- 構成要素を必ず記録する
- 適用業種を明記する

### Learning Reason（各FVパターン追加時に必須記録）

新規パターンを追加する際は、パターン末尾に必ず以下のセクションを記録すること。

```markdown
## Learning Reason
### 保存理由
[なぜこのFVパターンをライブラリに追加したか（CVへの効果・発見の経緯）]
### 再利用価値
[適用できるFV場面の頻度・業種の幅広さ]
### 特に優れている点
[視線誘導・ベネフィット伝達・信頼要素配置の観点で優れている点]
### 他案件への応用先
[適用推奨の業種・LP目的・ビジュアル方針]
```

## FV-001: フルスクリーン背景画像型

**適用業種**: レストラン・ホテル・美容・旅行・不動産  
**視覚的効果**: 没入感・高級感・ブランド訴求

### 構成要素
```html
<!-- 構造 -->
<section class="hero" style="
  min-height: 100vh;
  background: linear-gradient(rgba(0,0,0,.45), rgba(0,0,0,.7)), url('hero.jpg') center/cover;
  display: flex;
  align-items: center;
">
  <!-- ラベル -->
  <span class="label">英字キャッチ / カテゴリ名</span>
  <!-- H1 -->
  <h1>メインキャッチコピー<br>（2行以内）</h1>
  <!-- サブコピー -->
  <p>ベネフィットを一文で</p>
  <!-- CTA -->
  <a href="#" class="btn">行動喚起テキスト →</a>
</section>
```

### デザイン仕様
- オーバーレイ: `rgba(0,0,0, 0.4〜0.65)`（写真の明度による）
- H1サイズ: `clamp(40px, 7vw, 88px)`
- CTA配置: H1から40px下
- ラベル色: アクセントカラー（金・ブランドカラー）

### 注意点
- 画像なし時はグラデーション背景で代替
- モバイルでは画像の中心部にフォーカスを当てる
- テキスト可読性確保のためオーバーレイ必須

## FV-011: 飲食系ドアップ料理写真型（炭火・照り・湯気の三要素）

**適用業種**: 飲食全般（焼き鳥・カレー・ラーメン・焼肉・寿司・カフェ）  
**視覚的効果**: 食欲の即時刺激・来店動機の直接形成・CVR直結

### 構成要素
```
レイアウト:
  背景: 料理写真（画面の70〜100%占有 / 高解像度必須）
  オーバーレイ: 薄めのグラデーション（下部のみ 0.3〜0.6 — 上部は料理を最大表示）
  コピー: 左下 or 中央下寄り

コピー構造:
  ① 業態タグ（12px 小文字 / 上部 or ロゴ）
  ② 感情語コピー H1（「今夜も幸せ 焼き鳥時間」型）
  ③ 来店動機サブコピー（価格・素材・特徴の一文）
  ④ CTAボタン（「席を探す」「ネット予約」「地図を見る」等）
```

### 食欲を最大化する「三要素」

| 要素 | 効果 | 料理例 |
|-----|-----|------|
| 炭火の焦げ | 「香ばしさ」の視覚化 | 焼き鳥・焼肉・BBQ |
| 照り（つや）| 「旨み・ジューシー感」の視覚化 | 焼き鳥タレ・寿司・肉 |
| 湯気・蒸気 | 「熱さ・出来立て感」の視覚化 | ラーメン・鍋・焼きたて |

→ 三要素が揃うと「食欲×信頼（出来立て）×緊急性（今すぐ食べたい）」を同時に喚起

### FV料理写真の優先順位（GEN-003 10件から確立）

| 順位 | 写真タイプ | 効果 | 実証事例 |
|-----|---------|------|--------|
| 1位 | ドアップ料理写真 | 食欲直結・CVR最高 | SPICE HOLIC A/82・慶鳥 FV最高評価 |
| 2位 | 食材・素材写真 | 専門性・こだわりの証明 | とりすけ「朝びき丹波鶏」|
| 3位 | シェフ・職人写真 | 信頼性・人間味の付与 | 各分析推奨 |
| 4位 | 店内・空間写真 | 雰囲気の訴求のみ | — |
| 5位 | 内装写真のみ | 食欲刺激なし（最低）| A'unique失点の主因 |

### 実装メモ
```css

      rgba(0,0,0,.0)  100%    /* 上: 写真最大表示 */
```

## Learning Reason
### 保存理由
GEN-003（飲食系LP）10件の比較分析から、飲食LPのFV写真タイプが直接CVRに影響することが実証された。SPICE HOLICカレードアップ（A/82）vs A'unique内装写真（B-/65）の17点差、慶鳥の炭火串超アップ写真（FV最高評価）など複数の実証事例があり、「料理ドアップ写真 > 内装写真」を体系化する必要があったため。
### 再利用価値
飲食LP全般のFV設計において、写真選定の優先順位として毎回参照する。業種を問わず「主力商品のドアップ写真」という原則は物販・飲食・美容全般に転用できる。
### 特に優れている点
「炭火・照り・湯気の三要素」という具体的な撮影指示が、クライアントへの写真素材オーダー時に役立つ。「感情語コピー+ドアップ料理写真」の組み合わせ設計で食欲と来店動機を同時に形成できる点。
### 他案件への応用先
飲食系LP全般（焼き鳥・カレー・ラーメン・焼肉・寿司・カフェ・居酒屋）のFV設計。写真素材の撮影ディレクション指示書としても活用可。

## FV-012: 日常価値訴求型FVコピー（「毎日のために〜」型）

**適用業種**: 美容サロン・ネイル・エステ・整体・カフェ  
**視覚的効果**: 日常的な来店動機形成・リピート訴求・心理的ハードル低下

### コピー構造
```
「毎日のために [心地のよいスタイルを]」
「毎日が [輝くようなスタイル創りを]」
「あなたの日常に [ちょっとした贅沢を]」
```

### 心理設計
- 「ハレの日」ではなく「日常の充実」を訴求 → 記念日以外でも来ていいという心理的許可
- 継続的なリピート来店動機の形成と直結
- 「毎日」の繰り返し語が記憶への定着を強化

### 適用コピー例（実証済み）
```
「毎日のために 心地のよいスタイルを」（Hériter サロン B/70 確認）
「毎日が輝くようなスタイル創りを」（MADA A/80 確認）
```

### 配置ルール
- H1またはH2コピーとして配置（補足で「日常使いできる美容室」等を付加）
- 背景: 施術シーンの自然光写真（過度な加工なしの温かみある写真が最適）
- CTAは「まずは体験予約」「初回限定お試し」等の低ハードル型と組み合わせる

### Learning Reason
**保存理由**: Hériter B/70・MADA A/80の2件連続確認で確立。「ハレの日専用から日常使いへ」という心理的再定義がリピート動機に直結することが確認された。  
**再利用価値**: 美容系LP全般（サロン・エステ・整体・カフェ）のFVコピー設計時に即適用可。  
**特に優れている点**: 「毎日」という言葉が来店ハードルを下げ、かつリピート動機を同時に形成する二重効果。  
**他案件への応用先**: 美容系全般・整体・ネイル・カフェ系LP。日常的サービスの全業種に転用可能。

## FV-013: スタッフ顔写真カルーセル型FV（採用特化）

**適用業種**: 採用LP（美容・医療・飲食・IT・福祉）  
**視覚的効果**: 「最初の1秒で人柄が伝わる」設計・来訪者の採用意欲最大化

### 構成要素
```
レイアウト（分割型）:
  左カラム（55%）: コピー + CTA
  右カラム（45%）: スタッフ顔写真カルーセル（3〜5名）

カルーセル各スライド:
  ① 顔写真（400×500px / object-fit:cover）
  ② 名前 + 役職/入社年度
  ③ 一言コメント（18文字以内）
```

### コピー構造
```
① 業種・ポジション明示バッジ
② H1: ライフスタイル or 悩み共感型コピー
③ サブコピー: 職場の魅力1文
④ CTA: 「まずは話を聞いてみる」（低ハードル）
```

### 心理設計
- FVで「誰と働くか」を即座に可視化することで「本当にいい職場か不明」という最大の採用障壁を先制除去
- 顔写真+名前で「実在する人物=リアルな職場」という信頼性を形成
- カルーセル（複数人表示）で「多様な人材が活躍している」を示す


## FV-027: ダブルミーニングコピー型FV（採用×ビジョン訴求）

**適用業種**: バス・交通・製造・建設・インフラ系採用LP（職業行為がビジョンと二重解釈できる業種）  
**視覚的効果**: 仕事の行為＝社会的使命という二重の意味が記憶に定着

### コピー構造
```
「I [動詞] the [概念].」（英語版）
「[職業行為動詞]で、[社会的使命]を。」（日本語版）

例（実証済み）:
「I drive the Future.」（Shachi Bus 57点/C+ 確認）
職業行為: drive = 運転する
ビジョン: drive = 推進する/リードする

転用例:
「BUILD the Future.」（建設）
「PROTECT the Tomorrow.」（警備）
「CONNECT the People.」（通信）
「CARE for the World.」（医療・福祉）
```

### 二重解釈の設計メソッド
```
STEP1: 職業の代表的な行為動詞を選ぶ（run/drive/build/care/protect/connect等）
STEP2: その動詞が社会的使命として読めるか確認
STEP3: 対象語（Future/People/Tomorrow等）を選定して意味が成立するか確認
STEP4: 英語または「動詞する + 使命」の日本語文で実装
```

### 注意点
- 日本語の場合は「〇〇で社会を動かす」「〇〇がつなぐ未来」等のパターンが自然
- 英語を使う場合はフォントに Jost / Montserrat 等のジオメトリックサンセリフを推奨
- 哲学型コピーのため、FVのみで業種・職種を伝えるのは難しい → ラベル補足が必須

### Learning Reason
**保存理由**: Shachi Bus採用LP（GEN-002-57点/C+）で初発見。「I drive the Future.」がバスドライバー採用LPで「運転する + 未来を推進する」の二重解釈を持つことが確認された。この技法の体系化で転用可能パターンとして登録。  
**再利用価値**: 交通・建設・製造・インフラ・警備・通信・医療採用LP全般に転用可能。  
**特に優れている点**: 職業の行為そのものがビジョン表現になるため、コピーに「仕事の誇り」と「会社のビジョン」が同時に宿る。  
**他案件への応用先**: バス・鉄道・建設・製造・警備・通信・医療・ロジスティクス採用LP全般。

```
             ├─ 導入企業ロゴが使える？    → FV-030（導入企業ロゴ直下型）
             ├─ LLMO・AI差別化訴求？      → FV-028（LLMO差別化型）
             └─ 課題解決フロー訴求？      → FV-023（課題解決フロー型）
```

Phase 2（50パターン到達後）から育成開始。

**現在のステータス**: Phase 1完了待ち（現在 5/50 パターン）

## FVパターン管理ルール

- パターンID: `FV-XXX`（3桁ゼロ埋め）
- 構成要素を必ず記録する
- 適用業種を明記する

### Learning Reason（各FVパターン追加時に必須記録）

新規パターンを追加する際は、パターン末尾に必ず以下のセクションを記録すること。

```markdown
## Learning Reason
### 保存理由
[なぜこのFVパターンをライブラリに追加したか（CVへの効果・発見の経緯）]
### 再利用価値
[適用できるFV場面の頻度・業種の幅広さ]
### 特に優れている点
[視線誘導・ベネフィット伝達・信頼要素配置の観点で優れている点]
### 他案件への応用先
[適用推奨の業種・LP目的・ビジュアル方針]
```

## FV-001: フルスクリーン背景画像型

**適用業種**: レストラン・ホテル・美容・旅行・不動産  
**視覚的効果**: 没入感・高級感・ブランド訴求

### 構成要素
```html
<!-- 構造 -->
<section class="hero" style="
  min-height: 100vh;
  background: linear-gradient(rgba(0,0,0,.45), rgba(0,0,0,.7)), url('hero.jpg') center/cover;
  display: flex;
  align-items: center;
">
  <!-- ラベル -->
  <span class="label">英字キャッチ / カテゴリ名</span>
  <!-- H1 -->
  <h1>メインキャッチコピー<br>（2行以内）</h1>
  <!-- サブコピー -->
  <p>ベネフィットを一文で</p>
  <!-- CTA -->
  <a href="#" class="btn">行動喚起テキスト →</a>
</section>
```

### デザイン仕様
- オーバーレイ: `rgba(0,0,0, 0.4〜0.65)`（写真の明度による）
- H1サイズ: `clamp(40px, 7vw, 88px)`
- CTA配置: H1から40px下
- ラベル色: アクセントカラー（金・ブランドカラー）

### 注意点
- 画像なし時はグラデーション背景で代替
- モバイルでは画像の中心部にフォーカスを当てる
- テキスト可読性確保のためオーバーレイ必須

**適用業種**: IT・SaaS・教育・金融・採用  
**視覚的効果**: モダン・テック感・清潔感

### 構成要素
```css
.hero {
```

## FV-003: テキスト中心型（ビジュアルレス）

**適用業種**: コンサル・士業・採用・B2B  
**視覚的効果**: 誠実さ・情報の明確さ・信頼性

### 構成要素
```
レイアウト:
  左カラム（60%）: コピー + CTA
  右カラム（40%）: 数字実績 or イラスト

コピー階層:
  ① 悩みラベル（12px 追跡あり）
  ② H1 キャッチ（48〜72px）
  ③ サブコピー（17〜18px 行間1.8）
  ④ CTAボタン
  ⑤ 安心訴求（「無料」「〇秒で申込」等）
```

## FV-004: 分割レイアウト型（左テキスト / 右画像）

**適用業種**: 美容・サロン・クリニック・物販  
**視覚的効果**: バランス・視線誘導・情報整理

### 構成要素
```
<div style="display: grid; grid-template-columns: 1fr 1fr; min-height: 100vh;">
  <div class="hero-text">  ← 左: コピー + CTA </div>
  <div class="hero-image"> ← 右: 画像（object-fit: cover）</div>
</div>
```

### モバイル対応
- 縦並び: 画像を上、テキストを下
- 画像高さ: `min(50vh, 400px)`

## FV-005: 動画背景型

**適用業種**: ブランド訴求・観光・イベント・高級サービス  
**視覚的効果**: 最高の没入感・プレミアム感

### 構成要素
```html
<video autoplay muted loop playsinline
  style="position:absolute; inset:0; width:100%; height:100%; object-fit:cover; z-index:-1;">
  <source src="hero.mp4" type="video/mp4">
</video>
```

### 注意点
- ファイルサイズ: 5MB以下推奨（WebM形式）
- 必ずmuted + autoplay属性
- Fallback画像を必ず設定
- 画像重視案件に限定使用

```javascript
  const d = Math.floor(diff / 86400000);
  const h = Math.floor((diff % 86400000) / 3600000);
  const m = Math.floor((diff % 3600000) / 60000);
  const s = Math.floor((diff % 60000) / 1000);
  // DOM更新
}
setInterval(update, 1000);
```

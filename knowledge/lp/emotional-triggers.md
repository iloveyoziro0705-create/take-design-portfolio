## このファイルの目的

LPコピーで使用する感情訴求トリガーを体系化し、再現可能な形で保存する。  
感情に訴えるコピーを「なんとなく」ではなく、**どのトリガーを使うか意図的に選択できる**状態を作ること。

## TRIG-001: 悩み共感トリガー

**利用目的**: 「この人は私のことを知っている」という信頼感を即形成し、離脱を防ぐ  
**向いている業種**: エステ・フェイシャルサロン・整体院・スクール・クリニック・転職サービス

### 心理メカニズム
悩みを具体的に言語化されると「わかってくれる人がいる」という安心感が生まれる。  
「こんな悩みはありませんか？」という問いかけが「YES」の連鎖を作り、受け入れ態勢を高める。

### コピー例
```
「ファンデーションで隠す毎日、もう終わりにしませんか。」
「朝の鏡を見るたびに溜め息をついている方へ。」
「毛穴・くすみ・乾燥肌でお悩みの方へ」
「腰痛で毎朝ベッドから起き上がるのが憂鬱な方へ」
「英語を話せない自分を10年以上後悔してきた方へ」
```

### 利用タイミング
- FVの最初のバッジ or H1
- Worryセクションの見出し
- LP全体の冒頭フレーム

### 期待効果
- FV離脱率の低下（ターゲットが「これは私のことだ」と認識）
- スクロール率の向上
- CTRの向上（広告→LP遷移後の滞在時間増加）

### 実装パターン
```html
<!-- バッジ型（FV冒頭） -->
<div class="worry-badge">毛穴・くすみ・乾燥肌でお悩みの方へ</div>

<!-- H1型（大見出し） -->
<h1>ファンデーションで隠す毎日、<br>もう終わりにしませんか。</h1>

<!-- リスト型（Worryセクション） -->
<ul>
  <li>✓ 年齢とともにハリ・弾力が失われてきた</li>
  <li>✓ 毛穴の開き・くすみが改善されない</li>
  <li>✓ エステの勧誘が怖くて行けない</li>
</ul>
```

### Learning Reason
**保存理由**: beauty-salon-lp/index.html のFV改善（STEP1-①）で「完全個室で自分だけの時間を」というサービス説明型H1から「ファンデーションで隠す毎日、もう終わりにしませんか。」という悩み訴求型H1への変更を実施。悩みの言語化がCV設計の起点になることを体系化。  
**再利用価値**: 悩みが明確な全業種（美容・医療・教育・転職・健康）のFV/Worryセクション設計で毎回参照。

## TRIG-002: 不安除去トリガー

**利用目的**: 購買を阻む具体的な不安・恐れを先制して打ち消す  
**向いている業種**: エステ・高額サービス・医療・投資・法人向けサービス

### 心理メカニズム
「失敗したらどうしよう」「騙されないか」という不安は行動の最大阻害要因。  
不安を先に言語化して「それは心配しなくていい」と示すことで購買障壁を下げる。

### コピー例
```
「無理な勧誘は一切しません。」
「コース契約は不要。都度払いでOK。」
「初めてのエステでも、女性スタッフが丁寧に対応します。」
「解約手数料ゼロ。いつでも退会できます。」
「30日間返金保証つき。合わなければ全額返金。」
```

### 利用タイミング
- FV内のサブコピー or バッジ
- 価格セクション（高額感の払拭）
- FAQ（よくある不安への回答）
- 固定フッターCTA内

### 期待効果
- 購買迷いの解消（「やっぱりやめておこう」を防ぐ）
- FAQ閲覧率の低下（先制解消により不要に）
- CV直前の離脱防止

### 主要な「不安カテゴリ」と対応コピー
| 不安の種類 | 対応コピー例 |
|-----------|-----------|
| 勧誘・契約 | 「勧誘なし / コース不要」 |
| 金額・支払い | 「都度払いOK / 追加料金なし」 |
| 初体験 | 「初めての方も安心 / 女性スタッフのみ」 |
| プライバシー | 「完全個室 / 人目を気にせず」 |
| 効果への不信 | 「○○名の実績 / 口コミ★4.8」 |

### Learning Reason
**保存理由**: beauty-salon-lp分析・Worryセクション設計から、エステ系LPの最大転換障壁が「勧誘への恐れ」と「高額契約への不安」であることが確認された。これらの不安を「先制除去」するコピー設計が美容LP全般で有効。

## TRIG-003: 理想投影トリガー

**利用目的**: 購買後の理想状態を具体的に想起させ、欲求を高める  
**向いている業種**: 美容・ダイエット・転職・スクール・フィットネス

### 心理メカニズム
人は「今の痛み回避」より「未来の喜び獲得」を想像したときに購買意欲が高まる。  
具体的で感情を伴う「理想の未来描写」が欲求を形成する。

### コピー例
```
「素肌に自信を持てる自分へ。」
「鏡を見るたびにニッコリできる毎日へ。」
「好きな服を躊躇せず着られる体へ。」
「英語で世界と話せる自分へ。」
「会議でハッキリ意見を言える自分になる。」
```

### 利用タイミング
- FVのサブコピー（悩みH1の直後）
- Before/Afterセクション（AFTER側の描写）
- 最終CTAセクション（「こんな自分になれる」誘導）

### 期待効果
- 欲求形成（「なりたい自分」のイメージを固定）
- 口コミ・体験談の説得力向上（理想→現実という流れ）
- 購買決定の後押し

### Learning Reason
**保存理由**: FV-031の5層構造（悩み→理想→証拠→オファー→CTA）において「理想層」として体系化。悩み訴求の直後に理想を提示するという順番が心理的にスムーズな購買導線を形成することを確認。

## TRIG-004: 憧れ刷込トリガー

**利用目的**: 高級感・特別感を演出し「ここに属したい」という憧れを形成する  
**向いている業種**: ラグジュアリー美容・高級飲食・プレミアムサービス・高額スクール

### 心理メカニズム
「あの人みたいになりたい」「そのステージに上がりたい」という憧れは購買動機の根幹。  
ブランドビジュアル・コピーのトーンで「選ばれた人たちの世界」を演出する。

### コピー例
```
「本物の贅沢を、あなただけに。」
「選ばれた人だけが知るプライベートサロン。」
「一流が通うエステティック。」
「世界基準の技術を、完全個室で。」
```

### 利用タイミング
- ブランドコンセプトセクション
- FV（ラグジュアリー路線のみ）
- オーナーストーリー

### 期待効果
- 価格感度の低下（高い = 価値があると感じる）
- 特定ターゲット層の自己選別（憧れに共鳴する人だけが残る）

### Learning Reason
**保存理由**: FV-002（ラグジュアリー美容型）の分析から、高価格帯サービスでは「共感」より「憧れ」が購買動機になることを確認。価格帯×訴求タイプの組み合わせ判断に必要なため保存。

## TRIG-005: 損失回避トリガー

**利用目的**: 「今やらないこと」のコストを意識させ、即行動を促す  
**向いている業種**: 全業種（期間限定オファー・限定数・割引）

### 心理メカニズム
プロスペクト理論: 人は「得る喜び」より「失う痛み」を約2倍強く感じる。  
「この機会を逃すともったいない」という感覚が先送り防止に機能する。

### コピー例
```
「今月中に申込まないと通常料金に戻ります。」
「残り3名様分のみ。」
「この価格で体験できるのは初回だけです。」
「毎日少しずつ悪化している肌を、今日から変えませんか。」
「放置すれば放置するほど、改善に時間がかかります。」
```

### 利用タイミング
- 価格セクション（割引の期限表示）
- 最終CTAセクション
- 固定フッターCTA
- Hero緊急性バッジ

### 期待効果
- 先送り防止（「また今度でいいか」の打消し）
- CV率の向上（特に迷っている見込み客への後押し）

### 実装例（緊急性バッジ）
```html
<div class="urgency-badges">
  <span class="badge">🎁 初回限定</span>
  <span class="badge outline">⏰ 期間限定特価</span>
  <span class="badge">50%OFF</span>
</div>
```

### Learning Reason
**保存理由**: STEP1-②（オファー強化）で「初回限定」「期間限定特価」「50%OFF」の3バッジを追加した際の設計根拠として体系化。損失回避トリガーは全業種で有効なため最重要カテゴリとして保存。

## TRIG-006: 比較優位トリガー

**利用目的**: 競合との差を明確にし「ここでなければならない理由」を作る  
**向いている業種**: 美容・SaaS・教育・採用（競合が多い業種全般）

### 心理メカニズム
「なぜここを選ぶのか」という問いへの答えを先に提供する。  
「他と何が違うの？」という心理的問いに答えることで選択の合理性を提供する。

### コピー例
```
「勧誘なし・コース不要は当サロンだけの約束です。」
「完全個室×女性スタッフのみ。他にはない安心を。」
「大手では実現できない、1対1のカウンセリング。」
「解約率98%削減。他社ツールでは実現できない数字。」
```

### 利用タイミング
- 「選ばれる理由」セクション
- FVサブコピー（差別化1文）
- 比較表セクション

### 期待効果
- 検索競合との差別化
- 「似たような店が他にもある」という離脱の防止

### Learning Reason
**保存理由**: beauty-salon-lp Reasonsセクションの設計から。「完全個室」「勧誘なし」「都度払い」という差別化要因が強力なCVポイントになることを確認。比較優位の明示が「選ばれる理由」セクション設計の核心。

## TRIG-007: 希少性トリガー

**利用目的**: 「少ない・貴重・入りにくい」という価値を演出し欲求を高める  
**向いている業種**: 高額サービス・完全予約制サロン・限定商材・人気飲食

### 心理メカニズム
希少なものは価値があるという心理バイアス。  
「誰でも入れる」サービスより「入りにくいけど選ばれる」方が価値が高く感じられる。

### コピー例
```
「完全個室・完全予約制（受け入れは1日4名まで）」
「新規は月10名様限定でお受けしています。」
「モニター様3名を募集します。（定員になり次第終了）」
「このコースは年2回のみご案内しています。」
```

### 利用タイミング
- FV内バッジ
- 価格セクション（制限の説明）
- 最終CTAセクション

### 期待効果
- 価格感度の低下（希少 = 高価値）
- 「今すぐ」行動の誘発

### Learning Reason
**保存理由**: FV-031（悩み訴求5層構造）のオファー設計で「緊急性バッジ」を配置した際の心理根拠として体系化。

## TRIG-008: 緊急性トリガー

**利用目的**: 先送り行動を防ぎ、今すぐ申し込む理由を作る  
**向いている業種**: 割引LP・期間限定オファー・セール・採用（期限がある場合）

### 心理メカニズム
「いつでもできる」ことは「後回し」になる。締め切りが行動を促す。  
具体的な期限・残数・条件を提示することで「今すぐ」の必要性を作る。

### コピー例
```
「残り○日で通常価格に戻ります。」
「今月の新規受付は残り○名です。」
「このキャンペーンは○月○日（○）まで。」
「先着10名様限定プレゼントつき。」
```

### 利用タイミング
- FV緊急性バッジ
- 価格セクション（期限明記）
- 最終CTA直前

### 期待効果
- CV率の即効向上
- 直帰（今日は見るだけ）の防止

### 注意事項
- 虚偽の緊急性は禁止（コンプライアンス）
- 期限・残数は実態に基づいて設定する

### Learning Reason
**保存理由**: TRIG-005（損失回避）と関連するが、「締め切り・期限」に特化した実装が多いため独立したカテゴリとして保存。

## TRIG-009: 自己投資トリガー

**利用目的**: 「費用」ではなく「自分への投資」として再フレーミングし、高価格への抵抗を下げる  
**向いている業種**: 美容・スクール・健康・フィットネス・ビジネス系セミナー

### 心理メカニズム
同じ1万円でも「消費」と感じると後ろめたいが、「投資」と感じると正当化できる。  
「自分のために使う」という概念をコピーで植え付けることで購買後悔を軽減する。

### コピー例
```
「毎月1回、自分のために使う90分。」
「この1回が、これからの自分を変える。」
「今のうちに肌を整えることが、5年後の自分への贈り物。」
「一番お金をかけるべき場所は、一生使う顔です。」
「スキルへの投資は、永遠に誰にも奪われない資産。」
```

### 利用タイミング
- 価格セクション（高価格の正当化）
- FVサブコピー（日常贅沢訴求）
- 最終CTAセクション（背中を押すコピー）

### 期待効果
- 価格感度の低下
- 購買後の満足度向上（「良い買い物をした」）
- リピート率向上（「定期的に来ることが自分への投資」）

### Learning Reason
**保存理由**: FV-012（日常価値訴求型FV）との連動設計。「毎日のために通うエステ」という概念が「消費」から「投資」へのリフレーミングと同一構造であることを体系化。

## TRIG-010: 変化実感トリガー

**利用目的**: 施術・サービス前後の変化を具体的に描写し、効果への確信を形成する  
**向いている業種**: 美容・ダイエット・整体・スクール・転職

### 心理メカニズム
「Before / After」の具体的な変化描写が「本当に効果があるのか？」という疑念を解消する。  
数字・体験談・具体描写が三位一体で「確信」を形成する。

### コピー例
```
「施術後、同僚に『何かいいことあった？』と言われました。」
「頬がリフトアップした感じ。翌朝鏡を見て自分でびっくり。」
「3回通ったら、毛穴が目立たなくなってきた。」
「6ヶ月で10kg減量。自分でも信じられない変化でした。」
「TOEIC 450点→820点。3ヶ月でここまで変われるとは。」
```

### 4部構成ストーリー型（最強パターン）
```
来店前の悩み  : 「[具体的な悩み状況]だった」
来店の決め手  : 「[選んだ理由]が決め手になった」
施術後の変化  : 「[具体的な変化の描写]」
感想コメント  : 「[感情を伴うコメント]」
```

### 利用タイミング
- Before/Afterセクション
- お客様の声（Voice）セクション ← **ここが最強**
- 実績・事例セクション

### 期待効果
- 「自分も変われる」という自己効力感の形成
- 口コミの説得力向上（個人の体験→自分の未来想起）
- CVR向上（確信形成→行動）

### 実装パターン（ストーリー型Voice）
```html
<div class="voice-story">
  <div class="voice-story-item">
    <span class="label">来店前の悩み</span>
    <span>[悩みの具体的状況]</span>
  </div>
  <div class="voice-story-item">
    <span class="label">来店の決め手</span>
    <span>[選んだ理由]</span>
  </div>
  <div class="voice-story-item">
    <span class="label">施術後の変化</span>
    <span>[変化の描写]</span>
  </div>
</div>
<p class="voice-comment">[感情を伴うコメント]</p>
```

### Learning Reason
**保存理由**: STEP1-④（口コミストーリー型への再構成）でbeauty-salon-lpのVoiceカードを「来店前の悩み→来店理由→施術後の変化→感想」の4部構成にリニューアル。単純なコメント羅列より信頼性・共感度が大幅に向上することを確認。

# FV Copy Library — ファーストビューコピー辞典

**バージョン**: 1.0  
**作成日**: 2026-06-24  
**管理ルール**: `knowledge/governance.md` の SST マップ参照  
**目的**: LP制作時に「なんとなくコピーを書く」を排除し、業種×ターゲット×感情×オファーに応じた最適FVコピーを即選択できる状態を作ること

---

## 利用ルール

1. LP制作時は必ずこのライブラリを参照してから FV コピーを作成する
2. 新規コピー作成より**既存パターンの流用・カスタマイズを優先**する
3. 新パターンを発見したら末尾に追記（Learning Reason 必須）
4. 各パターンの `TRIG-XXX` は `emotional-triggers.md` を参照
5. 各パターンの `OFFER-XXX` は `offer-library.md` を参照

---

## FV-COPY-001: 肌悩み直撃型

**コピータイプ**: 悩み訴求型  
**利用業種**: 美容サロン・エステ・フェイシャル・スキンケアサロン  
**利用条件**: ターゲットの肌悩みが明確（毛穴・くすみ・乾燥・ニキビ跡など）  
**ターゲット**: 肌悩みを抱えた女性（20〜50代）、セルフケアに限界を感じている人  
**相性オファー**: OFFER-001（初回割引）/ OFFER-003（無料診断）

### コピー構造

```
悩み: 「ファンデーションで隠す毎日、もう終わりにしませんか。」
理想: 「すっぴんに自信が持てる肌へ。」
証拠: 「累計3,200名の施術実績 / ★4.8（312件）」
オファー: 「初回限定 通常12,000円 → 5,980円（50%OFF）」
CTA: 「今すぐLINEで予約する →」
```

### 実例コピー

```
【バッジ】毛穴・くすみ・乾燥肌でお悩みの方へ

【H1】ファンデーションで隠す毎日、
もう終わりにしませんか。

【サブ】プロの手技で、あなた本来の素肌を引き出します。
完全個室・完全予約制で、ゆったり通えるフェイシャルエステ。

【証拠バー】累計3,200名 | ★4.8（312件）| 経験8年

【オファー】初回体験 5,980円（通常12,000円）
【CTA】LINEで予約する →
```

### 利用感情
- **主**: TRIG-001（悩み共感）
- **副**: TRIG-005（損失回避）、TRIG-008（緊急性）

### 期待効果
- FV離脱率 20〜30%低下（ターゲット自認による滞在）
- 「私のことだ」という共感によるスクロール促進
- 悩み言語化による信頼形成

### Learning Reason
**保存理由**: beauty-salon-lp（Luminos Skin）STEP1改善で「完全個室で自分だけの時間を」→「ファンデーションで隠す毎日」への変更でFV効果が大幅向上。悩み直撃がCVの起点になることを体系化。  
**再利用価値**: 美容系全般で最頻利用。悩みが明確な業種ならほぼ全て応用可能。

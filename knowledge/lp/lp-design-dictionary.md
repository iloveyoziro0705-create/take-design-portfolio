## Pattern-LP-001: 美容サロン基本構成

**業種**: 美容サロン（ヘア・ネイル・エステ・脱毛）  
**目的**: 新規来店予約獲得  
**登録日**: 2026-06-20  
**参照案件**: beauty-salon-lp（Lumière Beauty Salon）

### セクション構成
```
1. Hero（FV）
   - 大型ビジュアル or グラデーション背景
   - キャッチコピー（感情訴求）
   - サブコピー（ベネフィット）
   - CTA（予約ボタン）
   - 信頼バッジ（実績数・評価）

2. 実績・数字
   - 顧客数・満足度・経験年数
   - 数字を大きく表示

3. 特徴・強み（3〜4項目）
   - アイコン + 見出し + 説明文
   - グリッドレイアウト

4. サービス一覧
   - カード型またはリスト型
   - 料金表示あり/なし（業種による）

5. お客様の声（SNS型）
   - 写真 + テキスト
   - 星評価
   - 3〜6件表示

6. Before/After または 施術例
   - ビジュアル比較
   - キャプション

7. スタッフ紹介
   - 顔写真 + 名前 + 資格 + 一言コメント

8. アクセス・店舗情報
   - Googleマップ埋め込み
   - 営業時間・定休日

9. FAQ
   - アコーディオン形式
   - 5〜8項目

10. CTA（最終）
    - 予約フォームまたはLINE誘導
    - 電話番号表示
```

### 配色パターン
- 高級系: `#faf7f3`(背景) + `#c8a56a`(金) + `#3b2f2a`(テキスト)
- ナチュラル系: `#f9f5f0` + `#b5936c` + `#3d2b1f`
- クール系: `#f5f5f7` + `#8b7355` + `#1a1a1a`

### タイポグラフィ
- 見出し: Noto Serif JP（明朝体）
- 本文: Noto Sans JP
- ラベル/英字: Jost, Cormorant Garamond

### 心理学トリガー
- 社会的証明（口コミ・実績数）
- 権威性（資格・年数・受賞歴）
- 希少性（限定オファー・残り席数）
- 美的一貫性（ブランドカラーの統一）

### CV要因
- LINEボタンの視認性（固定ヘッダーCTA）
- Before/Afterビジュアルの訴求力
- 口コミの具体性（名前・地域・年代）

## Pattern-LP-002: レストラン・飲食店オープンLP

**業種**: 飲食店（レストラン・カフェ・居酒屋）  
**目的**: オープン告知・予約獲得  
**登録日**: 2026-06-20  
**参照案件**: restaurant-lp

### セクション構成
```
1. Hero（FV）
   - フルスクリーン背景画像（料理 or 店内）
   - ダークオーバーレイ（0.4〜0.6）
   - オープン日 / キャッチコピー
   - 予約CTAボタン

2. コンセプト
   - シェフのメッセージ or 店のこだわり
   - 横幅広めのテキストブロック

3. 人気メニュー
   - 料理写真 + 料理名 + 説明
   - グリッドまたはスライダー

4. 雰囲気・空間
   - 店内写真ギャラリー
   - マルチカラム写真並び

5. アクセス
   - 地図 + 住所 + 営業時間

6. 予約CTA（最終）
   - 電話 / ネット予約
```

### 配色パターン
- ラグジュアリー和食: `#0f0f0f`(BG) + `#f5e6c8`(金テキスト) + `#d8a76a`(アクセント)
- ナチュラルカフェ: `#faf8f5` + `#6b4c2a` + `#e8d5b7`
- モダンフレンチ: `#1a1a2e` + `#e8c17a` + `#ffffff`

### タイポグラフィ
- 見出し: serif（Georgia, Noto Serif JP）
- 本文: Noto Sans JP

### 心理学トリガー
- 視覚的食欲喚起（料理写真の品質）
- 希少性（期間限定メニュー・席数）
- 権威性（シェフ経歴・受賞歴）

### CV要因
- 予約ボタンのアクセシビリティ（電話 + ネット予約の2段構え）
- 料理写真の解像度・撮影品質

## Pattern-LP-003: 採用LP（保育士・介護・医療系）

**業種**: 医療・福祉・介護・保育  
**目的**: 求職者の応募獲得  
**登録日**: 2026-06-20  
**参照案件**: hoikushi-mikata-lp

### セクション構成
```
1. Hero（FV）
   - スタッフ集合写真 or 職場風景
   - 「こんな悩みありませんか？」型コピー
   - 年収・待遇の数字訴求
   - 応募CTAボタン

2. 求職者の悩み（共感）
   - 現状の課題をリスト化
   - 「あなたのことです」と直接語りかける

3. この職場が選ばれる理由
   - 数字ベースの強み（3〜5項目）
   - アイコン + 短い説明

4. 働き方・待遇
   - 給与・休日・福利厚生
   - 表形式 or カード形式

5. 先輩スタッフの声
   - 写真 + 年齢・経験年数 + コメント
   - リアルな体験談

6. 1日の仕事の流れ
   - タイムライン形式

7. 施設・環境紹介
   - 写真ギャラリー

8. 応募の流れ
   - ステップ形式（3〜4ステップ）

9. 応募フォームCTA
   - 名前・連絡先・希望シフトのみ
   - ハードルを下げる（「まずはお気軽に」）
```

### 配色パターン
- 保育: `#fff8f2` + `#ff8c69` + `#4a3728`
- 介護: `#f0f7ff` + `#4a90d9` + `#2c3e50`
- 医療: `#f5faf8` + `#2eb88a` + `#1a3a2a`

### 心理学トリガー
- 共感訴求（悩みの言語化）
- 安心感（職場環境・先輩の顔出し）
- 具体性（数字・条件の明示）
- 行動ハードルの低下（まずは見学OK等）

### CV要因
- 悩み言語化の精度（求職者との一致度）
- 待遇の数字の競合優位性
- 応募フォームの項目数（少ないほど高CV）

## Pattern-LP-004: 金融・資産運用LP

**業種**: 金融・投資・資産形成  
**目的**: セミナー申込・資料請求・面談予約  
**登録日**: 2026-06-20  
**参照案件**: finance-lp

### セクション構成
```
1. Hero（FV）
   - 数字インパクト（「〇〇万円の差」「利回り〇%」）
   - ターゲット悩み直球型コピー
   - 無料相談CTAボタン
   - 信頼バッジ（実績・取扱高）

2. こんな方に
   - ターゲット条件リスト
   - チェックマーク形式

3. サービスの特徴
   - 3〜4項目の強み
   - 図解 or アイコン

4. 実績数字
   - 顧客数・運用総額・利回り実績
   - カウンターアニメーション

5. 信頼性・安心感
   - 金融ライセンス・登録番号
   - メディア掲載・受賞歴

6. お客様の声
   - 年代・職業 + 具体的な成果コメント

7. 無料相談の流れ
   - 3ステップ
   - 「相談するだけでOK」の低ハードル化

8. CTA（最終）
   - 無料相談フォーム
   - 電話番号
   - リスク開示（コンプライアンス）
```

### 配色パターン
- 安定・信頼: `#f8f9fb` + `#1a3a5c` + `#2e7d32`
- プレミアム: `#0a0a14` + `#c8a96e` + `#e8e8e8`
- クリーン: `#ffffff` + `#0052cc` + `#172b4d`

### 心理学トリガー
- 損失回避（「このままでは〇〇万円損」）
- 権威性（資格・登録番号・実績）
- 社会的証明（顧客数・運用総額）
- 希少性（無料相談の枠数制限）

### CV要因
- 数字の具体性（抽象的な表現はCV低下）
- コンプライアンス表示による信頼感
- 相談ハードルの低下表現

## Pattern-LP-005: Eラーニング・オンラインスクールLP

**業種**: 教育・スクール・オンライン講座  
**目的**: 受講申込・無料体験申込  
**登録日**: 2026-06-20  
**参照案件**: elearning-lp

### セクション構成
```
1. Hero（FV）
   - 「〇〇ができるようになる」型の成果訴求
   - 受講生の変化ビフォーアフター
   - 無料体験CTAボタン
   - 信頼数字（受講生数・満足度）

2. こんな悩みを解決
   - 学習前の課題リスト
   - ターゲットとの共感

3. このスクールで得られること
   - スキル・資格・キャリア
   - 具体的な成果

4. カリキュラム・コース内容
   - モジュール構成
   - 学習ステップ

5. 受講生の声・実績
   - 転職成功・収入増加・資格取得
   - 写真 + 数字入りコメント

6. 講師紹介
   - 経歴・実績・写真
   - 権威性の証明

7. 受講料・プラン
   - 比較表
   - 最もおすすめプランをハイライト

8. よくある質問
   - 不安を先回りして解消

9. CTA（最終）
   - 無料体験/説明会申込
   - 期間限定割引（希少性）
```

### 配色パターン
- テック系: `#0f172a` + `#6366f1` + `#e2e8f0`
- 親しみやすい: `#fff9f0` + `#f97316` + `#1e293b`
- プロフェッショナル: `#f8fafc` + `#0ea5e9` + `#0f172a`

### 心理学トリガー
- 変革への期待（ビフォーアフター）
- 社会的証明（受講生の具体的成果）
- 希少性（入学枠・早期割引）
- コミットメント（無料体験で小さな一歩）

### CV要因
- 成果の具体性（「年収100万UP」「3ヶ月で転職」）
- 卒業生の信頼性（実名・顔写真・職業）
- 無料体験の申込ハードルの低さ

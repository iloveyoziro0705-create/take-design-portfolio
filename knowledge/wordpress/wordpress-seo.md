# WordPress SEO辞典

## 概要
WordPressサイトのSEO施策を業種・目的別に体系化する。

---

## SEO基本設定チェックリスト

### WordPress初期設定（必須）

- [ ] 設定 → 一般 → サイトURL（www有無を統一）
- [ ] 設定 → パーマリンク → 「投稿名」に変更
- [ ] 設定 → 表示設定 → 検索エンジンがインデックスする（ON）
- [ ] XMLサイトマップの生成と Google Search Console への送信
- [ ] robots.txt の設定確認
- [ ] Google Analytics 4 の設置
- [ ] Google Search Console の設置・所有権確認

## ページ別SEO設定

### トップページ

```
title: [地域名][業種名] | [サービスの強み] - [サイト名]
例: 渋谷の美容サロン | 最短当日予約・透明感トリートメント - Lumière

description: 150文字以内 / ベネフィット + 地域 + CTA
例: 渋谷で人気の美容サロン「Lumière」。駅徒歩2分、当日予約OK。
    トリートメントで艶やかな仕上がりを体験してください。

H1: キャッチコピー（ページに1つのみ）
H2: 各セクション見出し
H3: カード見出し
```

### 個別サービスページ

```
URL: /[service-name]/
title: [サービス名] - [地域名] | [サイト名]
例: /hair-treatment/ → 髪質改善トリートメント - 渋谷 | Lumière

H1: [サービス名]（[地域名]の[サービス]なら[サイト名]）
本文: 800文字以上 / サービス詳細・よくある質問を含む
内部リンク: 関連サービスページへ
```

### ブログ・コラムページ

```
キーワード戦略:
  メインKW: 検索ボリューム 100〜1,000（競合が低いゾーン）
  ロングテール: 「[地域]+[サービス]+[悩み]」

文字数目安:
  ローカル記事: 800〜1,500文字
  解説記事: 2,000〜4,000文字
  完全ガイド: 5,000文字以上

構成テンプレート:
  H1: [キーワード]完全ガイド | 〇〇が解説
  H2: [キーワード]とは
  H2: [キーワード]の特徴・メリット
  H2: [キーワード]の選び方
  H2: [キーワード]でよくある質問（FAQ）
  H2: まとめ
```

## ローカルSEO（地域ビジネス必須）

### Googleビジネスプロフィール最適化

```
必須設定:
  ✓ 正確な店舗名（サイト名と統一）
  ✓ カテゴリ（業種を正確に選択）
  ✓ 住所（Googleマップと完全一致）
  ✓ 電話番号（クリック通話対応）
  ✓ 営業時間（曜日別・祝日設定）
  ✓ Webサイトリンク
  ✓ 写真（店舗内外・スタッフ・商品）20枚以上
  ✓ 投稿（週1回更新推奨）
  ✓ Q&A（よくある質問を事業者が先に登録）

レビュー対応:
  ✓ 全レビューに返信（1週間以内）
  ✓ 高評価レビューへの礼儀ある返信
  ✓ 低評価レビューへの誠実な対応
```

## コアウェブバイタル（CWV）対策

### 目標値

| 指標 | 目標値 | 説明 |
|------|--------|------|
| LCP | 2.5秒以内 | 最大コンテンツ描画 |
| FID / INP | 100ms以内 | ユーザー入力への応答 |
| CLS | 0.1以下 | 視覚的安定性 |

### 改善施策

```
LCP改善:
  ✓ ヒーロー画像のプリロード: <link rel="preload" as="image" href="hero.jpg">
  ✓ 画像のWebP変換（EWWW Image Optimizer）
  ✓ CDN の活用
  ✓ サーバーレスポンスタイム 200ms以内

CLS改善:
  ✓ 画像に width/height 属性を必ず設定
  ✓ フォントのfont-display: swap 使用
  ✓ 広告・埋め込みコンテンツのスペース予約

INP改善:
  ✓ JavaScriptの最小化・遅延読み込み
  ✓ サードパーティスクリプトの非同期読み込み
```

## 内部リンク戦略

```
サイロ構造:
  トップ
  └── カテゴリページ（H2相当キーワード）
      └── 個別ページ（H3相当キーワード）

内部リンクルール:
  ✓ 関連する記事・ページへのリンク（3〜5個/ページ）
  ✓ アンカーテキストにキーワードを含める
  ✓ 「こちら」「詳細はコチラ」は使わない
  ✓ 孤立ページ（リンクが全くない）を作らない
```

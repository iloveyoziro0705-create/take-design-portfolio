# LP分析テンプレート — 標準手順書

**バージョン**: 1.2  
**作成日**: 2026-06-20  
**更新日**: 2026-06-20  
**適用範囲**: 全業種・全形式のLPに適用

---

## 使用方法

1. このファイルをコピーして `knowledge/learning-history/YYYY-MM-DD-[案件名].md` として保存
2. 各セクションを上から順に埋める（順序の入れ替え禁止）
3. 分析完了後、`lp-design-dictionary.md` への反映内容を確定させる
4. 感想・レビュー・批評・好みの記述は一切禁止

---

## 分析品質基準

| 基準 | 内容 |
|------|------|
| 再利用可能性 | 他案件に転用できる形式で記述する |
| パターン化 | 単発の観察ではなくPattern-XXX形式で分類する |
| 数値化 | 「多い」「少ない」ではなく具体的な数値で記録する |
| 中立性 | 感想・評価・批評の表現は一切使用しない |
| 完全性 | 全10セクションを必ず埋める（不明の場合は「要確認」と記載） |

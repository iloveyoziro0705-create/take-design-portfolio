## 最大の特徴: 7STEP × 4タイプ分岐インタラクティブ診断

本LPの核心機能。LP内完結型の対話式診断システム。

### 診断フロー（7問）

| STEP | 質問 | 選択肢 |
|------|------|-------|
| Q1 | 年齢 | 18〜24歳 / 25〜34歳 / 35〜44歳 / 45歳以上 |
| Q2 | 現在の状況 | 会社員 / アルバイト・派遣 / 無職 / 学生 |
| Q3 | いつから働きたい | 今すぐ / 1週間以内 / 1ヶ月以内 / まだ未定 |
| Q4 | 現在の貯金額 | 0〜5万円 / 5〜20万円 / 20万円以上 / 答えたくない |
| Q5 | 住まいの状況 | 実家 / 一人暮らし / 住む場所が必要 / 環境を変えたい |
| Q6 | 希望勤務地 | 関東 / 東海 / 関西 / どこでもOK |
| Q7 | 一番重視すること | 高収入・貯金 / 即入寮・寮の安心 / 女性向け・安全 / 人生リスタート |

### 診断結果 × 4タイプ

| タイプ | カラー | 特徴 |
|-------|--------|------|
| 最短貯金タイプ（高収入） | `--orange` | 製造・物流・高単価求人を提示 |
| 即入寮重視タイプ | `--blue` | 今すぐ入居可能・初期費用0円求人 |
| 女性安心タイプ | `--pink: #ec4899` | 女性寮あり・女性スタッフ多い求人 |
| 人生リスタートタイプ | `--navy: #1e3a8a` | 未経験歓迎・長期安定・ゆるい職場 |

### タイプ別分岐LPの構造

各タイプに以下のコンテンツが独立して存在:
- タイプ専用ヒーロー（見出し + ビジュアルカード）

## 設計理由

### なぜこの構成にしたのか

「住み込み求人に踏み出せない複数の不安」を一括解消する設計。
ターゲットは「お金・住まい・仕事・環境」すべてに課題を抱える多様な人々であり:

1. **FVで「30秒」の手軽さを約束**: 診断へのハードルを「30秒」という具体的な数字で最小化
2. **Stats（信頼数値）で「このサービスは実績ある」を先行伝達**: 10,000人・95%・3,000件・最短2日
3. **悩み6カードで「あなたのことを知っている」共感構築**: 具体的な悩みの言語化で自分事化
4. **比較表「住み込み vs 一人暮らし」**: 損得を可視化して合理的判断を促す
5. **診断による個別化**: 「あなた専用の答え」を提示することでエンゲージメントを最大化
6. **タイプ別分岐LP**: 一つのLPで4つのターゲットを個別に説得する超効率設計

### なぜこのFVを採用したのか

「人気求人TOP3パネル」をFV右側に配置した理由:
- 「どんな求人があるか」を診断前に暗示することで、診断へのモチベーションを高める
- 「愛知・製造32万円〜」「神奈川・物流28万円〜」「長野・ホテル24万円〜」という具体的な収入数値が「自分もこれになれる」という期待を醸成
- 「即入寮」「家具付き」「女性OK」のバッジがターゲット別の安心感を同時訴求
- ライブ点滅バッジ（「住み込み新生活シミュレーション」）でサービスの稼働感を演出

### なぜこの診断を採用したのか

7STEP × 4タイプ分岐診断を選んだ理由:
- 求人マッチングは「大量の選択肢から絞る」疲弊感が最大障壁 → 診断が解決する
- 「自分専用の結果」を見せることでCVへの心理的コミットメントが上がる
- 7問という数は「多すぎず少なすぎず」 — 答えるたびにプログレスバーが進む達成感
- 4タイプへの分岐で、単一LPでは届かない「女性向け」「リスタート」層も包括
- 診断後に「あなたと同じタイプの人が選んでいる求人」を見せることで社会的証明が機能

### 競合と比較した差別化要素

| 差別化軸 | 内容 |
|---------|------|
| インタラクティブ診断機能 | 一般的な「求人一覧型LP」との最大の差別化 |
| タイプ別分岐LP | 1LPで4ターゲットを個別説得する構造的優位性 |
| 「30秒」という具体的な所要時間明示 | 診断への摩擦を限界まで下げる |
| 人気求人TOP3のFV内配置 | ファーストビューで「何があるか」を先行開示 |
| 「住まい + 仕事」同時解決の訴求 | 求人のみ or 住まいのみの競合とは異なる統合価値提案 |

### 現在ならどう改善するか

1. **診断結果のパーソナライゼーション強化**: 回答の組み合わせで結果を16種類以上に分岐（現在4タイプ）
2. **診断結果のSNSシェア機能**: 「私は最短貯金タイプでした」のシェアボタンで自然流入増加
3. **LINE連携**: 診断結果をLINEで送る機能 → LINEリスト獲得とフォロー可能
4. **リアルタイム求人数表示**: 「今日の新着: 32件」などのライブデータで緊急感演出
5. **Before/After の数値化**: 「入寮3ヶ月後の貯金額: 平均XX万円」という具体的な変化数値
6. **求人お気に入り機能の活用**: ハートボタンはあるが保存先（マイリスト機能）が未実装
7. **モバイルUX最適化**: 2414行の巨大LPなので、スクロール深度と各セクションの離脱率を分析し優先度の低いセクションを削除・後半に移動

## Learning Reason

**保存理由**: 自社ポートフォリオの中で最大規模・最高難度の実装（2414行）。住み込み求人という複雑な複合価値（住まい + 仕事 + 収入）をインタラクティブ診断で個別解決する設計の原型として確立。  
**再利用価値**: 「診断 → タイプ判定 → タイプ別分岐LP」という設計パターンは、マッチング系サービス全般に転用可能な最強構造。この設計の実装経験は、スクール・転職・不動産・保険系など多様な業種で応用できる希少ノウハウ。  
**次回同業種への教訓**: 分岐LPが多いと制作コストが指数的に増加する。次回は「共通コンポーネント設計（タイプ別CSS変数切り替えのみ）」を最初から採用すること（今回は既にこの方式を採用済み — `--accent` 変数での切り替えが正解だった）。巨大LPはセクション単位でのA/Bテストを想定した設計にすべき。

## 設計理由

### なぜこの構成にしたのか

「AIに仕事を奪われる不安」→「使う側になれ」という感情の逆転を軸に設計。
スクールLPで最も有効な構成は「不安 → 解決 → 証拠 → CTA」。
本LPではそこにAI特有の訴求軸を加え:

1. **哲学的大見出し**（AIは才能の差を埋める）で「このスクールは違う」と印象付ける
2. **使命表明**（なぜ教えているのか）で運営者のビジョンを先に見せる
3. **不安訴求**（使えないことがリスク）で現状維持の危険性を認識させる
4. **カリキュラム → 成果物**の順で「学べる内容」だけでなく「作れるもの」を強調

### なぜこのFVを採用したのか

「AIは、才能の差を埋める。」という大見出しを最大サイズ（82px）で打ち出した理由:
- AIスクール = テクニカル・難しいというイメージを「才能の差を埋める道具」に変換
- 凡才でも使えると伝えることでハードルを下げる
- ネイビー背景でなく白背景 + ネイビーテキストで、「清潔感 + 権威性」を両立
- 右カードはカリキュラムのプレビューとして機能 → スクロール前に学習内容を暗示

### なぜこのCTAを採用したのか

「無料相談」単一CTAにした理由:
- スクール系は「まず相談」が最も低ハードルな最初のアクション
- 「無料体験」と「無料相談」の2択は迷いを生む → 相談1本に絞ることで意思決定を簡単化
- 固定フッターCTA + ヘッダーCTA + フォームの3点展開で取りこぼしを防ぐ

### 競合と比較した差別化要素

| 差別化軸 | 内容 |
|---------|------|
| 「才能の差を埋める」 | 他スクールの「AI×副業収益化」「資格取得」とは異なる哲学的ポジショニング |
| 「成果物まで作る」 | 知識習得だけでなく実際のアウトプット重視 |
| 「5STEPカリキュラム」 | 段階的な習得プロセスの可視化 |
| 比較セクション | 他スクールとの差を明示（誠実さ → 信頼）|

### 現在ならどう改善するか

1. **FVにビジュアル強化**: 右カードに実際のAIツール使用画面スクリーンショット（FV-022パターン）
2. **受講生の顔写真**: Before/Afterに顔写真付き体験談（現在テキストのみ）
3. **料金セクション追加**: スクール系でFAIL-016（価格非開示）に当たる → 料金目安の明示
4. **FAQ追加**: 「どんな人が向いているか」「Mac/Win対応？」「受講期間は？」等
5. **LLMO差別化追加**: FV-028パターンを使った「AIを活用したLP/コンテンツ制作への展開」セクション

## 強み

| 強み | 具体的な実装 |
|------|------------|
| コピーの哲学性 | 「AIは才能の差を埋める」は他スクールにない切り口 |
| 構成の論理性 | 不安→解決→証拠→CTA の王道設計を忠実に実装 |
| ダーク系カラー設計 | ネイビー×ブルーがAI/テック感を演出しつつ清潔感を維持 |
| 比較セクション | 競合との比較が信頼性を高める（誠実なポジショニング）|

## 改善余地

| 改善余地 | 影響度 | 対処案 |
|---------|--------|-------|
| 料金情報が未掲載 | 高 | 料金目安 or「個別見積もり」の明示 |
| 受講生の顔写真なし | 中 | Before/Afterに写真付き体験談追加 |
| FAQセクション未実装 | 中 | 「自分向き？」「期間は？」等5問追加 |
| FVビジュアルが弱い | 低 | カードをAIツール使用UIに変更 |

## Learning Reason

**保存理由**: AIスクールLPの原型設計。「不安訴求→解決→カリキュラム→CTA」という構成と、哲学的大見出しによる差別化ポジショニングが確立できた。  
**再利用価値**: AI・テック教育系のLP設計において「AIは難しい→誰でも使える道具」という認知転換コピーの型として転用可能。  
**次回同業種への教訓**: スクール系で「料金」を明示しないと問い合わせの質が下がる。次回は料金目安セクション（STEP8に相当）を必ず追加すること。

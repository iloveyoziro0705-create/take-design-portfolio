## 目的

来店予約・宴会予約のCV獲得。「料理の魅力と雰囲気」を写真と感情語コピーで伝え、「今夜来たい」という即時来店欲求を喚起する。

---

## ターゲット

- 25〜50歳 / 会社員・グループ
- 「職場の仲間と飲みに行く店を探している」
- 「特別な日の食事場所を探している（誕生日・記念日等）」
- 「宴会・歓送迎会の幹事で店を探している」

## 使用したFV

**FV-026 + FV-025 複合型**: 感情語コピー + 老舗こだわり訴求

**来店誘導型（居酒屋・カジュアル系）**:
- コピー: 「今夜は、ここで語ろう」
- サブコピー: 「○○産○○を、○○年続く秘伝のタレで。あの頃の仲間と、また来たくなる場所。」
- 背景: 料理ドアップ写真 + 薄い暗幕オーバーレイ

**老舗・高級系**:
- タグライン: 「創業○○年 / ○○産○○食材直送」
- コピー: 「素材の声を聞き、料理に命を吹き込む」

## 実装した構成

### 来店誘導型（ラーメン・焼肉・カジュアル系）
```
1. FV — 料理写真フルスクリーン + 感情語コピー + 地図/電話CTA
2. 看板メニュー（3〜4品）— 料理写真 + 食材こだわり + 価格
3. 店のこだわり — 産地・調理法・シェフの哲学
4. 店内雰囲気写真ギャラリー
5. お客様の声（Googleマップ等から引用）
6. アクセス — Googleマップ埋め込み + 営業時間 + 定休日
7. 最終CTA — 地図 + 電話の2択（来店誘導）
```

### 予約型（高級和食・イタリアン・宴会対応）
```
1. FV — 料理/内装写真 + 老舗こだわりコピー + 予約CTA
2. コンセプト + シェフ・産地こだわり（左右交互レイアウト）
3. おすすめコース・メニュー紹介（価格明示）
4. 宴会プラン一覧（CTA-016）
5. 個室・設備紹介
6. メディア掲載・受賞歴
7. お客様の声
8. アクセス情報
9. 最終CTA — ネット予約 + 電話の2択
```

## 設計上の核心ポイント

### 飲食LPの業態別CTA設計（IND-002の核心）
```
業態によってCVアクションが根本的に異なる:

来店誘導型（ラーメン・焼肉・カウンター系）:
→ 「予約ボタン」は必要なし（「来てください」が正解）
→ 地図ボタン + 電話ボタンの2択が最適解（CTA-010）
→ 「予約ボタンなし = 欠陥」ではなく「業態に最適化されたCTA」

予約誘導型（高級・コース料理・宴会対応）:
→ ネット予約ボタン + 電話の2択
→ ホットペッパーグルメ等の外部予約サイトへの誘導でOK

宴会専用（歓送迎会・忘新年会向け）:
→ 宴会コース料金の明示（CTA-016）が必須
→ 「人数・日時・コース」の相談フォームがあるとCV率UP
```

### 「食欲」を生む写真設計
- FVの料理写真は「ドアップ・スチーム・ツヤ感」が最重要
- 背景が暗い方が料理の色が映える（黒皿・木目・鉄板など）
- 1800×1200px以上のHD写真を必ず用意（スマホのカメラNG）

### 価格透明性の重要性
- 「時価」表示は離脱率UP → 価格レンジを必ず明示
- 「お1人様○,○○○円〜」のように下限価格を示すと安心感

---

## 改善点

1. 「Googleマップ口コミ」の埋め込みまたはスクリーンショット引用で第三者評価を可視化
2. 「季節のおすすめメニュー」の更新が必要（静的LPだと季節感が消える）
3. インスタグラム連携（料理写真のUGC活用）でリアルタイム感を演出

---

## 次回に活かすこと

1. 飲食LPのCTAは「業態の予約文化」に合わせること（予約不要店に予約ボタンは邪魔）
2. 料理写真のクオリティがLP全体のクオリティを決める（写真がなければLPは作れない）
3. 「産地・生産者名・調理歴史」は競合差別化の核心 — 「なぜここで食べる必要があるか」の回答

---

## 再利用できる知識

1. **業態別CTA設計の原則**: 来店型/予約型/宴会型でCTAが異なるという設計思想はサービス業全般に応用可能（美容: 予約 / 医療: 3チャンネル / 飲食: 業態別）
2. **感情語コピーの有効性**: 「今夜、ここで笑おう」「あの頃の仲間と」のような感情語コピーは飲食だけでなく採用・スクール系でも「帰属感・共感」を生む
3. **産地×歴史の信頼性訴求**: 食材産地の透明性は飲食だけでなく、スキンケア（原材料）・健康食品・農産物直送など「素材の品質」が差別化軸の業種に応用可能

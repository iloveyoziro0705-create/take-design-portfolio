## 設計理由

### なぜこの構成にしたのか

「感情 → 視覚的欲求 → 信頼 → 行動」というレストランLPの王道設計。
飲食LPで最重要なのはビジュアルによる「行きたい気持ちの喚起」であり:

1. **感情語FV**（"その一皿が、特別な夜を完成させる。"）で「非日常体験」を先に予感させる
2. **Concept セクション**（"思い出が生まれる場所"）でレストランのブランド哲学を伝える
3. **ギャラリー（Interior）** で空間の非日常感を視覚的に確認させる
4. **こだわり3点（Story）** で料理・技術・空間の各ベネフィットを明示
5. **口コミ → アクセス → CTA** の順で「信頼 → 場所確認 → 予約」へ誘導

最小構成LP（9セクション）でも漏れなく伝えられるよう、各セクションの情報密度を高く設定。

### なぜこのFVを採用したのか

Unsplash全画面写真 + ダーク系カラーを選んだ理由:
- 料理の「非日常感」はテキストより写真一枚の方が瞬時に伝わる
- rgba(0,0,0,.45)〜.75のグラデーションオーバーレイで写真の視認性を保ちながら読みやすさを確保
- ゴールド（#d8a76a）のCTAボタンが暗い背景に映え、視線誘導として機能する
- `clamp(46px, 8vw, 92px)` の超大見出しが「その一皿」の存在感を演出

### なぜこのCTAを採用したのか

「WEB予約する」単一CTAにした理由:
- 飲食LPのCVアクションは予約の1点のみ → 迷わせない設計
- Hero（最上部）と最終セクション（最下部）の2箇所に配置してスクロールのどの位置でも拾える
- 最終CTAセクションも背景写真付きで「予約する場所感」を演出、CTAボタンへの視線誘導を強化

### 競合と比較した差別化要素

| 差別化軸 | 内容 |
|---------|------|
| ダーク高級感デザイン | 白基調の飲食LPと異なりブラック×ゴールドで「大人のレストラン」を演出 |
| 感情語コピー「その一皿が、特別な夜を完成させる。」| 料理の説明ではなく「夜の体験」を売る訴求 |
| Seriff + Noto Sans JP の混在 | 本文はサンセリフ、見出しはセリフ体で格調あるタイポグラフィ |
| 記念日・デート・女子会への3ターゲット訴求 | 単一ターゲットではなくシーン訴求でリーチ拡大 |

### 現在ならどう改善するか

1. **ヘッダー追加**: 現在ヘッダーなし → 固定ナビ + 予約ボタンCTAで常時予約口を確保
2. **実際のメニュー写真**: Unsplash写真から実際の料理写真に差し替えで信頼性大幅向上
3. **オンライン予約フォーム統合**: CTA-016（宴会予約フォーム）実装で外部遷移なし予約完結
4. **インスタグラムフィード埋め込み**: 料理写真の継続的な更新をLP内で表現
5. **シェフプロフィールセクション追加**: 「誰が作っているか」が高級レストランへの信頼に直結
6. **Limited感の演出**: 「今週の特別コース」「本日の仕入れ食材」など季節感・限定感を追加
7. **口コミ数の拡充**: 3件→最低10件以上のカルーセルで「多くの人に支持」を演出

## Learning Reason

**保存理由**: 飲食系LPのダーク高級感デザインの原型。ダーク×ゴールドの配色（`#0f0f0f` × `#d8a76a`）は飲食・バー・高級和食系で標準採用できるトークンとして確立。  
**再利用価値**: 最小構成LP（385行）の設計モデル。「感情語FV → コンセプト → 料理 → 空間 → 口コミ → 予約」という飲食LP必須フローを最小行数で実装した参照ファイルとして活用可能。  
**次回同業種への教訓**: 飲食LPで最も重要なのは実際の料理・店舗写真。Unsplash素材での納品はポートフォリオとして成立するが、実案件では必ず実写素材を確保すること。ヘッダーCTAと固定フッターCTAは飲食系でも必須。

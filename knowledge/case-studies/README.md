# Case Studies

## 概要

実際に制作・納品した案件の記録を蓄積するフォルダ。
LP分析（`learning-history/`）とは異なり、**自社制作案件の実記録**を保存する。

---

## ファイル命名規則

```
YYYY-MM-DD-[業種]-[プロジェクト名].md
```

### 例
```
2026-06-20-beauty-salon-lp.md
2026-06-20-recruit-lp.md
2026-06-20-clinic-homepage.md
```

---

## 使用ルール

1. 案件完了後、必ず `case-study-template.md` を使って記録する
2. 採用したPattern-LP-XXX / FV-XXX / CTA-XXX を必ず記録する
3. クライアント評価・成果は後日追記で構わない
4. 感想・主観表現は禁止。構造的事実のみ記録する

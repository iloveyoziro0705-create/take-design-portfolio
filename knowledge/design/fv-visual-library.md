# FV Visual Library — ファーストビュービジュアル辞典

**バージョン**: 1.0  
**作成日**: 2026-06-24  
**管理ルール**: `knowledge/governance.md` 参照  
**目的**: FVで使用するビジュアルレイアウト・構図・人物配置を体系化し、FVのビジュアル品質を再現可能にする

---

## 利用ルール

1. FV設計時はこのライブラリを参照してから画像選定・生成を行う
2. `fv-copy-library.md` の FV-COPY-XXX と必ずセットで選定する
3. FV-VISUAL と FV-COPY の感情トリガー（TRIG-XXX）が一致していることを確認する
4. 業種・コピータイプに合わせた構図を選ぶ

# FV-VISUAL-001: 顔アップ型

**コピータイプ相性**: FV-COPY-001（悩み訴求）/ FV-COPY-002（理想訴求）/ FV-COPY-006（権威訴求）  
**向き業種**: 美容・エステ・フェイシャル・医療  
**利用条件**: 高品質な人物写真がある / 施術後のクオリティが伝わる写真  
**主感情**: TRIG-001（悩み共感）/ TRIG-002（不安除去）

### 構図設計

```
[FVレイアウト — デスクトップ]
┌─────────────────────────────────┐
│  [コピーエリア: 左40-50%]  │ 顔写 │
│  バッジ                    │ アッ │
│  H1（大）                  │ プ写 │
│  サブコピー                │ 真   │
│  証拠バー                  │      │
│  CTAボタン                 │      │
└─────────────────────────────────┘

[FVレイアウト — モバイル]
┌──────────────┐
│  顔写真（上） │
│  コピー（下） │
│  CTAボタン    │
└──────────────┘
```

### 人物ビジュアル仕様

| 項目 | 推奨値 |
|------|-------|
| フレーミング | 胸元〜頭頂（顔が画面右半分を占める） |
| 視線方向 | カメラ正面 or やや左（コピー方向） |
| 表情 | 柔らかい笑顔 / 施術中は穏やかな目閉じも可 |
| 背景 | ぼかし（ボケ量多め）/ サロン内装がうっすら見える |
| 照明 | 左側からのキー光（顔の右側に陰影） |
| 肌 | リタッチあり・均一・光沢感 |

### コピー配置ルール
- H1: 視覚的中心・40〜48px（PC）/ 28〜32px（SP）
- バッジ: H1の上8px / 11〜12px フォント
- サブコピー: H1の下16px / 14〜15px / 2〜3行
- 証拠バー: サブコピーの下24px / 横並び3項目
- CTA: 証拠バーの下24px / 大ボタン（min-height: 52px）

### コード実装参考
```css
.fv { display: grid; grid-template-columns: 1fr 1fr; min-height: 100svh; align-items: center; }
.fv-text { padding: 0 48px 0 80px; }
.fv-image { height: 100%; object-fit: cover; }
```

### Learning Reason
**保存理由**: 美容・エステLPでFV改善時に顔アップ型が最も直帰率低下に寄与。視線一致（カメラ正面）が信頼感を即形成することを実案件で確認。  
**再利用価値**: 美容・医療系FV全般で最も使用頻度が高い構図。

# FV-VISUAL-002: 上半身型（正面）

**コピータイプ相性**: FV-COPY-002（理想訴求）/ FV-COPY-005（実績訴求）/ FV-COPY-009（ストーリー）  
**向き業種**: 美容・採用・スクール・整体  
**主感情**: TRIG-002（不安除去）/ TRIG-003（理想投影）

### 構図設計

```
[デスクトップ — 人物右配置パターン]
┌─────────────────────────────────┐
│ コピーエリア(左45%) │ 人物(右55%) │
│ バッジ             │             │
│ H1                 │  上半身     │
│ サブコピー         │  正面向き   │
│ 実績バー           │  全身〜腰上 │
│ CTAボタン          │             │
└─────────────────────────────────┘
```

### 人物ビジュアル仕様

| 項目 | 推奨値 |
|------|-------|
| フレーミング | 腰〜頭頂（上半身全体が見える） |
| 視線 | 正面 or わずかに左（コピー方向） |
| 表情 | 明るい笑顔・自信ある表情 |
| 服装 | 業種ユニフォーム or 清潔感のある私服 |
| ポーズ | 立ち姿 / 腕を自然に下ろす / 軽いポーズOK |
| 背景 | サロン・オフィス・スタジオ（ぼかし） |

### 差分ポイント vs FV-VISUAL-001
- 顔アップより「職種・サービス」の雰囲気が伝わりやすい
- 採用系では「この人と一緒に働きたい」感が出る
- ユニフォームが見えるので信頼感形成に有効

# FV-VISUAL-003: ライフスタイル全身型

**コピータイプ相性**: FV-COPY-003（比較訴求）/ FV-COPY-011（採用悩み）/ FV-COPY-018（ストーリー）  
**向き業種**: 採用・スクール・フィットネス・保険  
**主感情**: TRIG-003（理想投影）

### 構図設計

```
[フルスクリーン背景 + オーバーレイコピー]
┌──────────────────────────────────┐
│  [背景: ライフスタイル写真]        │
│  薄い暗オーバーレイ (rgba 0,0,0,.4)│
│                                    │
│     バッジ                         │
│     H1（白文字）                   │
│     サブコピー（白文字）           │
│     CTAボタン                      │
└──────────────────────────────────┘
```

### ビジュアル仕様

| 項目 | 推奨値 |
|------|-------|
| シーン | オフィス・公園・カフェ・屋外など「理想の日常」 |
| 人物サイズ | 全身〜腰上 / 画面の50〜70%を占める |
| 視線 | 横向き or 斜め（自然な動作感） |
| 動き | 歩く・笑う・話す・パソコンに向かう |
| オーバーレイ | `rgba(0,0,0,.35〜.55)` 写真の明度による |
| コピー位置 | 左寄せ or 中央 |

### 注意点
- テキスト可読性のためオーバーレイ必須
- 人物が画面端に位置するとバランスが崩れやすい
- フルスクリーンのため高解像度写真が必須（最低1440px幅）

# FV-VISUAL-004: ビフォーアフター型

**コピータイプ相性**: FV-COPY-004（損失回避）/ FV-COPY-005（実績訴求）  
**向き業種**: 美容・エステ・医療・ダイエット  
**主感情**: TRIG-010（変化実感）/ TRIG-005（損失回避）

### 構図設計（パターンA: 左右分割）

```
┌────────────────────────────────────┐
│ BEFORE(左)    │ AFTER(右)           │
│ ─────────     │ ─────────           │
│ 暗め・困り顔  │ 明るい・笑顔        │
│               │                     │
│ コピーオーバーレイ（下部）          │
│  H1 + CTA                          │
└────────────────────────────────────┘
```

### 構図設計（パターンB: 上下分割、モバイル優先）

```
┌──────────────┐
│ BEFORE（上） │
│ ────────     │
│ AFTER（下）  │
│              │
│ H1 + CTA    │
└──────────────┘
```

### ビジュアル仕様

| 項目 | Before | After |
|------|--------|-------|
| 照明 | フラット・やや暗め | 柔らか・明るめ |
| 色温度 | クール（4,500K） | ウォーム（5,500K） |
| 表情 | 困り顔・伏し目・無表情 | 笑顔・喜び・前向き |
| 背景 | グレー・白 | 白・明るいベージュ |
| フレーミング | 問題部位が見えるクロップ | 問題解決部位 + 表情 |

### 注意点（薬機法・景品表示法）
- 「治る」「必ず」「100%」表現禁止
- 個人差がある旨を必ず明記
- 過度なレタッチ禁止

## FV-VISUAL選定フローチャート

```
写真素材の有無
    ├─ あり
    │   ├─ 人物写真あり
    │   │   ├─ 顔が見える → FV-VISUAL-001（顔アップ）or 002（上半身）
    │   │   ├─ ライフスタイル → FV-VISUAL-003
    │   │   ├─ BA写真あり → FV-VISUAL-004
    │   │   └─ 権威者写真 → FV-VISUAL-005
    │   └─ 空間・料理写真
    │       ├─ 飲食 → FV-VISUAL-006（空間）or 007（料理）
    │       └─ サロン内装 → FV-VISUAL-006
    └─ なし
        ├─ SaaS/BtoB → FV-VISUAL-008（UI型）or 011（グラデ）
        ├─ スクール → FV-VISUAL-011（グラデ）
        ├─ 採用 → FV-VISUAL-011（グラデ）
        └─ 診断型訴求 → FV-VISUAL-012
```

# Industry Visual Patterns — 業種別ビジュアルパターン辞典

**バージョン**: 1.0  
**作成日**: 2026-06-24  
**管理ルール**: `knowledge/governance.md` 参照  
**目的**: 業種ごとの「正解ビジュアル」を体系化し、CVRに直結するビジュアル選定を再現可能にする

---

## 利用ルール

1. LP制作時は `industry-patterns.md`（IND-XXX）と合わせて本ファイルを参照する
2. 各業種の「高CVパターン」を最優先で採用する
3. 「NGパターン」は理由を理解した上で回避する
4. `fv-visual-library.md` の FV-VISUAL-XXX と必ず紐付けて選定する

---

# IND-V-001: 美容・サロン系ビジュアルパターン

**対象業種**: 美容院・ネイルサロン・エステ・脱毛・まつ毛・フェイシャル

---

## 推奨ビジュアル方針

### ターゲット心理とビジュアルの関係

| ターゲット心理 | 必要なビジュアルメッセージ | 対応パターン |
|-------------|----------------------|-----------|
| 「効果があるか不安」 | Before/Afterで実績を見せる | FV-VISUAL-004 |
| 「勧誘が怖い」 | 安心感のある笑顔・清潔な空間 | FV-VISUAL-001, 006 |
| 「自分に合うか不安」 | 同年代・同悩みの人物を見せる | FV-VISUAL-001 |
| 「行ってみたい」 | 憧れの空間・理想の肌を見せる | FV-VISUAL-006 |

---

## 推奨写真仕様

### 人物写真

| 項目 | 推奨 |
|------|-----|
| 性別・年代 | 30〜45歳女性（ターゲットと一致させる） |
| 表情 | 施術後の穏やかな満足感 / リラックス / 笑顔 |
| 肌感 | 透明感・均一な肌色・自然なツヤ（過剰レタッチ禁止） |
| 服装 | 白・ベージュ・淡いピンク / 清潔感優先 |
| ヘア | 清潔感・自然なスタイル（派手すぎない） |

### 空間写真

| 項目 | 推奨 |
|------|-----|
| 照明 | 間接照明・暖色（3,000〜3,500K）/ 自然光OK |
| 清潔感 | 散乱物ゼロ / 生花・グリーン配置 |
| 色調 | ホワイト + ベージュ + ゴールドorピンク差し色 |
| 個室感 | 仕切り・カーテン・プライベート感が見える |

### 施術写真

| 項目 | 推奨 |
|------|-----|
| 手元 | 清潔な手（ナチュラルネイルのみ） |
| 光 | やわらかい暖色 / 機器の光は自然に |
| 構図 | 施術者の手 + 顧客の肌がわかるアングル |

---

## 推奨カラーパレット（ビジュアル色調）

| ブランドトーン | 主色 | 差し色 | 背景色 |
|------------|------|-------|-------|
| ラグジュアリー | ゴールド #C8A56A | チャコール #3C2A27 | クリーム #FAF8F3 |
| ナチュラル | テラコッタ #B5936C | オリーブ #6B8C5F | アイボリー #F9F5F0 |
| クリーン・モダン | ローズ #C4878C | ホワイト #FFFFFF | ペールピンク #FBF0F0 |
| ポップ | コーラル #E05050 | ホワイト | ライトピンク #FFF0F3 |

---

## 高CVビジュアルパターン

### パターンA（最高CVR）: 悩み訴求 × 顔アップ × ビフォーアフター
```
FV: 顔アップ型（FV-VISUAL-001）
  → ターゲットと同年代の女性・悩みが伝わる表情
  → 左コピー「悩み訴求H1」/ 右に顔クロップ

中段: ビフォーアフター（FV-VISUAL-004）
  → 3事例以上 / 部位別（毛穴・くすみ・たるみ）

信頼: 施術者の手元クローズアップ + 資格バッジ
```

### パターンB（高CVR）: 空間型 × 安心訴求
```
FV: 個室空間（FV-VISUAL-006）
  → 完全個室・清潔感・暖かい照明
  → 「完全個室」「勧誘なし」テキストオーバーレイ

スタッフ: 権威者ポートレート（FV-VISUAL-005）
  → 施術者の資格・経歴と顔写真

実績: 数字型（FV-VISUAL-010）
  → 口コミ数・施術件数・リピート率
```

### パターンC（ブランディング型）: 憧れ × ラグジュアリー
```
FV: ダーク高級感空間 or 逆光人物
  → 「この世界観に入りたい」という憧れ喚起

素材: パリ産コスメ・高級機器の物撮り
カラー: ダーク × ゴールド
コピー: 短く・詩的（機能より感情）
```

---

## NGパターン（美容）

| NGパターン | 理由 | 代替 |
|---------|------|------|
| 過度なレタッチ肌 | 現実離れして信頼を損なう | 自然なツヤ感に留める |
| 無表情・眼が笑っていない | 冷たさ・怖さを演出 | 柔らかな笑顔 |
| ゴチャゴチャした背景 | 清潔感を損なう | ぼかし or 単色背景 |
| 男性スタッフの写真のみ | 女性ターゲットへの安心感低下 | 女性スタッフを必ず含める |
| 白衣だけの「医療っぽい」演出 | 温かみがなくなる | ユニフォーム or 私服ナチュラル |
| 素材写真の顔が外国人のみ | ターゲットとの乖離 | 日本人女性を使用する |

# IND-V-002: 採用系ビジュアルパターン

**対象業種**: 介護・保育・飲食・建設・IT・医療職採用

---

## 推奨ビジュアル方針

### ターゲット心理とビジュアルの関係

| ターゲット心理 | 必要なビジュアルメッセージ | 対応パターン |
|-------------|----------------------|-----------|
| 「職場の雰囲気が不安」 | スタッフが楽しそうに働く写真 | FV-VISUAL-003 |
| 「本当に待遇が良いの？」 | リアルな現場・具体的な数字 | FV-VISUAL-010 |
| 「自分に合う仕事か」 | 同世代・同境遇のスタッフ事例 | FV-VISUAL-002 |
| 「転職のリスクが怖い」 | 安心できる顔・穏やかな雰囲気 | FV-VISUAL-001 |

---

## 推奨写真仕様

### 人物写真（スタッフ）

| 項目 | 推奨 |
|------|-----|
| 人数 | 個人 or 3〜5名グループ（多すぎない） |
| 表情 | 笑顔・充実感・自然体（演じていない感） |
| 年代 | ターゲット求職者と同年代 |
| 服装 | 実際のユニフォーム（リアリティが大事） |
| シーン | 実際の職場（演出OK・自然な動作） |

### 職場環境写真

| 項目 | 推奨 |
|------|-----|
| 明るさ | 明るい・開放感（暗い職場はNG） |
| 整理 | 整頓されている（散乱しない） |
| 設備 | 新しい機器・きれいなユニフォーム室など |

---

## 高CVビジュアルパターン

### パターンA: リアルスタッフ × 充実感演出
```
FV: ライフスタイル型（FV-VISUAL-003）
  → 職場でイキイキ働くスタッフの自然な姿
  → 「楽しそう」「自分もこうなれる」を視覚化

中段: スタッフ個別紹介
  → 顔写真 + コメント + 経歴（転職前後）

実績: 待遇数字（月収・残業時間・定着率）
```

### パターンB: 数字実績型 × 安心感演出
```
FV: 実績数字型（FV-VISUAL-010）
  → 定着率/残業時間/給与UP率を大きく表示

中段: グループ写真 + スタッフの声

差別化: 他社との比較表（FV-VISUAL-009）
```

### パターンC: ストーリー型 × Before/After
```
FV: 上半身型（FV-VISUAL-002）
  → 転職成功者のポートレート

ストーリー: 転職前（困った顔）→ 転職後（笑顔）
  → 具体的な変化（残業時間・収入・家族時間）

CTA: 「私もこうなれる」低ハードル導線
```

---

## NGパターン（採用）

| NGパターン | 理由 | 代替 |
|---------|------|------|
| 素材写真の外国人スタッフ | リアリティ低下・嘘くさい | 実際のスタッフ写真必須 |
| 暗い・疲れた表情のスタッフ | ネガティブイメージ直結 | 自然な笑顔・充実感 |
| 整理されていない職場写真 | 「汚い職場」印象 | 整理後に撮影 |
| キメすぎたスタジオ写真 | リアリティ不足・怪しさ | 職場での自然なシーン |
| 全員正面整列写真 | 硬い・怖い印象 | 自然な動作・会話シーン |

# IND-V-003: 医療・整骨院系ビジュアルパターン

**対象業種**: 整骨院・整体・クリニック・歯科・鍼灸

---

## 推奨ビジュアル方針

| ターゲット心理 | 必要なビジュアルメッセージ | 対応パターン |
|-------------|----------------------|-----------|
| 「本当に治るか」 | 改善実績・症例写真 | FV-VISUAL-004 |
| 「痛い施術は嫌」 | 穏やかな施術シーン | FV-VISUAL-006 |
| 「先生を信頼できるか」 | 院長権威・資格 | FV-VISUAL-005 |
| 「清潔か」 | 明るく清潔な院内写真 | FV-VISUAL-006 |

---

## 推奨写真仕様

### 院長・スタッフ写真

| 項目 | 推奨 |
|------|-----|
| 服装 | 白衣 or 機能的ユニフォーム（権威感） |
| 表情 | 穏やか・頼りになる・安心感 |
| 背景 | 清潔な院内 or 白 |
| 構図 | 正面〜斜め45°（上半身） |

### 施術シーン

| 項目 | 推奨 |
|------|-----|
| 患者 | リラックスした表情・目を閉じている |
| 施術者 | 集中・丁寧な手元 |
| 環境 | 清潔・整頓・明るい |
| 光 | 暖色・柔らかい |

### 院内写真

| 項目 | 推奨 |
|------|-----|
| 明るさ | 明るく清潔（暗い→怖い） |
| 設備 | 最新機器・きれいなベッド・整頓 |
| 動線 | 受付→施術室が想像できる構成 |

---

## 推奨カラーパレット

| ブランドトーン | 主色 | 副色 | 背景 |
|------------|------|-----|-----|
| 信頼・安心 | グリーン #2EB88A | ホワイト | #F0FDF4 |
| 医療・クリーン | ブルー #1A3A5C | ホワイト | #F0F4F8 |
| 温かみ・親しみ | オレンジ #F97316 | ホワイト | #FFF7ED |

---

## 高CVビジュアルパターン（医療）

### パターンA: 院長権威 × 施術実績
```
FV: 権威者ポートレート（FV-VISUAL-005）
  → 院長白衣・資格証 / 「〇年・〇件」表示

中段: 施術シーン（リラックスした患者 + 丁寧な手元）

実績: 症例改善数 / 口コミ評価（★4.9 / ○件）
```

### パターンB: 症状訴求 × 安心訴求
```
FV: グラデ背景（FV-VISUAL-011）
  → 「腰痛・肩こり・頭痛」テキスト主体

Before/After: 症状改善のグラフ or 患者の声写真

院内: 清潔な院内 + スタッフ笑顔写真
```

### パターンC: 比較型 × 根本治療訴求
```
FV: 比較型（FV-VISUAL-009）
  → 「一般的な施術院 vs 当院」の違いを視覚化

特徴: 独自メソッドの図解 or 動画

CTA: 「まず相談」低ハードル
```

---

## NGパターン（医療）

| NGパターン | 理由 |
|---------|------|
| 針・血・痛そうなシーン | 恐怖感 → 離脱 |
| 薬機法違反表現とセットの写真 | 法的リスク |
| 暗い・汚い院内写真 | 信頼喪失 |
| 笑顔ゼロの院長写真 | 怖さ・冷たさ |

# IND-V-004: 飲食系ビジュアルパターン

**対象業種**: レストラン・居酒屋・カフェ・イタリアン・和食

---

## 推奨ビジュアル方針

| 目的 | 必要なビジュアル | パターン |
|-----|--------------|---------|
| 「食べたい」を作る | 料理の美しいクローズアップ | FV-VISUAL-007 |
| 「行きたい」を作る | 雰囲気ある店内・空間 | FV-VISUAL-006 |
| 「信頼できる」を作る | シェフ・料理人の権威 | FV-VISUAL-005 |
| 「特別な日に」 | 記念日・デートシーン | FV-VISUAL-003 |

---

## 料理写真ルール（最重要）

### 必須品質基準

| 項目 | 基準 |
|------|-----|
| 解像度 | 最低2000px以上（FV使用時は4000px推奨） |
| シャープネス | メイン被写体にジャスピン |
| 光 | 自然光 or プロ照明（蛍光灯NG） |
| 演出 | 湯気・光沢・素材感を強調 |
| 背景 | 統一感（木目 or 大理石 or ダーク） |

### アングル別用途

| アングル | 適した料理 | 心理効果 |
|---------|---------|--------|
| 俯瞰（真上） | 丼・パスタ・ピザ | ボリューム感 |
| 斜め45° | コース料理・盛り合わせ | 立体感・高級感 |
| 水平（横） | バーガー・サンドイッチ | 断面美・食欲直撃 |
| クローズアップ | 肉・刺身・デザート | 素材感・鮮度訴求 |

---

## 推奨カラーパレット（飲食業態別）

| 業態 | 主色 | 差し色 | 背景 | 感情 |
|-----|-----|-------|-----|-----|
| 高級フレンチ | ゴールド #D4AF37 | ダーク #0A0A0A | ブラック | TRIG-004 |
| 居酒屋・和食 | ダークレッド #8B1A1A | ゴールド | ダーク | TRIG-004 |
| カフェ | ブラウン #6F4E37 | クリーム | ベージュ | TRIG-003 |
| イタリアン | テラコッタ #C25B32 | グリーン | ウォームホワイト | TRIG-004 |

---

## 高CVビジュアルパターン（飲食）

### パターンA: 料理クローズアップ型（食欲直撃）
```
FV: 料理クローズアップ（FV-VISUAL-007）
  → 最も自信のある一品をフルスクリーン
  → オーバーレイ + コピー: 「この一皿のために、また来る。」
  → CTAボタン: 「今すぐ予約する →」
```

### パターンB: 空間型（非日常・記念日訴求）
```
FV: 店舗空間型（FV-VISUAL-006）
  → 間接照明・食事シーン・カップルor家族
  → コピー: 「大切な人と、特別な時間を。」

シェフ: 権威者ポートレート（FV-VISUAL-005）
  → 経歴・受賞歴・コメント

料理: ギャラリー形式（5〜8枚）
```

### パターンC: ストーリー型（こだわり訴求）
```
FV: シェフのポートレート（FV-VISUAL-005）
  → 「この味が作りたくて、独立しました。」

素材: 産地写真 + 仕入れシーン
  → こだわりの可視化

料理: 完成品 + 素材のセット展示
```

---

## NGパターン（飲食）

| NGパターン | 理由 |
|---------|------|
| 暗い・ピンぼけの料理写真 | 食欲減退・信頼低下 |
| 蛍光灯下の料理写真 | 色が死ぬ・不味そうに見える |
| 料理なしのテキストのみFV | 飲食LPの最大のNG |
| 人が少ない（ガラガラ）店内写真 | 「流行っていない」印象 |
| 均一すぎるストック写真 | 「うちの料理ではない」と思われる |

# IND-V-005: スクール・教育系ビジュアルパターン

**対象業種**: プログラミング・デザイン・英語・資格・転職スクール

---

## 推奨ビジュアル方針

| ターゲット心理 | 必要なビジュアル | パターン |
|-------------|--------------|---------|
| 「自分でもできる？」 | 受講生の変化・成功事例 | FV-VISUAL-002 |
| 「講師は信頼できる？」 | 講師の経歴・権威 | FV-VISUAL-005 |
| 「どんな環境で学べる？」 | 学習空間・授業シーン | FV-VISUAL-003 |
| 「効果があるの？」 | 転職成功数・収入UP実績 | FV-VISUAL-010 |

---

## 推奨写真仕様

### 受講生写真
- 完全未経験からの変化が伝わる表情
- Before（困り顔・悩み顔）→ After（笑顔・自信）
- 実際の受講生を使用（素材写真より信頼度UP）
- 年代・背景をターゲットに合わせる

### 講師写真
- プロフェッショナルな服装（スーツ or カジュアルスマート）
- 背景: ホワイトボード・PC・書籍
- 知的で親しみやすい表情
- 会社ロゴ・受賞歴などを小道具で添える

### 学習環境写真
- 明るい・整頓・集中できる雰囲気
- PCや教材が見える
- 複数人学習 or 個別サポートシーン

---

## 推奨カラーパレット（スクール）

| ブランドトーン | 主色 | 副色 | 背景 |
|------------|------|-----|-----|
| テック・IT | インディゴ #6366F1 | シアン #0EA5E9 | ダーク #0F172A |
| ビジネス系 | ネイビー #1A3A5C | オレンジ #F97316 | ホワイト |
| 女性向け | コーラル #FF6B8A | パープル #A855F7 | ライト |
| オール向け | ブルー #3B82F6 | グリーン #22C55E | ホワイト |

---

## 高CVビジュアルパターン（スクール）

### パターンA: 受講生変化型（ストーリー）
```
FV: 受講生上半身（FV-VISUAL-002）
  → 「未経験→転職成功」の顔写真 + 一言コメント
  → 数字: 転職成功率○% / 平均収入UP○万円

中段: Before/After（FV-VISUAL-004）
  → 転職前後の年収・業種・生活の変化

実績: 転職先企業ロゴ一覧
```

### パターンB: 実績数字型 × グラデ背景
```
FV: グラデ背景（FV-VISUAL-011 / インディゴ系）
  → 「転職成功率91% / 受講生3,200名」数字大表示

講師: 権威者ポートレート（FV-VISUAL-005）
  → 現役エンジニア・業界経験年数

無料体験: 「まず試す」CTA強調
```

### パターンC: 診断型インタラクション
```
FV: 診断型（FV-VISUAL-012）
  → 「プログラミング向きか3分で診断」

結果: 診断結果ページ → スクール詳細誘導

CTA: 無料体験申し込み
```

---

## NGパターン（スクール）

| NGパターン | 理由 |
|---------|------|
| 素材写真のモデルを「受講生」と称する | 信頼損失・景品表示法リスク |
| 過剰な収入保証イメージ（「月収100万確実」） | ステマ・景品表示法リスク |
| 全員がスーツ姿の堅すぎる写真 | 親しみにくさ・壁感 |
| ロゴだけで顔が見えない | 人間味・信頼感の欠如 |

# IND-V-006: SaaS・BtoB系ビジュアルパターン

**対象業種**: 業務効率化ツール・SaaS・BtoB・DX支援

---

## 推奨ビジュアル方針

| ターゲット心理 | 必要なビジュアル | パターン |
|-------------|--------------|---------|
| 「使いこなせるか不安」 | 直感的なUI・簡単操作感 | FV-VISUAL-008 |
| 「本当に効果がある？」 | 導入実績・数字 | FV-VISUAL-010 |
| 「競合との違いは？」 | 比較表・機能一覧 | FV-VISUAL-009 |
| 「自社に合うか」 | 業種別導入事例 | FV-VISUAL-002 |

---

## 推奨写真仕様

### UI・スクリーンショット
- 実際の画面（or 高品質モックアップ）を使用
- MacBook Pro 斜め配置が最もCVR高い
- ダッシュボード・レポート画面がわかりやすい
- グラフ・数字が見える（効果の可視化）

### ビジネス人物
- スーツ or ビジネスカジュアル（業種に合わせる）
- 会議・PC作業・コラボシーン
- 多様性（年代・性別）を意識
- 明るいオフィス環境

### 導入事例写真
- 実際のクライアント担当者（許可を得て）
- 会社ロゴ + 実績数字のセット

## 高CVビジュアルパターン（SaaS）

### パターンA: UIモック × 課題訴求（最多採用）
```
FV: UI型（FV-VISUAL-008）
  → 左コピー「毎月○時間削減できます」
  → 右MacBook斜め: ダッシュボード画面
  → 背景: ダークグラデ（プロフェッショナル感）

実績: 導入企業ロゴ一覧 + 数字
  → 「4,200社導入 / 継続率97%」

CTA: 無料トライアル or デモ予約
```

### パターンB: 実績数字 × 導入事例
```
FV: 実績数字型（FV-VISUAL-010）
  → 「導入後、月次業務68時間→12時間」数字大表示

事例: 企業名 + 担当者写真 + コメント
  → 3〜5社の具体的な変化

比較: 導入前後の比較（FV-VISUAL-009）
```

### パターンC: 比較型 × 乗り換え訴求
```
FV: 比較型（FV-VISUAL-009）
  → 「Excelで管理 vs ○○」の機能比較表

ROI: 投資対効果の計算機 or 数字
  → 「年間○万円削減シミュレーション」

CTA: 30日無料トライアル
```

## NGパターン（SaaS）

| NGパターン | 理由 |
|---------|------|
| UIが古い・汚いスクショ | 「使いにくそう」第一印象 |
| 曖昧な効果表現（「便利です」） | BtoBは数字で判断する |
| 人物写真のみ・UIなし | 「何ができるか」が伝わらない |
| 料金非開示のみCTA | BtoB担当者は予算確認が必須 |

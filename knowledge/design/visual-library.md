# Visual Library — ビジュアル辞典

**バージョン**: 1.0  
**作成日**: 2026-06-24  
**管理ルール**: `knowledge/governance.md` 参照（ビジュアル設計の定義はここに集約）  
**目的**: LP/HP制作時に「何を見せるか」を再現可能なナレッジとして保存し、ビジュアル品質を標準化する

---

## 利用ルール

1. LP/HP制作時は画像選定・生成・撮影指示の前にこのライブラリを参照する
2. 既存パターン（VISUAL-XXX）の流用・カスタマイズを優先する
3. 感情トリガー（TRIG-XXX）と必ず紐付けて選定する
4. 新パターンを発見したら末尾に追記（Learning Reason 必須）

# 人物ビジュアル（VISUAL-001〜012）

## VISUAL-001: 笑顔女性ポートレート（安心・親しみ）

**カテゴリ**: 人物  
**利用業種**: 美容サロン・エステ・医療・採用  
**利用シーン**: FVヒーロー・スタッフ紹介・ビフォーアフター後  
**期待効果**: 不安除去・親近感形成・クリック率向上  
**感情トリガー**: TRIG-002（不安除去）/ TRIG-010（変化実感）

### 推奨仕様

| 項目 | 推奨値 |
|------|------|
| 構図 | 上半身〜胸元 / 正面〜やや斜め |
| 視線 | カメラ正面（視線一致） |
| 表情 | 自然な笑顔（歯が見える / 目が笑っている） |
| 背景 | 白・クリーム・ナチュラルテクスチャ |
| ライティング | 自然光 or ソフトボックス（影なし） |
| 色温度 | 5,000〜5,500K（温かいニュートラル） |
| 服装 | 白・ベージュ・淡いピンク / 清潔感優先 |
| 余白 | 左右どちらかにコピーエリアを確保（左人物・右コピー推奨） |

### NG事項
- 引きつった笑顔・眼が笑っていない
- 過度なメイク・極端なフォトショップ加工
- 背景が雑然としている
- 暗い・コントラストが強すぎる

### Stable Diffusion / Midjourney プロンプト例
```
portrait of japanese woman, warm smile, natural light, white background,
upper body, clean and professional, soft bokeh, 50mm lens, 5500K,
beauty salon aesthetic, high quality --ar 3:4
```

## VISUAL-004: ビフォーアフター人物型

**カテゴリ**: 人物  
**利用業種**: 美容・ダイエット・整体・医療  
**利用シーン**: Before/Afterセクション・FV証拠要素  
**感情トリガー**: TRIG-010（変化実感）/ TRIG-005（損失回避）

### 推奨仕様

| 項目 | Before | After |
|------|--------|-------|
| ライティング | やや暗め・フラット | 明るめ・柔らか |
| 表情 | 困り顔・伏し目 | 笑顔・前向き |
| 背景 | 白・グレー | 白・明るいベージュ |
| 色温度 | クールトーン（4,000K） | ウォームトーン（5,500K） |
| 構図 | 左配置 | 右配置 |

### 注意事項（薬機法対応）
- 「○○が治る」という表現とセットで使用しない
- 個人の感想であることを明記する
- 過度な加工は景品表示法違反リスクあり

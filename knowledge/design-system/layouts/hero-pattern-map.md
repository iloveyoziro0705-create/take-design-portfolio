# Hero Pattern Map

**バージョン**: 1.0
**作成日**: 2026-06-28
**用途**: 業種 × ターゲット × CV目的 から最適パターンを即選択するナビゲーションマップ

---

## 使い方

1. **縦軸**（業種）でブロックを探す
2. **横軸**（CV目的）で列を絞る
3. 該当セルの`HERO-XXX`を`hero-layout-library.md`で確認
4. スコアと業種分類で最終決定

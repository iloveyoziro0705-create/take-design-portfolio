**バージョン**: 1.0

**対象セクション**: Feature / サービス紹介 / 特徴 / 選ばれる理由 / メリット / 強み

---

## このライブラリの使い方

1. `feature-pattern-map.md` の業種×目的マップで候補を絞る
2. `feature-ranking.md` でCV期待値・スマホ適性・汎用性を確認
3. 本ファイルで候補パターンの詳細を確認
4. FEATURE-XXX を選定し、production-rules.md の STEP 3.5 に記録する
5. **ゼロからFeatureを設計することは禁止**

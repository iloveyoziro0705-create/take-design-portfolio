# Feature Pattern Map

**バージョン**: 1.0
**作成日**: 2026-06-28
**用途**: 業種 × CV目的 × ターゲット からFeatureパターンを素早く選定する

---

## 使い方

1. **業種** からパターン候補を絞る
2. **CV目的** で候補を絞り込む
3. `feature-ranking.md` でスコア確認
4. `feature-layout-library.md` で詳細確認
5. 採用パターンを記録して制作開始

## 【最重要】スクール目的の先行判定（GEN-006-007から確立）

分析を始める前に必ずスクールの「目的」を判定すること。目的によって信頼性訴求軸が根本的に異なる。

```
[スクール目的判定フロー]
① キャリア転職型 → 転職成功率/就職率/月収UP数字が必須
② 習い事・情緒型 → 楽しさ/コミュニティ/体験価値が信頼性の核
③ 資格合格型    → 合格率/講師資格/試験特化データが必須
④ 英会話・スキルUP型 → 満足度%/継続率/体験談が有効
⑤ 女性専用型    → 専用感/継続率/在宅実績が差別化軸
⑥ 投資・金融教育型 → 経済的自由フレーム/講師権威/3軸No.1

⚠️ 転職型に「楽しさのみ」はFAIL / 習い事型に「転職率」は的外れ
```

## スクール系LP分析の核心原則（GEN-006 11件完了版）

### 1. FV評価の最重要ポイント
- **「受講後に何が得られるか」が数字で訴求されているか**
  - 良い例: 「3ヶ月で転職成功率92%」「TOEIC 200点アップ保証」「合格率95%」「60日で話せる」
  - 悪い例: スクール名ロゴ+美しいビジュアルのみ / 英語コピー「Design Your Future.」（GEN-006-004反面教師）

### 2. 信頼性評価の三点セット
```
卒業生の声 = 顔写真 + 職業（転職先・変化後） + 具体的変化（「○ヶ月で〜」）
```
- 三点すべて揃っていれば信頼性+3点以上
- 匿名・顔写真なし・テキストのみ → 信頼性スコア大幅減点

**GEN-006で新確認の信頼性指標:**
- **3軸No.1**（生徒数×満足度×卒業実績）= 単一No.1より強力（GEN-006-011）
- **会員継続率**（継続率95.1%）= 「入った人が辞めない」という品質間接証明（GEN-006-010）
- **満足度93%** = 具体的な数字が「No.1」より根拠として機能しやすい（GEN-006-009/011）

### 3. CTA設計の評価基準
- 無料体験CTAが **FV・中盤・最終** の3箇所以上に存在するか
- 文言の「摩擦の低さ」（「まず〇分だけ体験」「予約1分」「受け取る」等）
- スマホ固定フッターの有無
- **「試してみる」は弱い** → 「今すぐ申し込む」「無料で受け取る」に変えると改善（GEN-006-011指摘）

### 4. 料金・オファー評価
- 月額換算の表示: 「月〇〇円から」で心理的ハードルを低下させているか
- 分割払い対応の明示（高額スクールで特に重要）
- 無料体験の「気軽さ」が伝わる文言・デザインか
- **料金不明示はGEN-006-001/006/007と3連続確認** → 必ずチェックする

### 5. スクール系3大欠損チェック（FAIL-015）
```
□ 料金プランがLP内に明示されているか（「お問い合わせ」だけは-8点）
□ FAQセクションが存在するか（未経験不安・費用・継続性の3点最低）
□ 最終CTAが存在するか（コンテンツで終わっていないか）
```
→ 3つ全て欠損でCTA/オファー/導線の3軸が大幅減点 → D〜C+確定

## 分析チェックリスト

### FV（/20）
```
□ FVに「受講後の成果」が数字で示されているか
□ ターゲットの悩み・課題への共感コピーがあるか
□ FV内に無料体験・申込CTAが存在するか
□ FV写真が「受講者のポジティブな姿」または「成果物」か
□ スクール/講座の特徴が3秒で伝わるか
```

### 導線設計（/20）
```
□ 「このような悩みはありませんか？」→ 共感セクションがあるか
□ 「得られること・身につくスキル」セクションが明確か
□ カリキュラム詳細（具体的な学習内容）が記載されているか
□ 卒業生・修了生の実績セクションがあるか
□ 料金プランページへの誘導が自然か
□ FAQ: 「未経験でも大丈夫？」「続けられるか不安」等の不安先取りがあるか
```

### CTA設計（/20）
```
□ FV内にCTAが存在するか
□ 中盤（LP全体の中央付近）にCTAがあるか
□ 最終セクションにCTAがあるか（3箇所以上が合格ライン）
□ 無料体験の「ハードルの低さ」が文言で伝わるか（「まず体験」「1分で申込」等）
□ スマホ固定フッターに申込ボタンがあるか
□ CTAボタンの色がページ内で目立つか（背景色と高コントラスト）
```

### 信頼性（/15）
```
□ 卒業生の声: 顔写真がついているか
□ 卒業生の声: 転職先・変化後の職業が明記されているか
□ 卒業生の声: 「〇ヶ月で〜」の具体的変化が記載されているか
□ 受講者数・修了者数が数字で明示されているか
□ 合格率・就職率・転職成功率が記載されているか（資格・転職系）
□ メディア掲載・受賞歴・認定資格等の第三者権威があるか
□ 講師の実績・資格が明示されているか
```

### オファー（/15）
```
□ 無料体験レッスンの内容が具体的に説明されているか
□ 料金プランが明示されているか（「お問い合わせください」のみは0点）
□ 月額換算の記載があるか
□ 分割払い・ローン対応が明示されているか（高額スクール）
□ キャンペーン・期間限定特典があるか
□ 無料体験に「申込→連絡→体験」のステップが示されているか
```

### モバイル最適化（/10）
```
□ スマホ表示でCTAボタンがタップしやすいサイズか（44px以上）
□ 申込フォームの項目が5個以内か
□ 画像がスマホ幅で適切に表示されているか
□ テキストが読みやすいフォントサイズか（16px以上）
```

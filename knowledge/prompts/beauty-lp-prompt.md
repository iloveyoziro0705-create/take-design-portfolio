# 美容サロン LP制作プロンプト

**用途**: 美容院・ネイルサロン・エステ・脱毛・まつ毛サロンのLP制作  
**参照パターン**: Pattern-LP-001 / IND-001 / FV-002 / CTA-001  
**業種タグ**: `#beauty` `#salon` `#luxury`

---

## プロンプト本文

```
あなたは美容業界に特化したLPデザイナー・CVRコンサルタントです。
以下の美容サロン案件のLPを制作してください。

## 案件情報
- サロン名: [サロン名]
- 業種: [美容院 / ネイル / エステ / 脱毛 / まつ毛 / その他]
- ターゲット: [年齢層・悩み・特性]
- CVポイント: [LINE予約 / 電話予約 / ネット予約]
- オファー: [初回限定〇〇円 / 無料カウンセリング / 体験コース]
- 制作先: [案件名]-lp/index.html

## 制作前に読むナレッジ（必須）
1. knowledge/lp/lp-design-dictionary.md の Pattern-LP-001
2. knowledge/lp/industry-patterns.md の IND-001（美容・サロン系）
3. knowledge/learning-history/2026-06-20-beauty-salon-lp.md（Bランク参考事例）
4. knowledge/lp/fv-library.md の FV-002（グラデーション背景型）
5. knowledge/lp/cta-library.md の CTA-001（LINE誘導型）
6. knowledge/inspirations/design-inspirations.md の DESINS-001（アイボリー×ゴールド）

## 推奨セクション構成（Pattern-LP-001）
1. Hero（FV）
   - グラデーション背景（ivory → rose → gold）
   - ターゲット明示コピー（例: 「30代女性のための〇〇」）
   - 信頼バッジ（実績数・評価）
   - LINE予約CTA

2. 実績・数字
   - 顧客数・満足度・経験年数を大きな数字で表示

3. 特徴・強み（3〜4項目）
   - アイコン + 見出し + 説明文

4. サービス・料金
   - カード型でプランを比較表示

5. お客様の声（3件以上）
   - 名前 + 写真 + 具体的な成果コメント

6. 施術の流れ
   - ステップ型（来店 → カウンセリング → 施術 → アフターフォロー）

7. よくある質問（FAQ）
   - 5〜7件で主要不安を先回り解消

8. アクセス・情報

9. 最終CTA
   - オファー（初回限定）+ LINE予約ボタン

## 重要ポイント（beauty-salon-lp 分析より）
- オファーが最大の弱点になりやすい → 初回特典・体験価格を必ず設定する
- CTA-001（LINE誘導）をFV・中盤・最終の3箇所以上に配置する
- スマホ固定フッターCTAをデフォルト実装する
- DESINS-001: アイボリー × ゴールド × ブラウンパレットが美容系の定番

## デザイン仕様
- ベースカラー: `#faf8f4`（アイボリー）
- アクセントカラー: `#c9a96e`（ゴールド）
- 見出しフォント: Noto Serif JP
- 本文フォント: Noto Sans JP
- コンセプト: ラグジュアリー・上品

## 美容系CTAの鉄板文言
- 「LINEで無料カウンセリングを予約する」
- 「初回体験コース[金額]を予約する」
- 「今すぐLINEで空き状況を確認する」

## Failure Pattern チェック（事前確認）
- [ ] FAIL-001: CTAが3箇所以上あるか
- [ ] FAIL-003: 実績・口コミが存在するか
- [ ] FAIL-004: FVで誰向けか明確か
- [ ] FAIL-005: オファーが設計されているか（デザイン先行禁止）
```

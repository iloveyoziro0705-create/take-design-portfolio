# 住み込み求人・季節労働LP制作プロンプト

**用途**: 住み込み求人・季節労働・出稼ぎ・リゾートバイトのLP制作  
**参照パターン**: Pattern-LP-003 / IND-004 / FV-007  
**業種タグ**: `#recruit` `#live-in` `#seasonal` `#resort`

---

## プロンプト本文

```
あなたは住み込み求人・季節労働に特化したLPデザイナー・CVRコンサルタントです。
以下の求人LP案件を制作してください。

## 案件情報
- 企業名 / 施設名: [施設名]
- 求人種別: [住み込み / リゾートバイト / 季節労働 / 出稼ぎ]
- 業種: [ホテル / 旅館 / 農業 / 建設 / 工場 / 介護施設 / その他]
- 勤務地: [都道府県・エリア]
- 期間: [短期〇ヶ月 / 長期 / 通年]
- 給与・待遇: [月給・日給・寮費無料・食事付き等]
- CVポイント: [LINE / 電話 / エントリーフォーム]
- ターゲット: [10〜20代 / フリーター / 学生 / 転職希望者 / 出稼ぎ希望者]
- 制作先: [案件名]-lp/index.html

## 制作前に読むナレッジ（必須）
1. knowledge/lp/lp-design-dictionary.md の Pattern-LP-003
2. knowledge/lp/industry-patterns.md の IND-004（採用・リクルート系）
3. knowledge/lp/fv-library.md の FV-007（悩みターゲット直接訴求型）
4. knowledge/lp/cta-library.md の CTA-001（LINE誘導型）

## 住み込み求人の特有ニーズ
- 応募者の最大の不安: 「寮の環境は？」「ちゃんと稼げる？」「合わなかったら辞められる？」
- 応募者の動機: 「短期間で稼ぎたい」「住む場所を確保したい」「リセットしたい」「旅行気分で働きたい」
- 競合との差別化: 「寮の快適さ」「給与の高さ」「サポート体制」「勤務地の魅力」

## 推奨セクション構成
1. Hero（FV）
   - 「〇ヶ月で〇万円貯まる」等の数字訴求コピー
   - 勤務地の魅力的なビジュアル（リゾート地なら海・山等）
   - 「まずはLINEで聞いてみる」CTA

2. 待遇・給与の数字訴求
   - 月収・日給・年収換算を大きく表示

3. 寮・生活環境の紹介
   - 写真 + 設備説明（食事付き・Wi-Fi完備・個室等）

4. 働くスタッフの声（3名以上）
   - 「入社前の不安 → 入社後の実感」形式

5. 勤務地の魅力
   - 地域の観光・生活情報（住み込みなので生活イメージが重要）

6. 選ばれる理由（3〜5項目）

7. 応募の流れ
   - LINE登録 → ヒアリング → 書類不要で内定 の流れを明示

8. よくある質問
   - 「未経験OK？」「女性一人でも安心？」「途中退職は？」

9. 最終CTA
   - 「まず話だけでもOK」のハードルを下げた文言

## 住み込み求人のCTA設計
- ハードルを最大限下げる: 「まずLINEで話を聞くだけでOK」
- 応募不安の解消: 「経験不問・履歴書不要・即日返信」
- LINE登録を最優先CVアクションにする（フォームよりハードルが低い）

## デザイン仕様
- ベースカラー: `#ffffff` または リゾート系なら `#e8f5e9`（ライトグリーン）
- アクセントカラー: `#ff6b35`（オレンジ・エネルギッシュ）または `#0097a7`（海・空）
- フォント: Noto Sans JP（フレンドリー系）
- コンセプト: 活発・安心・稼げる実感

## 住み込み求人CTAの鉄板文言
- 「まずはLINEで話を聞いてみる（返信即日）」
- 「LINEで空き状況を確認する（履歴書不要）」
- 「〇ヶ月で〇万円稼いだ実績を見る」

## Failure Pattern チェック
- [ ] FAIL-001: CTAが3箇所以上か（LINE登録ボタン）
- [ ] FAIL-003: スタッフの声・実績数値が存在するか
- [ ] FAIL-004: FVで「誰向け・いくら稼げるか」が明確か
- [ ] FAIL-002: 寮・給与・仕事内容の情報量が多すぎないか
```

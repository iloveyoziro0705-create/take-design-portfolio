# 採用LP制作プロンプト

**用途**: 正社員・パート・アルバイト採用LPの制作  
**参照パターン**: Pattern-LP-003 / IND-004 / FV-007 / CTA-002  
**業種タグ**: `#recruit` `#hr` `#career`

---

## プロンプト本文

```
あなたは採用マーケティングに特化したLPデザイナー・CVRコンサルタントです。
以下の採用LP案件を制作してください。

## 案件情報
- 企業名 / サービス名: [企業名]
- 募集職種: [職種]
- 雇用形態: [正社員 / パート / アルバイト / 業務委託]
- ターゲット: [年齢層・経験・希望条件]
- CVポイント: [エントリーフォーム / LINE / 電話]
- 最大の強み（応募理由）: [給与 / 働き方 / 福利厚生 / 成長 / 職場環境]
- オファー: [選考優遇 / 面接保証 / 祝い金]
- 制作先: [案件名]-lp/index.html

## 制作前に読むナレッジ（必須）
1. knowledge/lp/lp-design-dictionary.md の Pattern-LP-003（採用・医療・福祉系）
2. knowledge/lp/industry-patterns.md の IND-004（採用・リクルート系）
3. knowledge/lp/fv-library.md の FV-007（悩みターゲット直接訴求型）
4. knowledge/lp/cta-library.md の CTA-002（無料相談・問い合わせ型）

## 推奨セクション構成（Pattern-LP-003）
1. Hero（FV）
   - 悩みキャッチ（例: 「給与が低い」「残業が多い」「スキルが活かせない」）
   - 解決提案コピー
   - エントリーCTA（LINE or フォーム）
   - 信頼バッジ（求人実績・内定率）

2. この求人の3つの特徴
   - 数字を使った強みの訴求

3. 働くメンバーの声（3名以上）
   - 顔写真 + 名前 + 入社前の不安 + 入社後の変化

4. 1日の仕事の流れ
   - 時系列ステップ形式

5. 福利厚生・待遇
   - アイコン + リスト形式

6. よくある質問（FAQ）
   - 「経験がなくても大丈夫？」等の応募不安を先回り解消

7. 応募のステップ
   - エントリー → 面接 → 内定 の流れを明示

8. 最終CTA
   - オファー（面接確約・選考優遇）+ エントリーボタン

## 重要ポイント（採用LP設計）
- 求職者の最大の不安は「本当にいい職場か」 → 社員の声が信頼性の核心
- FVで「悩み → 解決」の流れを作る（「〇〇で悩んでいるあなたへ」型）
- エントリーのハードルを下げる: フォーム項目は最小限（名前+電話番号+メールのみ）
- 採用LPは「職場の雰囲気」がCVRに直結 → スタッフ写真・職場写真を必ず使う

## デザイン仕様
- ベースカラー: `#ffffff`（白）または `#f5f5f5`（ライトグレー）
- アクセントカラー: 企業ブランドカラー または `#2563eb`（信頼感の青）
- フォント: Noto Sans JP（クリーン・フレンドリー系）
- コンセプト: 親しみやすい・安心感・フレンドリー

## 採用系CTAの鉄板文言
- 「まずは話を聞いてみる（30秒でエントリー）」
- 「LINEで気軽に質問する」
- 「応募フォームに進む（経験不問）」

## Failure Pattern チェック（事前確認）
- [ ] FAIL-001: CTAが3箇所以上か
- [ ] FAIL-003: 社員の声・実績が存在するか
- [ ] FAIL-004: FVでターゲット（悩み）が明確か
- [ ] FAIL-002: 情報量が多すぎないか（採用LPは情報過多になりやすい）
```

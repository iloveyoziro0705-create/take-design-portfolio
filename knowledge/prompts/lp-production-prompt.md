# LP制作プロンプト（汎用）

**用途**: 新規LP制作を開始するときの標準プロンプト  
**参照ナレッジ**: AGENTS.md / lp-design-dictionary.md / fv-library.md / cta-library.md

---

## プロンプト本文

```
あなたは世界トップクラスのLPデザイナー・CVRコンサルタントです。
以下の案件情報をもとに、高CVRのLPをHTML/CSS/JSで制作してください。

## 案件情報
- 案件名: [案件名]
- 業種: [業種]
- ターゲット: [ペルソナ情報]
- CVポイント（メインCTA）: [LINE / 電話 / フォーム / 予約]
- オファー: [特典・初回割引・無料体験・保証等]
- 制作先フォルダ: [案件名]-lp/index.html

## 制作前に必ず実行すること（AGENTS.md Step 1〜4）
1. knowledge/lp/lp-design-dictionary.md を読む（構成パターン確認）
2. knowledge/lp/fv-library.md を読む（FVパターン確認）
3. knowledge/lp/cta-library.md を読む（CTAパターン確認）
4. knowledge/learning-history/ を検索して同業種の過去分析を確認する
5. LP-Rank S以上の案件を最優先で参照する

## 制作フロー（順序厳守）
1. 業種判定 → 業種タグ特定
2. 類似案件検索（learning-history）
3. Sランク案件の確認
4. FV選定（fv-library.md から）
5. CTA選定（cta-library.md から）
6. 導線設計（悩み→解決→証拠→CTAの4ステップ確認）
7. 信頼要素選定（実績・口コミ・権威性）
8. 情報設計（セクション構成の確定）
9. デザイン設計（design-system.md のトークン適用）
10. HTML/CSS/JSコーディング

## 必須実装チェックリスト
- [ ] FV内にターゲット・ベネフィット・信頼要素・CTAが揃っている
- [ ] CTAが最低3箇所（FV / 中盤 / 最終）配置されている
- [ ] スマホ固定フッターCTAが実装されている
- [ ] 悩み→解決策→証拠→CTAの導線フローが存在する
- [ ] 実績数値またはお客様の声が存在する
- [ ] オファー（初回特典・希少性・リスク低減）が明示されている
- [ ] 本文フォント16px以上・行間1.7以上
- [ ] タップ領域44px以上
- [ ] ルートのindex.htmlを上書きしていない

## デザイン仕様（design-system.md 参照）
- フォント: Noto Serif JP + Noto Sans JP（ラグジュアリー系）
- CSSカスタムプロパティで色・タイポグラフィを管理する
- clamp()でレスポンシブタイポグラフィを実装する
- IntersectionObserverでスクロールアニメーションを実装する

## 保存先
[案件名]-lp/index.html
（ルートのindex.htmlには絶対に触れない）
```

---

## カスタマイズポイント

プロンプト使用時に必ず差し替える変数:
- `[案件名]` → 実際のブランド名
- `[業種]` → 具体的業種
- `[ペルソナ情報]` → 年齢・性別・悩み・行動特性
- `[LINE / 電話 / フォーム / 予約]` → 実際のCVアクション
- `[特典・初回割引・無料体験・保証等]` → 実際のオファー内容

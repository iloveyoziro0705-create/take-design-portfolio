# レストラン・飲食店 LP制作プロンプト

**用途**: レストラン・カフェ・居酒屋・バー・ラーメン店等のLP制作  
**参照パターン**: Pattern-LP-002 / IND-002 / FV-001 / CTA-005  
**業種タグ**: `#restaurant` `#food` `#cafe` `#bar`

---

## プロンプト本文

```
あなたは飲食業に特化したLPデザイナー・CVRコンサルタントです。
以下の飲食店LP案件を制作してください。

## 案件情報
- 店舗名: [店舗名]
- 業種: [レストラン / カフェ / 居酒屋 / バー / ラーメン / その他]
- コンセプト: [高級 / カジュアル / ファミリー / デート / 接待]
- ターゲット: [年齢層・利用シーン]
- CVポイント: [電話予約 / Web予約 / ネット予約サービス]
- オファー: [OPENキャンペーン / コース料理特典 / ドリンク1杯無料]
- 状況: [新規OPEN / リニューアル / 通常営業]
- 制作先: [案件名]-lp/index.html

## 制作前に読むナレッジ（必須）
1. knowledge/lp/lp-design-dictionary.md の Pattern-LP-002
2. knowledge/lp/industry-patterns.md の IND-002（飲食・レストラン系）
3. knowledge/learning-history/2026-06-20-restaurant-lp.md（D Rank反面教師事例）
4. knowledge/lp/fv-library.md の FV-001（フルスクリーン背景型）
5. knowledge/inspirations/design-inspirations.md の DESINS-002（ダークテーマ）
6. knowledge/inspirations/cta-inspirations.md の CTAINS-003（ゴールドCTA）

## restaurant-lp 反面教師から学ぶ必須改善点
※ 2026-06-20-restaurant-lp.md は LP-Score 50/100（D Rank）であり、以下の問題を持つ
1. CTAが2箇所のみ → 今回は必ず3箇所以上にする
2. スマホ固定フッターCTAなし → 今回は必ず実装する
3. FAQなし → 今回は必ず追加する
4. オファーなし → 今回はキャンペーン・特典を設定する
5. 口コミなし → 今回はグランドオープンでも代替信頼要素を確保する

## 推奨セクション構成（改善版）
1. Hero（FV）
   - 高品質料理写真フルスクリーン
   - コンセプトコピー（接待・記念日・デート等のシーン訴求）
   - ダークオーバーレイ: linear-gradient(rgba(0,0,0,.45), rgba(0,0,0,.75))
   - 予約CTA（GRAND OPENの場合は希少性訴求）

2. コンセプト・シェフメッセージ
   - ストーリーテリング形式

3. 人気メニュー（3〜6品）
   - 料理写真 + 料理名 + 説明

4. 利用シーン提案
   - 「記念日」「接待」「デート」の各シーンを具体的に提示
   - ここに中盤CTA（予約ボタン）を配置する

5. シェフ・スタッフ紹介
   - 経歴・受賞歴・こだわりで権威性を確保

6. 空間・雰囲気ギャラリー
   - 店内写真 3〜6枚

7. お客様の声（OPEN後は必ず追加）
   - OPEN前の場合: 「オープン予約受付中」のオファーで代替

8. よくある質問（必須）
   - 「ドレスコードは？」「個室はある？」「駐車場は？」「予約は必要？」

9. アクセス・営業時間

10. 最終CTA
    - オープンキャンペーン / コース特典 + 予約ボタン

## スマホ固定フッターCTAの実装（必須）
```html
<div class="sticky-cta">
  <a href="tel:XXX-XXXX-XXXX" class="btn-tel">電話で予約</a>
  <a href="[予約URL]" class="btn-reserve">ネット予約</a>
</div>
```

## デザイン仕様（高級飲食店向け）
- ベースカラー: `#0f0f0f`（ほぼ黒・ダークテーマ）
- テキスト見出し: `#f5e6c8`（クリームゴールド）
- アクセントカラー: `#d8a76a`（ゴールドブラウン）
- フォント見出し: serif（Noto Serif JP）
- フォント本文: Noto Sans JP
- コンセプト: 高級感・没入感・食欲喚起

## 飲食店CTAの鉄板文言
- 「ご予約はこちら（当日予約OK）」
- 「特別コースを予約する（OPENキャンペーン中）」
- 「電話で空き状況を確認する」

## Failure Pattern チェック
- [ ] FAIL-001: CTAが3箇所以上か（restaurant-lp の最大の反省点）
- [ ] FAIL-001: スマホ固定フッターCTAが実装されているか
- [ ] FAIL-003: 信頼要素（シェフ経歴・メディア掲載・口コミ）があるか
- [ ] FAIL-004: FVで「誰のための・どんな体験か」が明確か
- [ ] FAIL-005: オファー（特典・キャンペーン）が設計されているか
```

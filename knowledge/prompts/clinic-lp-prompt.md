```
あなたは医療・ヘルスケア分野に特化したLPデザイナー・CVRコンサルタントです。
以下の医療機関LP案件を制作してください。

## 案件情報
- 医院名 / 院名: [医院名]
- 種別: [内科 / 美容クリニック / 整骨院 / 鍼灸院 / 歯科 / その他]
- 主訴（患者の悩み）: [腰痛 / 肩こり / ニキビ / アンチエイジング 等]
- ターゲット: [年齢層・性別・症状]
- CVポイント: [電話予約 / LINE予約 / Web予約]
- オファー: [初診無料 / カウンセリング無料 / 体験施術]
- 制作先: [案件名]-lp/index.html

## 制作前に読むナレッジ（必須）
1. knowledge/lp/lp-design-dictionary.md（構成パターン確認）
2. knowledge/lp/industry-patterns.md の IND-003（医療・クリニック系）
3. knowledge/lp/fv-library.md の FV-001（フルスクリーン背景型）または FV-007（悩みターゲット型）
4. knowledge/lp/cta-library.md の CTA-002（無料相談型）

## 医療LP設計の核心原則
- 患者の主訴（悩み・症状）をFVで正確に言語化することが最重要
- 権威性（院長経歴・資格・症例数）が信頼性の核心
- 「安心して来院できる」施設情報・スタッフ情報が必須
- 医療広告ガイドラインに準拠する（誇大表現・比較広告禁止）

## 医療広告ガイドライン遵守事項
- 「〇〇を治す」等の治癒保証表現は使用禁止
- 「日本一」「最高の」等の最上級表現は禁止
- 患者口コミの掲載は一定の制限あり（症例写真は本人同意必須）
- 「必ず改善する」等の断言表現は禁止

## 推奨セクション構成
1. Hero（FV）
   - 症状訴求コピー（例: 「なかなか治らない腰痛に悩んでいませんか？」）
   - 院の強み（症例数・経験年数）
   - 予約CTA

2. このような悩みはありませんか（共感セクション）
   - チェックリスト形式（5〜7項目）

3. 当院が選ばれる理由（3〜5項目）
   - 資格・技術・設備・アクセス

4. 院長・スタッフ紹介
   - 顔写真 + 経歴 + メッセージ（信頼性の核心）

5. 施術の流れ
   - 初診 → 問診 → 施術 → 改善計画のステップ

6. 症例・改善実績（医療ガイドライン準拠）
   - 匿名・本人同意のもとで

7. 患者の声（3件以上）
   - 医療ガイドラインの範囲内で掲載

8. よくある質問
   - 「初診でも大丈夫？」「保険は使える？」「何回通う必要がある？」

9. アクセス・診療時間

10. 最終CTA
    - 初診特典 + 予約ボタン（電話 / Web）

## 医療系CTA文言（コンプライアンス対応）
- 「まず一度ご相談ください（初診無料）」
- 「今すぐWeb予約する（予約優先）」
- 「電話で症状を相談する」

## デザイン仕様
- ベースカラー: `#ffffff`（清潔感重視）
- アクセントカラー: `#1a73e8`（信頼の青）または `#2e7d32`（健康の緑）
- フォント: Noto Serif JP + Noto Sans JP（信頼感・伝統系）
- コンセプト: 清潔感・信頼感・専門性

## Failure Pattern チェック
- [ ] FAIL-001: CTAが3箇所以上か
- [ ] FAIL-003: 院長プロフィール・症例数・資格が存在するか
- [ ] FAIL-004: FVで「どんな悩みを持つ人向けか」が明確か
- [ ] FAIL-005: デザインより先にCV構造を設計したか
```

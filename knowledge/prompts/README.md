# Prompts Library

## 概要

実務で成功した指示文・制作プロンプトを業種別・用途別に管理するライブラリ。
セッション開始時にこのフォルダのプロンプトを参照することで、素早く高品質な制作を開始できる。

---

## ファイル一覧

| ファイル名 | 用途 |
|----------|------|
| `lp-analysis-prompt.md` | LP分析を開始するときの標準プロンプト |
| `lp-production-prompt.md` | 新規LP制作を開始するときの標準プロンプト |
| `beauty-lp-prompt.md` | 美容サロン・エステ・脱毛LP向け |
| `recruit-lp-prompt.md` | 採用LP（正社員・パート・アルバイト）向け |
| `job-lp-prompt.md` | 住み込み求人・季節労働・出稼ぎLP向け |
| `clinic-lp-prompt.md` | クリニック・整骨院・鍼灸院LP向け |
| `restaurant-lp-prompt.md` | レストラン・飲食店LP向け |
| `wordpress-production-prompt.md` | WordPress制作向け |

---

## 使用ルール

1. 案件の業種に合ったプロンプトを選択する
2. プロンプトをコピーし、`[案件名]` `[業種]` `[ターゲット]` 等の変数を実際の案件情報に置き換える
3. 制作後、プロンプトを改善した場合は更新する
4. 新業種対応のプロンプトが完成したらこのフォルダに追加する

---

## 変数フォーマット

プロンプト内の `[変数名]` は実案件の情報に必ず差し替える。

| 変数 | 置き換え内容 |
|------|------------|
| `[案件名]` | ブランド名・プロジェクト名 |
| `[業種]` | 具体的な業種名 |
| `[ターゲット]` | ペルソナ情報 |
| `[CVポイント]` | メインCTAの内容 |
| `[オファー]` | 特典・初回限定・無料体験等 |

**CTAポイント確認（3箇所）**:

| 位置 | CTA文言 | 色 | 評価 |

| 最終 | 「無料の投資講座を試してみる」 | オレンジ | △（同上） |

**評価**:
- **3箇所CTAはスクール系として標準的**（FV・中盤・最終の3点配置）
- **オレンジカラーの一貫性**: ネイビー/ホワイト背景に対して高い視認性
- **問題点**: 中盤・最終CTAの「試してみる」は全シリーズで最も弱い文言の一つ

**改善案**:
```
× 「無料の投資講座を試してみる」
○ 「今すぐ2時間の無料講座を受け取る」
◎ 「5,000円の投資講座を今すぐ無料で申し込む」
```

## ⑥ 参考パターン・知見抽出

### 参考パターン1: 3軸No.1訴求（生徒数×満足度×卒業実績）

**構造**:
```
日本一の生徒数・満足度・卒業実績
```

**機能する理由**:
- 単一No.1（「会員数No.1」）は「他の指標はどうなの？」という疑念が生まれる
- 3軸同時No.1は「量（生徒数）× 満足度（品質）× 卒業実績（成果）」の三位一体を証明
- 読者のあらゆる不安軸（多い=流行っている / 満足=質が良い / 卒業実績=効果がある）を同時解消

**実装ポイント**: 3つの指標は「入口（規模）× 中間（満足）× 出口（成果）」の順で選ぶと網羅性が高い。

### 参考パターン2: カリキュラム詳細開示による価値有形化

**構造**:
```
[基礎編 - 6項目]
□ 投資商品の種類・株式市場・株の仕組み
□ 投資信託/FX/バランス投資の違い
□ NISA・iDeCoの活用方法
□ 資産が増えていく仕組み
□ 貯かかずに上手した老後資産の作り方（確認）
...

[発展編 - 6項目]
□ 投資先別の選び方
□ 会社分析の読み方
□ 低リスク/バランス投資
□ グローバル分散投資
□ FXの基礎と儲かる方法
...
```

**機能する理由**:
- スクールの価値は「無形サービス」→ 学べる内容を具体的に列挙することで価値を「有形化」
- 「これだけの内容が2時間で学べる」という量的価値の可視化
- 興味ある項目を見つけた瞬間にスクロール停止・申し込みへの動機付け

**適用場面**: 複数テーマを持つ教育系コンテンツ全般（スクール / セミナー / 動画教材）

### 参考パターン3: 「経済的自由」フレームによる投資教育の共感設計

**構造**:
```
[悩み列挙]
・共働きで老後が不安
・毎日仕事に没頭しているが、これでいいのか
・家族との時間と老後の準備どちらも欲しい

[問題提起]
「経済的自由のために必要なものとは」
↓
「投資をしている人が増えている（社会トレンド）」
↓
「でも、どんな勉強をすればいいか分からない」

[解決策]
「2時間の投資講座で基礎から発展まで学べる」
```

**機能する理由**:
- 「老後不安・時間不足」という普遍的悩みから「経済的自由」というポジティブな目標へのリフレーミング
- 「投資への関心はあるが一歩踏み出せない」という最大公約数の心理に刺さる
- 社会トレンド（投資人口増加）を挟むことで「今やらないと乗り遅れる」という損失回避を刺激

**スクール種別への応用**:
```
投資スクール   → 「経済的自由のために」
英会話スクール → 「キャリアの可能性を広げるために」
プログラミング → 「副業・転職でキャリアを変えるために」
```

### 参考パターン4: 価格アンカリング（5,000円→無料）

```
「5,000円相当が今だけ無料」「満額5,000円の投資講座を無料でプレゼント！」
```

**機能する理由**:
- 「無料」の前に「通常価格○円」を提示することで、無料の価値を具体化
- 「5,000円分の価値があるのに0円」というお得感の最大化
- 「無料だから質が低い」という疑念を「本来有料」という事実で払拭

**注意点**: 価格アンカリングと期限設定（カウントダウンタイマー等）を組み合わせるとCV率が最大化される。このLPは期限設定が「今だけ」と曖昧なため、相乗効果が弱い。

## Learning Reason

### 保存理由
GEN-006-011として、Global Financial School（投資・金融教育スクール）の完全6軸分析を実施。3軸No.1訴求 / カリキュラム詳細開示 / 価格アンカリング / 「経済的自由」フレームによる共感設計の4パターンが特に参照価値が高い。本シリーズ（GEN-006）の中で「信頼性セクションの充実度」が最も高いLPとして記録。

### 再利用価値
- 投資・金融教育スクール全般
- 複数テーマを持つ教育系サービスのカリキュラム開示設計
- 「経済的自由・老後不安」をターゲットにするあらゆる金融系サービス
- 無料セミナー・ウェビナー型LPのオファー設計

### 特に優れている点
「カリキュラム詳細開示（基礎編6項目/発展編6項目）」は無形の教育サービスを最も効果的に有形化する手法。何を学べるか・どこまで学べるかを具体的に列挙することで「2時間でこれだけ学べる」という価値感知を最大化している。

### 改善余地（他案件への逆活用）
- CTA文言を「試してみる」→「今すぐ無料で申し込む」に変更するだけでCV率改善見込み
- 「今だけ無料」を日付指定/タイマー型に変更することで緊急性の追加

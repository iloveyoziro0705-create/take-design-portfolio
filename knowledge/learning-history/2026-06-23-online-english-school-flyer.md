## ③ コピー・構成設計の知見抽出

### 知見①: 英会話スクールの「3つのPOINT」は品質×利便性×価格の三点固定

```
POINT 1: 品質訴求     →「プロのネイティブ講師 / 完全個人レッスン」
POINT 2: 利便性訴求   →「好きな時間に好きなだけ」
POINT 3: 価格訴求     →「嬉しい月額料金〇〇円」
```

英会話スクールのユーザーが持つ3つの不安（「先生は本当にうまいか？」「自分のスケジュールに合うか？」「月いくらかかるか？」）を3POINTで解消する構造。この順番は「品質確認 → 実用性確認 → 価格確認」という意思決定フローと一致している。

### 知見②: 受講ステップの4段階明示（フライヤーでの導線設計）

オンラインスクール特有の「どうやって始めるの？」という操作不安を、A4フライヤー1枚内でも4ステップで解消している。
```
入会手続き → コース決定 → 日時・講師選択 → レッスン開始
```
「申し込んだ後に何が起こるか」の透明性が申し込み障壁を下げる。Webの申込LPと同じ設計思想がフライヤーにも適用されている。

### 知見③: 「入会金無料！」のバッジ配置とFVの関係

FV右上（写真の空きスペース）に「入会金無料！」バッジを配置。
- 写真（感情訴求）+ バッジ（論理的安心材料）をFV内で同時に提供する設計
- 「自宅ではじめる〜」というキャッチに対して「でも費用は？」という次の疑問を右上バッジが即座に解消

### 知見④: 「自宅ではじめる英会話」コピーの構造

```
「自宅で」    → 場所の自由（通学不要・既存の生活導線）
「はじめる」  → 低ハードル感（今すぐ始められる）
「英会話」    → ストレートな訴求（英語ができるようになる）
```
シンプルだが「自宅で」というキーワードがオンライン英会話の本質的ベネフィット（通学なし / 好きな場所で）を1語で表現している。

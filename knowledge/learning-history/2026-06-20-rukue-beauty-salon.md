## ⑤ ヘアドネーションバナー（最重要発見）

### バナースコア: A（このページ最高評価の要素）

**バナー内容:**
「子供たちのために私たちができること 髪の毛を寄付するボランティア『ヘアドネーション』承ります」

**設計の評価:**

| 要素 | 内容 |
|------|------|
| 訴求軸 | CSR（企業の社会的責任） / 共感・感動 |
| ターゲット共感 | 小児がんや脱毛症で髪に悩む子供への支援 → 「このサロンで切れば社会貢献できる」 |
| 差別化機能 | 「ヘアドネーション対応」を打ち出す美容室はまだ多くない → 明確な差別化要素 |
| 信頼性向上 | 社会貢献に取り組む店＝誠実・信頼できる、というヒューリスティック |
| CTA誘引力 | ヘアドネーションを検討中の人が「このサロンで切る理由」になる |
| 配色 | 青系（信頼・安心を連想させる色） |

**なぜA評価か:**
- 技術・価格・雰囲気だけで差別化が難しい美容室市場において、CSR（社会貢献）は稀有な差別化軸
- ヘアドネーションを希望する顧客は「ドネーション対応サロンを探している」という明確なニーズを持っており、集客の入口になる
- 「良いことをしているサロン」という印象が口コミを生む

**美容室LPへの応用:**
```
CSR要素として組み込める要素:
- ヘアドネーション対応（31cm以上の髪を必要とする子供への寄付）
- オーガニック製品使用（環境負荷低減）
- 地元農産物との協力（地域貢献）
- バリアフリー対応（インクルーシブ設計）
```

## ⑩ 今回学んだ重要知識

1. **ヘアドネーション（CSR要素）設計**: 美容室LPにCSR要素を組み込む手法として「ヘアドネーション対応」が有効。社会貢献×差別化×集客の3つを同時に実現する。beauty-salon-lp設計時の必須チェック項目に追加。
2. **FVキャッチコピーなし問題（FAIL-004強化）**: 写真+ロゴだけでは「誰向けか」「何が得られるか」が伝わらない。プライベートサロンほど価値が伝わりにくいため、必ずH1キャッチコピーが必要。
3. **マンション型プライベートサロンのポジショニング**: 「1対1 / 予約制 / 子連れOK / リーズナブル」の4点を訴求軸にする設計パターン。FV写真にキッズスペースを映り込ませる「無言のターゲティング」は優れた手法。
4. **ギャラリーのカテゴリ設計**: 「Madam/Short/Long」という年齢層×スタイル複合分類。ターゲット別の共感を生む。40〜60代向けMadamカテゴリは多世代対応の明示として有効。
5. **ニュース更新停止の危険性**: 古い日付（2019年）のニュースは「本当に営業しているか？」という疑念を生む。更新継続できないならニュースセクション自体を設置しない方が良い。→ failure-patterns.md 追加候補（NewsStale）
6. **美容室3点特徴の標準軸**: 「カウンセリング（技術・安心）→ 空間・雰囲気 → 価格・継続性」の順番は業界標準として使えるテンプレート。
7. **キッズスペースの無言訴求**: FV写真にキッズ向けアイテムを映り込ませることで、「子連れOK」を文字化せずにターゲットに伝える手法。説明文の省略と視覚的暗示の効果的な組み合わせ。
8. **Q&Aセクション設置**: ナビゲーションに「Q&A」が存在 → 美容室の不安解消設計として有効。特にプライベートサロンは「本当に一人で対応してる？予約は？場所は？」という疑問が多いためFAQが重要。

### 他案件への流用可能な知識

1. **グランドオープン案件**: ヘアドネーション等のCSR要素を初期設計から組み込む提案（信頼性+差別化の同時実現）
2. **マンション型プライベートサロン**: FVコピー候補「あなただけの時間を」「気を使わない居場所」「1対1だから伝わる」を提案テンプレートとして保有
3. **多世代対応サロン**: ギャラリーに「Madam」「Lady」等の年齢層別カテゴリを設置する提案
4. **既存サイト改修案件**: ニュース停止チェックを必ず実施。Instagramフィード埋め込みへの切り替えを提案
5. **ウェブLP全般**: FVにキャッチコピーなしの問題（FAIL-004）を発見した場合の修正提案コピー例として「マンション型サロンFVコピー候補」を参照可能

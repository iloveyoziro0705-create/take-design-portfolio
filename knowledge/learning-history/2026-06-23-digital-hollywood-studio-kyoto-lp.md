## ⑧ 参考パターン・知見抽出

### 参考パターン1（良い事例）: 2コース並列CTAブロック

**構造**:
```
[コース/料金紹介]
┌────────────────────┬────────────────────┐
│ Webデザイナー専攻   │ ネット動画クリエイター専攻│
│ 4月生募集中        │ 7月生募集中          │
│ 申込締切3/25（水）  │ 申込締切5/31（日）   │
│ [詳細・相談はこちら]│ [詳細・相談はこちら] │
└────────────────────┴────────────────────┘
```
→ 2コースを比較しながら申し込み先を選べる設計。多コース提供のスクールでは「どのコースか迷う」という離脱を防ぐための有効な手法。

### 参考パターン2（反面教師）: ブログ/イベントで終わる最終セクション問題

LPの最後のセクションが「ブログ/イベント記事一覧」になっている。

**問題の構造**:
```
ユーザーがLPを読み切った（最も申し込む意欲が高い状態）
       ↓
最後のセクションがブログ記事一覧
       ↓
「ブログを読もうかな」→ 申込から外れた行動に移行
または「これ以上見るものがないから離脱」
```
→ LPの最後は必ず「申込CTA」で終わるべき。ブログ/イベントは別ページまたはフッター前のオプショナル情報として処理する。

## Learning Reason

### 保存理由
GEN-006-006として、GEN-006シリーズ初のLP全体確認事例として記録。全6軸採点が可能になり、スクール系LPの「Dランク構造」の典型例として記録する。特に「ブログで終わる最終セクション問題」「料金不明示+FV内CTAなし」という複合的なFAILパターンが重なった場合の点数水準（43点/D）を確認できる。

### 再利用価値
- スクール系LPの「やってはいけない設計」の参照事例
- 地域特化型スクールLP（地域校）の設計課題の把握

### 特に優れている点
「申込締切日の具体日付（3/25・5/31）」による緊急性演出は有効。締切日がなければさらに点数が下がっていた。

### 他案件への応用先（反面教師）
- 料金を「詳細ページ」に隠すスクールLP全般 → 料金はLP内で明示すべき
- 最後のセクションがブログになっているすべてのLP → 最終CTAに差し替える

## ★★★ 最大の発見: 「仕事哲学型採用LP」という完全新カテゴリ

### GEN-002で初めて出会う「仕事の本質を語る」設計思想

これまでのGEN-002分析での採用LPの設計軸:

| 設計軸 | 代表例 |
|------|------|
| 求職者の悩み共感型 | 「今の職場に不満はありませんか」|
| 理想の未来提示型 | 「好きを仕事に。」（LAVA）|
| データ左脳訴求型 | ロボットペイメント |
| ブランドビジョン型 | 「ひらけ、世界。ひらけ、自分。」（日本生命）|
| **仕事哲学型** | **「仕入れの極意」← 八百鮮（GEN-002初確認）** |

「仕事哲学型」は「この仕事の本質・奥深さ・プロの世界を語る」アプローチ。
求職者への感情訴求でも企業ビジョンの説明でもなく、**「仕事そのものの魅力」**を前面に出す。

**「極意」という言葉の設計効果**:

| 要素 | 効果 |
|-----|------|
| 「極意」= 奥義・真髄 | 「この仕事には習得する価値のある深みがある」という示唆 |
| 「MIND」= 心・精神 | 技術だけでなく「考え方・マインド」が必要な仕事という格上げ |
| シリーズ構成（01/02）| 「まだ続きがある」という没入感と「全部読みたい」という好奇心 |

## 「度肝を抜いて、常連をつくれ」— 社員への命令形コピー

### GEN-002初確認: 求職者でなく「社員への命令形」で採用する逆転設計

通常の採用LPのコピーは「求職者に語りかける」形式:
```
「あなたの挑戦が...」（求職者へ）
「好きを仕事に。」（求職者へ）
「ひらけ、世界。」（求職者へ）
```

八百鮮の「度肝を抜いて、常連をつくれ」は**「社員への命令形」**:
```
「度肝を抜いて、常連をつくれ」（社員への指令）
```

**この設計の革新性**:

| 効果 | 内容 |
|-----|------|
| **採用基準の可視化** | 「こういう仕事ができる人がほしい」という採用水準を直接示す |
| **カルチャーフィット訴求** | 「このコピーに共感できるか」でセルフスクリーニングが起きる |
| **職人への敬意** | 命令形だが「あなたはプロになれる」という期待が込められている |
| **記憶定着** | 「常連をつくれ」という行動指針が頭に残る |

「度肝を抜いて」= 常識外れの仕入れでお客様を驚かせる  
「常連をつくれ」= 一度来たお客様をリピーターにする  
という仕入れ担当者の2大ミッションを一文に圧縮。

コピーの仕組み:
```
「[お客様への感情訴求（驚かせる）]して、[経営指標（リピート）]をつくれ」
```

## 「八百鮮で稼ぐ」ナビゲーション命名の革新

### 給与ページを「稼ぐ」という動詞で命名する設計

通常の採用サイトの給与ページ命名:
```
「給与・待遇」「福利厚生」「処遇情報」「条件・報酬」
```

八百鮮の命名:
```
「八百鮮で稼ぐ」
```

**「稼ぐ」という動詞の効果**:

| 要素 | 効果 |
|-----|------|
| 動詞（能動形）| 「給与をもらう（受動）」から「稼ぐ（能動）」へのフレーム転換 |
| 自社名 + 稼ぐ | 「八百鮮という場所で稼げる」という明確なメッセージ |
| 金銭的動機の肯定 | 「お金のために働くことへの罪悪感」を解消 |
| 「稼げるスーパー」という差別化 | 一般的な小売業の「給与が低い」イメージへの逆張り訴求 |

→ **GEN-002新パターン: 「稼ぐ」動詞による給与ページ命名**

## 「極意01」「極意02」番号付きシリーズ型コンテンツ

各セクションに「極意01」「極意02」と番号を振るシリーズ構成。

```
極意01: 「度肝を抜いて、常連をつくれ」（インパクト仕入れ）
極意02: 「市場で野菜を買い、身を...」（市場直仕入れ）
（極意03以降が続く構成と推測）
```

**この設計の効果**:
- 番号付きでシリーズ化 → 「続きが気になる」没入型コンテンツ
- 各極意に独立したコピーと写真 → 1セクションで完結する読み応え
- 仕事の「奥義集」的な構成 → 「この会社に入ると仕入れのプロになれる」という期待感

## 問題点

### 問題1: スタッフの「入社前後の変化」が不明確

「極意01」「極意02」は仕事哲学の語りが中心で、
「入社前は〇〇で不安だったが、今は〇〇」という個人の変化が明示されていない。
仕事哲学の語りは「プロとしての今」を見せるが、「初心者でも始められるか」の安心感が薄い。

### 問題2: FAQ・応募ステップが不在

「仕入れの極意」コンテンツは魅力的だが、
「どうやって応募するのか」「選考はどう進むのか」が見えない。

### 問題3: CTA文言が「ENTRY」のみ

「ENTRY」ボタンはあるが、IND-004推奨の「低ハードル文言」がない。
「まずは話を聞くだけでもOK」「見学歓迎」等のハードル下げが採用LPには必要。

## 新規パターン（IND-004追記対象）

### パターン1: 仕事哲学型FVコピー

```
「仕入れの極意」（八百鮮）
「料理の流儀」（飲食店）
「コードの美学」（エンジニア採用）
「営業の真髄」（セールス採用）
```

「求職者の悩み共感」でも「ライフスタイル訴求」でもなく、**「この仕事の奥深さ・哲学」**を前面に出すコピー。職人志向・プロ志向・好奇心型の求職者に刺さる。「極意」「流儀」「真髄」「美学」等の語彙が仕事の格を上げる。

### パターン2: 社員への命令形コピー（採用基準の可視化）

```
「度肝を抜いて、常連をつくれ」（八百鮮）
「コードで世界を変えろ」（エンジニア採用）
「患者の人生に責任を持て」（医療職採用）
```

求職者に語りかけるのではなく、**「この仕事ができる人への命令形」**で書くことで、採用基準を暗示的に可視化し、共感できる求職者が自然に残るセルフスクリーニング効果を生む。

### パターン3: 「稼ぐ」動詞による給与ページ命名

```
「[会社名]で稼ぐ」
「稼ぎ方を知る」
```

給与・待遇情報ページを「稼ぐ」という能動的な動詞で命名することで、「お金のために働く」という動機を肯定し、「稼げる職場」という印象を自然に与える。小売・飲食・製造等「給与が低いイメージ」を覆したい業種で特に有効。

### パターン4: 番組・エンタメ形式のインタビューコンテンツ

```
「SHOWTIME SHOW」（八百鮮）
「〇〇LIVE」「THE INTERVIEW」
```

採用LPのインタビューコンテンツを「番組名+サブタイトル」で命名し、YouTubeやSNS動画文化に親和的なエンターテインメント感を持たせる。「静的ページを読む」から「コンテンツを見る」への転換。

## Learning Reason

### 保存理由
GEN-002採用系LP分析8件目。「仕事哲学型FVコピー」「社員への命令形コピー（採用基準の可視化）」「稼ぐ動詞による給与ページ命名」「番組形式インタビュー」という4つの新規パターンを確認。GEN-002でこれまで分析した中で最も「採用LP設計の常識を覆す」アプローチとして記録価値が高い。

### 特に優れている点
「仕事哲学型採用LP」という完全新カテゴリの発見。求職者を「悩みの解消」でなく「この仕事の奥深さへの好奇心」で引き込む設計思想は、従来のIND-004のフレームを拡張する革新的発見。「極意01/02」シリーズ構成とSHOWTIME SHOWという命名センスも他社採用LPにない独自性がある。

### 他案件への応用先
- 「仕事哲学型コピー」: 職人・専門職（料理人・職人・エンジニア・デザイナー等）の採用LP全般
- 「社員への命令形コピー」: 強いカルチャーを持つスタートアップ・専門商社・製造業の採用LP
- 「稼ぐ動詞命名」: 小売・飲食・物流等「給与が低いイメージ」を覆したい業種
- 「番組形式コンテンツ」: YouTube/SNSとの相性を重視するZ世代向け採用LP

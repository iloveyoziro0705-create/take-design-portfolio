## ★ デザイン統一：GEN-002史上初の満点（10/10）

### 「黒×白×赤」が建設業採用LPの理想形

戸田建設のデザインはGEN-002分析6件の中で、デザイン統一スコアが唯一の**満点**（10/10）。

| デザイン要素 | 効果 |
|-----------|------|
| 黒（ベース）| 重厚さ・スケール感・プロフェッショナリズム |
| 白（テキスト）| 視認性・清潔感 |
| 赤（アクセント）| 力強さ・情熱・ダイナミズム |
| 大型タイポグラフィ「TEAM TODA」| インパクト・チームへの参加感 |
| 建設現場写真（クレーン・鉄骨）| 仕事のスケール感を視覚化 |

**「このデザインを見た瞬間に業種がわかる」** — ゼネコン採用LPとして最高の視覚的整合性。
LAVAの白×ゴールド（ヨガ/ライフスタイル）、クラスメソッドの白×イエロー（IT/スタートアップ）と同様に、
業種の「らしさ」とデザインが完全に一致している。

**建設業系カラーパレット（新規記録）**:
```
黒: #0d0d0d または #1a1a1a（重厚・スケール）
白: #ffffff（コントラスト・視認性）
赤: #cc0000 または #e60012（力強さ・情熱）
→ モノクローム + レッドアクセントが「ゼネコン・建設・製造業」採用LPの定番
```

## 「TEAM TODA」ブランドネーミングの設計

### ★新規パターン: TEAM [会社名] 型の参加型ブランディング

```
「TEAM TODA」
```

通常、大手企業の採用サイトは「〇〇株式会社 採用情報」「〇〇RECRUIT」等の機能的な命名が多い。
戸田建設は「TEAM」を冠した採用専用ブランドを構築。

**「TEAM」が生む効果**:

| 心理効果 | 内容 |
|---------|------|
| **大企業イメージの緩和** | 「大きな組織の歯車」から「チームの一員」へのイメージ転換 |
| **参加型訴求** | 「このチームに加わりませんか」という能動的な誘引 |
| **連帯感の先行提示** | 「入社前から自分もTEAM TODAの一員」という所属感 |
| **統一ブランド** | TEAM TODAとして採用LP・説明会・名刺等を統一できる |

応用テンプレート（他業種への展開）:
```
「TEAM [会社名]」（大手ゼネコン・製造業等）
「[会社名] FAMILY」（医療・介護・保育系）
「[会社名] CREW」（航空・ホテル・飲食系）
「[会社名] GUILD」（IT/クリエイター系）
```

## PROJECT STORY セクションの革新的設計

### 採用LPにプロジェクト事例を組み込む手法

「PROJECT STORY 01」と番号付きで実際のプロジェクト事例を採用LP内に配置。

**この設計の機能**:

| 機能 | 内容 |
|-----|------|
| **仕事の解像度向上** | 「ゼネコン = 大きな建物を作る」という曖昧なイメージから「具体的なプロジェクト」への変換 |
| **スケール感の訴求** | 大型案件の写真・数字で「自分もこの規模の仕事ができる」という期待感 |
| **社員の活躍の見える化** | プロジェクトを通じて「どんな人がどんな役割で活躍しているか」を具体化 |
| **採用LP × 会社実績の統合** | 会社紹介コンテンツと採用訴求を同一ページで両立 |

**「WORKS」→「PROJECT STORY」の連動**:  
ただの仕事紹介（WORKS）だけでなく、実際のプロジェクトの「ストーリー」を見せることで、
求職者が「この会社で自分はどんな物語の一部になれるか」を具体的に想像できる。

→ IND-004 新パターン: **PROJECT STORY型コンテンツ（仕事の物語化）**

## コンテンツ構成の整理された5大区分

戸田建設の採用サイトは以下の5大区分で構成:

```
WORKS（仕事・案件）
MEMBER（社員インタビュー・職種紹介）
PROJECT STORY（プロジェクト事例）
PERSONALITY（ABOUT/CULTURE — 企業理念・文化）
RECRUIT INFO（採用要項）
```

この構成は「求職者が採用決定に必要な情報の全量」を体系的に揃えている。
LAVAの「ABOUT US / MESSAGE / JOB TYPE / BLOG / ENTRY」と同様に、  
複数ページにわたる採用サイトの理想的な情報設計として参照価値が高い。

## 問題点

### 問題1: スタッフの声が「入社前後の変化」に至らない

4職種のMEMBER紹介（顔写真+職種+コピー）はあるが、
IND-004最重要評価軸「スタッフの声（20点満点）」の核心である
「顔写真 + 実名 + 入社前の不安 → 入社後の変化」が揃っているかが不明。

大手ゼネコン採用で求職者が最も不安に思う「現場は大変じゃないか」「文系でも大丈夫か」等の
不安先取りコンテンツが明示されていない。

### 問題2: 給与・待遇の不開示

「RECRUIT INFO」ページに詳細があると思われるが、
採用LPのトップページには給与情報が確認できない。
大手ゼネコンは給与水準が高い傾向があるだけに、
「給与を早期に見せる」だけで応募意欲を大幅に高められる。

### 問題3: 中盤CTAが弱い

FV右上と最終フッターにエントリーボタンはあるが、
MEMBER・PROJECT STORYを見て「応募したい」と思った瞬間のCTAが弱い。
IND-004では「FV・中盤・最終の3点セット」が必須（現状は2点セット）。

### 問題4: FAQ・応募ステップが不在

建設業・ゼネコン採用特有の不安（現場は危険じゃないか・体力は必要か・転勤は多いか等）を
先取りするFAQが確認できない。

## Learning Reason

### 保存理由
GEN-002採用系LP分析6件目。「TEAM [会社名]型参加型ブランド」「PROJECT STORY型コンテンツ」「建設業系カラーパレット」の3新規パターンを確認。デザイン統一スコアがGEN-002史上初の満点（10/10）。「デザインの完成度が高くても給与・FAQ不在で56点止まり」という飲食系OINOSと同種の構造的問題を持つケースとして記録。

### 特に優れている点
「TEAM TODA」というネーミング戦略と黒×白×赤のデザインの組み合わせはGEN-002で最も強力なブランドビジュアル。PROJECT STORYというコンテンツ設計思想は「採用LPで仕事の解像度を最大化する」手法として、ゼネコン以外の業種でも汎用的に使える。

### 他案件への応用先
- 「TEAM型ブランディング」: 製造・建設・警備・物流等の体育会系大手企業採用LP
- 「PROJECT STORY」: ゼネコン・コンサル・広告・映像制作等の「作品・案件」で語れる業種
- 「黒×赤配色」: 建設・製造・警備・防衛関連の採用LP全般

### FV評価（詳細）

**①「今だけ限定の補助金で〜」コピーの構造分析**
```
「今だけ限定」    → 希少性・緊急性（限定感）
「補助金で」     → 費用障壁の解消（国が支援してくれる安心感）
「私らしく働く」  → 自分ゴト化（ライフスタイル訴求 / 女性ターゲット共感）
「自分になる」   → 変容・成長ストーリーの予感
```
「コスト削減+自己実現」という2軸を1文に凝縮した高密度コピー。

**②「経済産業省 リスキリング支援事業」権威バッジの機能**
```
機能①: 信頼性の即時担保（政府認定＝詐欺ではないという安心）
機能②: 補助金の正当化（なぜ70%OFFが可能かの根拠）
機能③: 「今が始め時」の外部要因（制度が続く間に申し込むべき緊急性）
```
→ FV最上部に配置することで、コピーを読む前に「公的機関のお墨付き」を先に認識させる設計。

**③ 3バッジ型数値訴求の設計**

| バッジ | 訴求内容 | 機能 |
|-------|---------|------|
| 月々¥4,500〜 | 料金の明示（補助金適用後） | 価格不安の完全解消 |
| 補助金で最大70%OFF | 割引の根拠と規模 | オファーの納得感 |
| 5件案件保証 | 就業・収入への安心保証 | リスク低減 |

→ 3バッジが「安い（¥4,500〜）」「なぜ安いか（70%OFF）」「稼げるか（5件保証）」の3不安を一度に解消する設計。

**④ 写真のターゲット共感設計**
- 笑顔の女性がノートPCを操作 → 「在宅でWebデザイン」という理想像の視覚化
- 自宅らしい背景 → 「今の自分の生活に近い未来」としてリアリティがある
- ポジティブな表情 → 「自分も同じになれる」という感情移入を促進

**⑤ 1ヶ月という短期間訴求**
- 「6ヶ月」「1年」等の長期スクールより心理的ハードルが低い
- ただし「1ヶ月で在宅キャリアを目指せる」の「目指せる」は曖昧表現（「転職できる」ではない）

**FV評価: 17 / 20**

## ⑥ 参考パターン・知見抽出

### 参考パターン1: 補助金活用型オファー設計（スクール系最強オファー構造）

**構造**:
```
[権威バッジ] 経済産業省 リスキリング支援事業
       ↓
[メインコピー] 補助金で「私らしく」働く（費用+自己実現の2軸）
       ↓
[3バッジ] ¥4,500〜 / 70%OFF / 5件案件保証
```

**機能する理由**:
1. 国の制度を根拠にすることで「なぜこんなに安いのか」という疑念を解消
2. 「今だけ限定」で緊急性を演出（制度が続く間に申し込むべきという外部要因）
3. 月額¥4,500という「コーヒー代以下」の認知ラベリングが価格心理障壁を最小化

**適用可能スクール**:
- IT・プログラミング・Webデザイン・動画編集・DX研修系のすべて
- 経済産業省・厚生労働省の各種補助金・給付金対象スクール全般

**注意点**:
- 補助金制度は時限的 → 「今だけ」が本当に正しいか（誇大表示リスク）
- 補助金申請の複雑さを別セクションで解消する必要あり（手続きを代行するか否か）

### 参考パターン2: 3バッジ型FV数値訴求

**機能する理由**: FV内の3つの数値バッジが「不安の三角形」を一度に解消する。
```
不安①「高いのでは？」   → 月々¥4,500〜
不安②「なぜそんなに安い？」→ 補助金で最大70%OFF
不安③「稼げるのか？」   → 5件案件保証
```
各バッジが「次の不安への答え」になっており、バッジを読む順序に論理的な必然性がある。

**実装ポイント**: バッジは3つが上限（4つ以上は視認性・優先度が下がる）。最も重要な数字を左に置く（¥4,500〜 → コスト感覚が最初の関門）。

### 参考パターン3: 在宅ワーク志向×転職スクールの女性ターゲティング

**配色**: ライトピンク × パープル × ホワイト → 女性層の安心感・清潔感を演出  
**コピー**: 「私らしく働く自分になる」→ 女性の「自己実現×自由」という感情価値へのダイレクト訴求  
**写真**: 自宅×ノートPC×笑顔 → 「今の生活環境の延長線上に未来がある」リアリティ演出

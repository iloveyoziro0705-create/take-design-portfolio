## ③ FV（ファーストビュー）詳細分析

### FVスコア: 60 / 100（C評価）

**写真の評価:**
- モダンな都市型カフェの内観（広角・明るい・スタイリッシュ）
- 木製家具・大きな格子窓・ペンダントライト → ヨーロッパ系都市カフェの雰囲気
- お客様が実際に在席している → 賑わいと「行ってみたい感」
- 写真の品質は高い

**問題点:**
- **キャッチコピーが一切ない**（このシリーズ7サイト目で依然として多い）
- FVに料理写真がない → 飲食LPとして「何が食べられるか」がFVで伝わらない
- 店舗名「A'unique」の読み方が不明（アユニーク？ア・ユニーク？）
- 「ここが何のお店か」がFVだけでは判断しにくい（カフェ？レストラン？バー？）

**飲食LPのFV設計原則（IND-002参照）との比較:**
```
推奨: FVに看板メニューの写真 → 実現できていない
推奨: 「料理写真の品質」が最重要 → 内装写真のため未達
現状: 雰囲気は伝わるが「何を食べに行くか」が不明
```

**改善案:**
```
現状: [カフェ内装写真]（テキストなし）

改善後:
[看板料理の写真（パスタ or エスプレッソ）をFVに]
「ESPRESSO と PASTA で彩る、大阪谷町の午後。」（H1）
```

## ⑧ 複合業態の問題（最重要指摘）

このサイト最大の問題は**業態の複合性が訴求を分散させている**こと。

**確認できるサービス一覧:**
1. エスプレッソ（バリスタ）
2. パスタ
3. ピザ
4. モーニング
5. ダイニング（ディナー）
6. イベントスペースレンタル（ヨガ・ウェディング）
7. フラワーアレンジメント
8. ティーパーティ
9. オンラインショップ（物販）

**問題の本質:**
```
1つのホームページに9種のサービスを並べると…

来店目的の明確化: ✗（「何しに行くの？」が決まらない）
ターゲットの絞り込み: ✗（全員向け = 誰にも刺さらない）
LP訴求の集中: ✗（エスプレッソを押すの？パスタを押すの？）
SEO効果: ✗（「大阪 カフェ」「大阪 パスタ」「大阪 イベントスペース」が全部中途半端）
```

**設計の改善提案:**
```
解決策1: ターゲット別にLPを分ける
  - カフェ・ランチ向けLP（espresso / pasta / morning）
  - イベントスペース向けLP（wedding / yoga / flower）
  - 物販向けLP（online shop）

解決策2: ホームページでは「柱となるサービス1つ」に集中し、他はサブに
  例: 「本格エスプレッソとパスタの店」+ ヘッダーに「イベントはこちら」リンク
```

## ⑫ 今回学んだ重要知識

1. **複合業態サイトの分散問題**: 飲食+イベント+ヨガ+物販が同一LPに混在すると「何のお店か」が分からなくなる。1つのLPに1つの主役サービスを持たせる「LP分離設計」を提案できる根拠として記録。
2. **飲食LPにおける「ご予約」ボタン必須化**: このサイトには予約ボタンが一切存在しない。飲食LPの最重要CVアクションは「来店予約」であり、これがないことはCTAの致命的欠陥。飲食案件では予約ボタンをFV・メニュー直後・フッターの最低3箇所に配置する。
3. **飲食LPのFVには料理写真が必要**: このサイトのFVは内装写真のみ。飲食LPでは「内装写真 < 料理写真」として、FVに看板料理の写真を使うことで「食欲を刺激してからの来店誘導」が可能になる。
4. **MENUの左右交互レイアウト**: カテゴリ名（英語大文字）+ 大きな料理写真 + 品名リスト + MOREの左右交互配置は飲食LPのMenu設計として優れたパターン。MADAでも同様のパターンを確認しており、2件連続で有効性を確認。
5. **モザイクグリッドのサービス表示**: 5枚の異なるサイズ写真をグリッド状に並べ、各パネルにカテゴリ名とリンクを付ける手法。複数サービスを視覚的に豊富に見せる方法として有効（ただし業態混在の問題を解消してから実装すること）。
6. **飲食LPに暖色アクセントが必要**: 白×グレーのミニマリスト配色は清潔感は出るが「食欲を刺激しない」。飲食LPではウォームゴールド・テラコッタ・オレンジ等の暖色アクセントを少なくとも1色加えることを提案する。
7. **「SERVIC」タイポ事例**: 公開前に必ず全テキストの読み合わせを行う必要がある。特に英語セクション見出しはスペルミスが起きやすい。`SERVICE / CONTACT / RESERVE / CONCEPT` などの頻出英単語リストを制作前チェックリストに追加推奨。
8. **ONLINE SHOPボタンのヘッダー配置**: 物販を行う飲食・サービス業のサイトで、ヘッダーにONLINE SHOPへの導線ボタンを設置する手法。通常のナビゲーションと視覚的に差別化することで物販CVを確保。

### 他案件への流用可能な知識

1. 複合業態の案件では「LP分離設計」（カフェLP / イベントスペースLP / 物販LP）を最初に提案
2. 飲食LPの必須チェック: FV料理写真 / ご予約ボタン3箇所 / 価格明示
3. MENUセクションの「左右交互 + 大きな料理写真」レイアウトを飲食LP標準パターンとして採用
4. 白×グレーのみの飲食LP配色には暖色アクセントの追加を提案
5. 英語セクション見出しのスペルチェックを制作後・公開前に必ず実施

### 新規 Failure Pattern 候補

```
候補名: 業態分散型訴求（DilutedConcept）
実例: A'unique — カフェ+パスタ+ピザ+イベント+ヨガ+ウェディング+物販が1ページに混在
発生状況: 多様なサービスを提供する複合業態が、全サービスを1つのLPに詰め込むケース
CVへの悪影響: 「何の店か」が分からない → どのターゲットにも刺さらない
          「来店目的」が決まらない → 予約・来店の決断が遅れる
対策:
  1. メインサービス（カフェ・ランチ）に絞った1LPを作る
  2. イベント・スペースは別URLのLPに分離する
  3. ホームページはポータルとして機能させ、LPはターゲット別に分ける
failure-patterns.md 追加: 推奨
```

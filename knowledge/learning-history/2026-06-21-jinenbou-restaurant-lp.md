## 優れた点

### 1. キャッチコピーの完成度（★GEN-003最優秀コピー候補）

```
「ひとつひとつ丁寧に揚げる"とんかつ"。
  やわらかく、甘みのあるお肉をどうぞ。」
```

このコピーの構造分析:

| 要素 | コピー部分 | 効果 |
|-----|-----------|------|
| 職人プロセス語 | 「ひとつひとつ丁寧に揚げる」| 工程の丁寧さ・手仕事感を言語化 |
| 専門性宣言 | 「"とんかつ"」| カギ括弧で「本物のとんかつ」という哲学的宣言 |
| 食感語 | 「やわらかく」| 触覚を直接喚起 |
| 味覚語 | 「甘みのある」| 味覚を直接喚起 |
| 邀請語 | 「お肉をどうぞ。」| 「どうぞ」の温かみで緊張感なく誘引 |

**「食感語 × 味覚語の複合感覚訴求」** は GEN-003で初確認のパターン。  
「やわらかく（触覚）、甘みのある（味覚）」の組み合わせは視覚媒体で2種の感覚を同時喚起している。

とりすけ「今夜も幸せ 焼き鳥時間」= 感情語×時間軸  
煮干しらーめん「ガツッと薫る」= 嗅覚語×体感語  
**自然坊「やわらかく、甘みのある」= 食感語×味覚語** ← 新タイプ確認

### 2. 「"とんかつ"」カギ括弧による哲学的宣言

「"とんかつ"」という表現はとんかつを固有名詞・哲学的概念として扱う表現。  
「私たちにとってのとんかつとは何か」という店の哲学宣言であり、ブランド名「自然坊」の哲学型命名と呼応している。

**ブランド命名分析**:
「自然坊」= 「自然（nature）+ 坊（人・者の意）」= 「自然に従う者・自然を大切にする者」  
MADAの「BE YOURSELF」やHériterの「継承」と同様の **哲学型ブランド命名**。  
店名だけで「人工的でない・素材を活かす」という価値観が伝わる。

→ 美容系で分析したブランド命名類型を飲食系にも拡張して記録

### 3. とんかつ超アップ写真のFV活用

とんかつのFV写真は「揚げたての衣・肉の断面・大皿の盛り」が揃った高水準の食欲訴求。  
串焼き（慶鳥・とりすけ）と同様の「極近接食材写真」が最大の食欲喚起効果を発揮している。

### 4. 和紙テクスチャー × ダークブラウン配色

慶鳥と同系統の配色だが、慶鳥より「素朴・ナチュラル」寄りの印象。  
「自然坊」というブランド名の「自然さ」を配色でも表現しており、ブランドコヒーレンスが高い。

## 問題点

### 問題1: コピーがFV内に存在しない（FAIL-004 部分該当）

最も優れたコピー「ひとつひとつ丁寧に揚げる...」がFV**下**のセクションに配置されている。  
FVは写真のみで、訪問者はスクロールしないとこのコピーに出会えない。

**「FVにコピーがない = 3秒での離脱リスク」**

慶鳥の場合: FVに写真はあるがコピーなし → 65/B-  
自然坊の場合: 同じ問題 + ただしコピー自体の品質は高い → 65/B-（同点）

**改善案**: FV写真の上にコピーをオーバーレイする
- FVに「やわらかく、甘みのある"とんかつ"。」をテキストオーバーレイで追加するだけで+7〜8点

### 問題2: 価格表示なし

とんかつのランチ/ディナー価格が不明。  
「自然坊」というブランド名から「高級とんかつ店」とも「リーズナブルな専門店」とも読めるため、価格帯の明示が特に重要。

### 問題3: ネット予約がない

電話番号のみ。とんかつ専門店は「ランチ待ち問題」があるため、  
「ランチネット予約」「事前電話予約」の動線を強調すると来店率が上がる。

## 新規パターン（IND-002追記対象）

### パターン1: 食感語×味覚語の複合感覚訴求コピー

```
「やわらかく（触覚）、甘みのある（味覚）お肉をどうぞ。」
```

GEN-003で確認した感覚語コピーの新タイプ:
- 嗅覚語: 「ガツッと薫る」（煮干しらーめん）
- 動感語: 「焼肉踊る」（焼肉どんどん）
- 食感+味覚語: **「やわらかく、甘みのある」**（自然坊 ← 新確認）

飲食系コピーの感覚語分類:
| 感覚 | 例 | 効果 |
|-----|---|------|
| 嗅覚 | 「薫る」「漂う」「香ばしい」| 料理を前にした瞬間の記憶喚起 |
| 触覚（食感）| 「やわらかく」「とろける」「サクサク」| 口の中の体験を先行提示 |
| 味覚 | 「甘みのある」「旨みが凝縮」「深みのある」| 味の記憶を直接喚起 |
| 動感 | 「踊る」「弾ける」「広がる」| 動的な体験への期待感 |

### パターン2: 職人プロセスコピー「ひとつひとつ丁寧に」

「ひとつひとつ」「一枚一枚」「毎日手作り」等の工程の丁寧さを言語化するコピーパターン。  
「こだわり」という抽象語より、**動詞+工程名** での表現が説得力を持つ。

応用テンプレート:
- 「ひとつひとつ [工程] する "専門料理名"」
- 「毎朝 [工程] から作る [料理名]」

### 特に優れている点

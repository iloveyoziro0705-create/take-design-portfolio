## ③ ブランドコヒーレンス（今回の最重要発見）

### ブランドコヒーレンス評価: A

このサイト最大の優良要素。「Little Garden」というサロン名から始まるテーマの一貫性が非常に高い。

**一貫性マップ:**

| 要素 | テーマとの整合性 |
|------|--------------|
| サロン名 | Little Garden → そのものずばり |
| コンセプト名 | Natural & Garden → 直接的に反映 |
| FV写真 | 観葉植物・花・ナチュラル素材の内装 → 完全一致 |
| 家具選定 | アンティーク調・木材・革 → ガーデン＆自然素材 |
| バードケージ装飾 | 庭の小鳥のイメージ → テーマ強化 |
| コンセプトコピー | 「みんなが自然に集まってくる温かい場所」→ 庭の憩い場 |
| フォント | 筆記体（手書き風）→ ナチュラル・手作り感 |
| 配色 | 白・ベージュ・ウッド系 → ナチュラルトーン |

**なぜブランドコヒーレンスが重要か:**
```
コヒーレンスが高い = サイトを見るだけでどんなサロンか瞬時に理解できる
コヒーレンスが低い = 「オシャレな写真だけど何が特徴のサロン?」と混乱する

Little Gardenは写真を見ただけで「Natural & Garden系のおしゃれサロン」と
0.5秒で理解できる。これがブランドコヒーレンスの力。
```

**ウェブLP制作への応用:**
```
LP設計時の確認事項（Brand Coherence Check）:
□ サロン名・ブランド名のテーマはなにか
□ そのテーマがFV写真に反映されているか
□ そのテーマがコンセプトコピーに反映されているか
□ そのテーマが配色に反映されているか
□ そのテーマがフォント選定に反映されているか
□ 全部揃っていれば「ブランドコヒーレンス高」
```

## ⑤ イルミナカラー サポート契約サロン（重要発見）

### 製品提携明示スコア: A

**「イルミナカラー」とは:**
- ウエラ（Wella）社の高品質ヘアカラー商品
- 「髪のダメージを最小限におさえ、ツヤある仕上がり」が特徴
- 「イルミナカラー サポート契約サロン」= 認定を受けた正規取扱サロン

**なぜこれが重要な設計要素か:**

| 効果 | 内容 |
|------|------|
| 第三者認定による信頼性 | 「ウエラ社が認定した」という第三者のお墨付き → クライアントが言うより説得力高 |
| 技術力の証明 | サポート契約には技術習得が必要 → スタッフの技術力を暗示 |
| 高品質素材の訴求 | 「髪にやさしい」ことを商品ブランドで証明 |
| 検索流入の可能性 | 「イルミナカラー + 地域名」で検索するユーザーを取り込める |

**ウェブLP制作への応用（製品・ブランド提携明示のパターン）:**
```
美容室: イルミナカラー / ミルボン / TOKIO IE / ケラスターゼ 等の取扱い明示
エステ: 〇〇エステ認定/資格 / 使用機器のブランド
歯科: 認定医 / 学会会員 / 使用材料のブランド
整骨院: 柔道整復師資格 / 各種保険対応
```

## ⑪ 今回学んだ重要知識

1. **ブランドコヒーレンス設計（このシリーズ最重要発見）**: 「Little Garden」という命名から始まり、テーマ（Natural & Garden）・内装写真（植物・花・木材）・コピー（「自然に集まる場所」）・フォント（筆記体）・配色（ナチュラルトーン）が完全統一。これがLP設計で最初に決めるべき「テーマの一貫性」の最高水準例。Brand Coherence Check（テーマ確認チェックリスト）をLPプロセスに組み込む。
2. **製品・ブランド提携の明示による信頼性向上**: 「イルミナカラー サポート契約サロン」等の第三者認定・正規取扱の明示は、技術力の証明として非常に有効。美容室案件では取扱ブランド・認定資格・使用製品の明示を必ず提案する。
3. **「友だちの家に来たような」コピー**: 美容室の緊張感・「失敗したら怖い」という不安を、「友人宅の安心感」で置き換えるコピー手法。アットホーム系サロン全般に転用可能な秀逸な比喩表現。
4. **「仲間の手づくり」コピー**: DIY・手作りの要素をコピーで明示することで「既製品感のない温かみ」を演出。飲食・美容・ハンドメイド系に転用可能。
5. **Facebook埋め込み vs Instagram埋め込み**: 2017年頃はFacebook埋め込みが主流だったが、2026年現在はInstagramへの移行が必須。既存サイト改修時にFacebook→Instagram切り替えを提案。
6. **プレースホルダー公開問題（別パターン）**: RUKUEはニュース更新「停止」、Little GardenはNewsが「最初からプレースホルダーのまま公開」。どちらも信頼性を損なうが、Little Gardenは「未完成のまま公開してしまった」ケース。公開前チェックリストとして「テキストにサンプル文字が残っていないか」確認を標準化する必要がある。
7. **グランドオープンバッジとFVの組み合わせ**: Liceriaフライヤー同様、グランドオープン時は「GRAND OPEN」バッジをFVに入れることが有効。ただし開業後は削除するタイミングの設計も必要（いつまでも「GRAND OPEN」はおかしい）。
8. **ウェディングスタイル対応の暗示（FV写真への混入）**: FV写真の背景に「Welcome to our WEDDING PARTY」ボードが映っており、ウェディング前撮りや成人式等のハレの日ヘアスタイル対応を無言で訴求。言語化せずに対応サービスを伝える視覚的手法。
9. **筆記体フォントのブランディング効果**: ロゴ・セクションタイトルに筆記体（手書き風）フォントを使用することで「ナチュラル・手作り・温かみ」のブランドイメージを強化。ナチュラル系・ガーデン系・アンティーク系サロンに有効。

### 他案件への流用可能な知識

1. **Brand Coherence Check** — LP制作開始時に「サロン名 → テーマ → 写真 → コピー → 配色 → フォント」の一貫性チェックリストを実施
2. **製品提携明示** — 美容室は「イルミナカラー」「ミルボン」「TOKIO IE」等の取扱い明示を提案。他業種も同様の認定・資格を必ず発掘
3. **「友だちの家」型コピー** — アットホーム系サロン・カフェ・習い事スクール等に「〇〇に来たような安心感」のコピー形式を転用
4. **既存Facebook埋め込みのInstagram移行** — サイト改修案件でFacebook埋め込みを発見したら即Instagram移行を提案
5. **グランドオープン時のFV設計** — 「GRAND OPEN」バッジ + サロン内装の高品質写真 + テーマコピーの組み合わせを定番パターンとして保有

## Failure Pattern 候補（新規）

```
候補名: 開業後もGRAND OPENバッジが残存（StaleOpenBadge）
実例: Little Garden（グランドオープン時のサイトが長期運用されている可能性）
発生状況: グランドオープン時にFVに「GRAND OPEN」バッジを設置後、削除するタイミングを
         設定せず何年も表示し続けるケース
CVへの悪影響: 「本当に今もオープンしてるの？」「何年前のサイト？」という疑念
対策: グランドオープンLPを制作する際は、バッジ削除の期限（例: オープン3ヶ月後）を
     クライアントに事前に共有し、スケジュールに入れる
推奨: failure-patterns.md に追加検討
```

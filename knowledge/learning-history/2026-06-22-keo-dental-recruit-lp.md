## 最大の発見: 「採用は文系9割」というミスマッチ解消コピー

### ★新規パターン: 業種イメージ × 採用ターゲットのギャップを直接言語化

```
「採用は文系9割 多くの社員が活躍しています」
```

**このコピーが解消する心理的障壁**:

歯科業界は一般的に「医療知識が必要」「理系でないと入れない」という強いイメージがある。
しかしケーオーデンタルは「歯科材料の商社 = 営業職」が主力。

| 求職者の先入観 | コピーの解消方法 |
|-------------|---------------|
| 「医療系の会社 = 理系が有利」 | 「文系9割」という数字で明確に否定 |
| 「歯科の知識がなければ無理」 | 「多くの社員が活躍」で「知識より人柄・営業力」を示唆 |
| 「私には関係のない求人」 | 「文系9割」を見た瞬間に「自分でも応募できる」に転換 |

**応用パターン（GEN-002で使える）**:
```
「エンジニア採用の○割は文系出身です」（IT系採用LP）
「介護職の△割は異業種からの転職者です」（介護系採用LP）
「工場スタッフの□割は未経験スタートです」（製造系採用LP）
```

→ **業種固有の「参入障壁イメージ」を数字で打ち破るコピーパターン**

**効果**: 「この会社で何をするか」を視覚的に即座に伝える。
商社（営業）× 歯科（専門性）という異色の組み合わせを1枚で示す。

## 問題点

### 問題1: FVコピーがミッション型止まり（FAIL-004）

「歯科医療を もっとよく、もっと楽しく」は会社のミッション宣言であり、
求職者の悩みへの共感がない。

| コピーの型 | 現在 | 推奨 |
|---------|------|------|
| 現在 | ミッション型（会社の使命を表明）| — |
| 理想 | 悩み直撃型 or 理想提示型（求職者目線）| 「文系でも、歯科業界で活躍できる。」|

**改善案（FVへの追加コピー）**:
```
「文系でも、歯科業界で活躍できる。
  ケーオーデンタルの採用は文系9割。
  あなたの"伝える力"が、患者さんの笑顔につながります。」
```

「採用は文系9割」という強力なメッセージがPERSONセクションに押し込まれており、
**FVに配置すれば7〜8点のスコアアップポテンシャルがある**。

### 問題2: 給与情報が完全不在

IND-004禁止パターン最重要項目。
歯科材料商社の「営業職」の給与水準が全く不明。
「数字で知るケーオーデンタル」セクションに数字はあるが、給与ではない。

### 問題3: 「お客様インタビュー」がComing Soon

お客様（歯科医院）インタビューが「Coming Soon」表示。
信頼性要素が半分しか揃っていない状態でLPを公開している。
社員インタビューも「入社後のきっかけを紹介」という説明のみで、
入社前の不安→入社後の変化の具体性が不明。

### 問題4: FAQ・応募ステップが不在

歯科材料商社への応募ハードルは「医療知識がなくて大丈夫か」「どんな研修があるか」等。
これらを先取りするFAQがなく、応募フローも確認できない。

## 「業界体験型コンテンツ」という新しい設計思想

ケーオーデンタルの「歯科医療を支える私たちの仕事」セクションは、
採用LPとして独特の設計思想を持っている。

| サブセクション | 内容 | 設計の目的 |
|-------------|------|----------|
| 「業界を知る」 | 歯科業界の全体像 | 「まず業界を理解させる」 |
| 「仕事を知る」 | 具体的な営業の仕事内容 | 「仕事内容への納得感」 |
| 「現場を支える私たちの仕事」 | 歯科医院への直接サポート | 「社会貢献性の実感」 |
| 「データで知る」 | 会社の規模・実績 | 「数字で信頼感を構築」 |

通常の採用LPは「応募させる」ことが最優先。しかしケーオーデンタルは
**「業界・仕事を深く理解させてから応募させる」**長期育成型の設計。

**この設計の効果と課題**:
| | 評価 |
|--|------|
| 効果: 入社後ミスマッチを防ぐ（「思ってたのと違う」を減らす）| ○ |
| 効果: 業界への志望意欲が低い潜在層を育てられる | ○ |
| 課題: 「今すぐ応募したい」転職者には冗長に感じる可能性 | △ |
| 課題: コンテンツが多くCTAまでの距離が長い | △ |

→ **「採用育成型LP」**（採用LPの新しいカテゴリ）として記録価値あり

## 新規パターン（IND-004追記対象）

### パターン: 業種参入障壁イメージを数字で打ち破るコピー

```
「採用は文系9割」
「採用の〇割は未経験スタート」
「△割が異業種からの転職者です」
```

**適用シーン**: 「専門知識が必要そう」「自分には無理かも」という先入観が強い業種の採用LP
- 歯科・医療 → 「文系9割」
- IT/エンジニア → 「文系出身〇割」
- 建設・製造 → 「未経験スタート〇割」
- 介護・福祉 → 「異業種転職〇割」

数字で示すことで「私でも応募できる」という許可を明示的に与える効果がある。

## Learning Reason

### 保存理由
GEN-002採用系LP分析3件目。「採用は文系9割」という業種参入障壁を数字で打ち破るコピーパターンを初確認。また「採用育成型LP」（応募させるより業界理解を深めさせる）という新しい採用LP設計思想を発見。

### 特に優れている点
コーポレートカラー（ビビッドブルー）の徹底活用によるブランドコヒーレンスはGEN-002最高水準（9/10）。「文系9割」という採用ミスマッチ解消コピーは業界イメージのある全業種で応用できる汎用性の高いパターン。

### 他案件への応用先
- 「文系9割」パターン: 医療・IT・建設等の専門職イメージが強い業種の採用LP全般
- 「採用育成型コンテンツ」: 業界理解が必要な専門商社・BtoB営業職の採用LP
- 「新卒・中途分岐型CTA」: 両者が応募対象の採用LP設計時のUXパターンとして参照

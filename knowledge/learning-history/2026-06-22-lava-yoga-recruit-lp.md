## ★「好きを 仕事に。」コピー解剖

### GEN-002史上最高品質FVコピー

```
「好きを 仕事に。」
```

このコピーの構造:

| 要素 | 分析 |
|-----|------|
| 「好き」 | 求職者自身の内発的動機（ヨガへの愛着）|
| 「を」 | 「好き」を主語化。受動でなく能動の意志を示す |
| 「仕事に。」| 「仕事にしてもいい / できる」という許可と確信の提示 |
| 句点「。」 | 断言。希望でなく「これが実現できる場所」という宣言 |

**なぜこのコピーが機能するか**:

1. **ヨガLPとしての完璧なマッチング**: LAVA のインストラクター・スタッフはヨガファンが多い。「好き」がそのまま自分ごとになる。
2. **「仕事にしていいのか？」という疑問への回答**: ヨガが好きな人の多くが「趣味で終わらせていいのか」と迷う。「仕事に。」という断言がその迷いを吹き飛ばす。
3. **理想提示型（IND-004 パターンB）の最高峰**: 「理想の未来（好きなことを仕事にする）」を5文字で提示する圧縮技術。

GEN-002の採点ガイドでは「15点: 悩み共感コピー明確 + ターゲット明示 + 視覚的インパクトあり」→ ほぼ最高水準。  
悩み共感型ではなく**理想提示型**で高スコアを出した初ケース（IND-004のパターンB最高例）。

## FVに先輩スタッフが登場する設計

### 採用LP設計の重要発見

通常の採用LPでは「先輩スタッフの声」は中盤以降のセクションに配置される。
LAVAはFVの右側に**「2027年卒 先輩 まどか」**の顔写真+名前を配置。

**この設計の効果**:
| 効果 | 内容 |
|-----|------|
| 「親近感の即時形成」| FVを見た瞬間に「一緒に働く人の顔」が見える |
| 「本物のスタッフが働いている実在感」| 写真撮影という投資が「このサイトは本物」という信頼に変換される |
| カルーセル5名（S/5）| スタッフが複数いることを視覚化。多様な先輩がいる安心感 |

**「FVに先輩の顔がある採用LP」は少数派**。  
他の分析3件（ロボットペイメント・K.O.ケーオーデンタル・BtoB Cloud）では全てスタッフの顔はFV外。  
LAVAだけがFV内でスタッフ個人を「顔+名前」で見せている。

→ IND-004 推奨設計として追記価値あり: **「FVにスタッフ個人の顔+名前を入れる」**

### 問題2: LP内FAQ・応募ステップが不在

ナビゲーションに「よくある質問」はあるが、LP内（1ページ内）にFAQが表示されていない。

不安先取りFAQをLP内に配置すれば+3〜4点のポテンシャル。

## 「ライフスタイル採用LP」という新カテゴリ

GEN-002の分析を4件行った時点で、採用LPは大きく2タイプに分類できる:

| タイプ | 代表例 | 訴求軸 | FVコピー傾向 |
|------|------|------|------------|
| **キャリア訴求型** | ロボットペイメント・K.O.ケーオーデンタル | 給与・データ・成長・業界知識 | 数字・実績・理性訴求 |
| **ライフスタイル訴求型** | LAVA（本件）| 好き・価値観・ブランド共感 | 感情・理想・内発動機訴求 |

**ライフスタイル訴求型の特徴**:
- 「この仕事が好き」から入る → 応募意欲が高い人に直接刺さる
- ブランドデザイン・写真の質が信頼性の核
- 「給与より価値観の一致」を重視する層（ヨガ・アウトドア・アパレル等）で有効

**逆説的弱点**: 「好き」が強調されすぎると「給与は二の次でいい」という誤解が生まれ、  
待遇開示への意識が低下するリスクがある（実際に給与スコアが4/15と低い）。

## 新規パターン（IND-004追記対象）

### パターン: FVにスタッフ個人の顔+名前を配置する設計

通常「先輩スタッフ」は中盤セクション（スタッフの声）に登場するが、  
FV右側に顔写真+名前（+入社年度・職種）を置くことで**「最初の1秒で人柄が伝わる」**採用LPになる。

**配置案**:
```
[FV左: コピー + エントリーCTA]
[FV右: 先輩スタッフの顔写真 + 名前 + 職種 + カルーセル（複数名）]
```

**効果**: 採用決定の最大要因「本当にいい職場か」への答えを、  
スクロールなしで1秒で伝える。離脱率の大幅な低下が期待できる。

### パターン: ライフスタイル訴求型FVコピー

```
「好きを 仕事に。」（LAVA）
「[価値観/ライフスタイル] を、仕事に。」（展開パターン）
「[好き/情熱] から始まる、[仕事/キャリア]。」
```

ヨガ・アウトドア・アパレル・飲食・音楽等の「業種自体に熱量がある業界」で有効。  
内発的動機（好き・情熱）を仕事の原動力として打ち出す。

## Learning Reason

### 保存理由
GEN-002採用系LP分析4件目。初めてB-（68点）が登場し、前3件のC〜C+（47〜50点）と大きな差をつけた。差の核心は「FVコピーの質」と「FVにスタッフ個人の顔+名前を配置する設計」の2点。「好きを 仕事に。」はGEN-002で確認した中で最高品質のFVコピー。また「ライフスタイル訴求型採用LP」という新カテゴリを定義した。

### 特に優れている点
「好きを 仕事に。」5文字のコピーで内発的動機に直接訴求する圧縮技術。FVにスタッフカルーセルを配置して「最初の1秒で人の顔が見える」設計。白×ゴールドのブランドデザインとヨガ・ウェルネスブランドの完全な一致（コヒーレンス9/10）。

### 他案件への応用先
- 「好きを仕事に」型コピー: スポーツ・アウトドア・アパレル・飲食・音楽・ゲーム等「熱量がある業界」全般
- 「FVにスタッフ顔+名前」設計: 全採用LPで応用可能な最重要推奨設計
- 雇用形態5区分エントリー: 正社員・中途・アルバイト・業務委託を全方位で受け入れたい場合

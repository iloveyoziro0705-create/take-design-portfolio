## ③ FV評価（詳細）

### ①「Design Your Future.」英語コピーの問題

GEN-001（美容系）で繰り返し確認してきたパターン：**英語メインコピーはCV訴求力が弱い**。

```
「Design Your Future.」= 「あなたの未来をデザインしよう」
→ 意味は伝わるが「何が得られるか（アウトカム）」が抽象的
→ 「3ヶ月でWebデザイナーへ転職」「卒業生の転職成功率92%」等の
  具体的成果訴求に比べてCVへの直結力が著しく低い
```

デジタルハリウッドは大手ブランドであり、ブランド認知で補える部分があるが、**知名度のない中小スクールが同じ設計をすると致命的**。

### ②「Wスクール」コンセプトの不明瞭さ

FVに「Wスクールで〜」と書かれているが、**「Wスクール」が何を意味するかFVだけでは全く分からない**。

考えられる意味:
- A. 2つのスクールを掛け持ちする
- B. Wの字形を使った比喩（ダブルで学ぶ）
- C. スクール名の一部

LP冒頭のキーコンセプトが即座に理解できない → **ユーザーは「？」のまま次のセクションを読む動機が下がる**。

### ③ ハッシュタグ形式訴求の功罪

```
#やりたい仕事を見つけよう → ターゲットの悩みを代弁している（良い）
#文系デザイナー → 「文系でもできる」という参入障壁の解消（良い）
```

ただし、ハッシュタグ形式はSNS投稿感が強く、LPの「成果を保証する」という権威性とは相性が悪い。**同じ内容を「文系出身・完全未経験から3ヶ月でWebデザイナーへ」というコピーにした方がCV向上効果は高い**。

### ④ 4名コラージュ写真の多様性表現

```
強み: 「あなたと同じような人も通っている」多様性の表現
弱み: ターゲットが絞れず「誰向け？」の焦点が拡散する
```

スクール系LPでは、ターゲットを絞り込んだ「この人向け」という訴求の方がCVRは高い傾向がある（GEN-006-001「1期生転職成功率100%」のような単一成功事例の方が刺さる）。

### ⑤ 「無料スクール相談会」CTAの評価

- 「相談会」はハードルが低いCTA文言 → 良い
- ただしFVのCTAボタンが「詳細はこちら」と曖昧 → 「無料スクール相談会に申し込む」の直接文言の方がCV率が高い
- FV内にCTAが存在する点は合格

## ⑥ 参考パターン・知見抽出

### 参考パターン1（反面教師）: 大手ブランド依存型LP設計のリスク

デジタルハリウッドは業界内では有名スクールだが、このFV設計では：
- 英語メインコピー（訴求力弱）
- 概念的コンセプト「Wスクール」（即時理解不可）
- 成果数字なし

**同じ設計を中小スクールが採用した場合、ブランド力がない分だけ大幅なCV低下が起こる**。大手の「なんとなく成立するLP」は中小スクールの設計参考にしてはいけない事例として記録。

### 参考パターン2: ハッシュタグ形式の転用可能な訴求

「#文系デザイナー」というハッシュタグは「文系でもできる」という参入障壁解消として機能している。この訴求自体は有効だが、**コピーとして改変すれば訴求力が増す**。

```
NG（現状）: #文系デザイナー
OK（改変）: 文系出身・完全未経験でも3ヶ月でデザイナーになれる理由
```

ハッシュタグ形式はFVより「共感セクション」や「SNS投稿引用」に使う方が効果的。

### 参考パターン3: 「全額・ポイント可」支払い柔軟性バッジ

FV内で「全額・ポイント可」と支払い方法を明示している点は**支払い障壁の解消**として機能している。スクール系（高額商品）では有効な訴求。GEN-006の CV向上ポイントに追加候補。

## Learning Reason

### 保存理由
GEN-006-004として、大手スクール（デジタルハリウッドSTUDIO）のFV設計を記録。大手ブランドだからこそ見落とされがちな設計上の問題（英語コピー・概念的コンセプト・数字訴求なし）を「反面教師パターン」として記録することに価値がある。中小スクールのLP制作時に「大手と同じ設計にしてはいけない」ケーススタディとして参照できる。

### 再利用価値
- 「英語コピー型FVのリスク」を説明する根拠事例
- Webデザイン・クリエイティブ系スクールのターゲット設計参考

### 特に優れている点
「無料スクール相談会」というCVアクションの選択は適切（セミナー型の低ハードルCTA）。

### 他案件への応用先（反面教師として）
- 英語コピーを使いたいスクール → 「Design〜」ではなく「3ヶ月でWebデザイナーへ」の方が刺さる
- 「Wスクール」のような造語コンセプト → FVで使うには即時理解できる説明が必要

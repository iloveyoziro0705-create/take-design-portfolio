### FV評価（詳細）

**①「女性専用オンラインスクール」バッジの差別化力**

| 比較 | 表現 | 差別化強度 |
|-----|------|-----------|
| SHE likes（GEN-006-005） | 「女性向けキャリアスクール」 | 中（向けは排他性が弱い） |
| リモラボ（GEN-006-010） | 「女性専用オンラインスクール」 | **高**（専用 = 完全な安心空間） |

→「女性専用」は「女性向け」より排他性が強く、「この場所は完全に女性だけのコミュニティ」という**安心・安全訴求**として機能する。育児中・在宅志向の女性に強く響く。

**② 4職種アイコンタグ（SNS運用/Webデザイン/動画制作/オンライン秘書）**

```
強み: 上部に4職種を列挙 → 「何を学べるか」が即座に分かる
     SHE likesと同様のアウトカム視覚化パターン
弱み: 写真＋ラベルではなくアイコン＋テキストのみ → 視覚的インパクトがSHE likesより弱い
     FV上部の小さなエリアに配置 → 主役になっていない（モデル写真が主役）
```

→ SHE likes（GEN-006-005）の「4名写真＋職種ラベル」と比べると視覚的訴求力は劣る。ただし存在自体は機能している。

**③ メインコピー「理想の働き方を実現する」評価**

```
「スキルを身につけて視野を広げ」← プロセス訴求（どう変わるか）
「理想の働き方を実現する」    ← アウトカム訴求（何が実現するか）
```

- **強み**: 2段階（プロセス→アウトカム）の流れがある / ターゲット（働き方を変えたい女性）の欲求に直接刺さる
- **弱み**: 「理想の働き方」が抽象的すぎる（転職率○%・月収○万円等の具体数字なし）
- **比較**:
  - ZERO ENGLISH（GEN-006-009）の「英語力ゼロからでも60日で話せる」→ 具体的・数字あり
  - リモラボ（GEN-006-010）の「理想の働き方を実現する」→ 抽象的・数字なし

→ コピーとして情緒的には機能するが、競合との差別化力・信頼性補完には弱い。

**④ 2軸数字バッジ「累計受講者数6,100名以上 / 会員継続率95.1%」**

| バッジ | 数字 | 訴求内容 |
|-------|------|---------|
| 累計受講者数 | 6,100名以上 | 社会的証明（多くの人が選んでいる） |
| **会員継続率** | **95.1%** | **サービス品質の間接証明**（辞めない = 満足している） |

→ **「会員継続率」は本シリーズ（GEN-006）で初確認の新指標**。
- 通常スクールは「転職成功率」「合格率」「満足度」等を使うが、継続率は「現在も使い続けている会員の割合」
- 「一度入って満足できなければ辞める」というスクール特有の離脱懸念を数字で解消
- 「95.1%が継続」= コンテンツ品質・コミュニティの良さの強力な証明
- ローレルリース（月桂冠）デザインが権威感を付加

**⑤ リードジェネレーション型CTA「ロードマップ動画を受け取る」**

```
[CTA前コピー] SNSスキルを身につけ、安定して稼ぐ！← ベネフィット訴求
[CTAボタン]  ロードマップ動画を受け取る ▶        ← 行動指示
```

- 「申し込む」「登録する」ではなく「**受け取る**」→ 受け身で低ハードルの文言
- 「ロードマップ動画」を無料で配布 → メールアドレス取得（リードジェネ）→ナーチャリング→スクール申し込みのファネル設計
- このCVアクションは「スクール体験申し込み」より入口がさらに低い

**このファネル設計の意味**:
```
一般的スクールLP:  LP → 無料体験申し込み → 入塾
リモラボのLP:    LP → 無料動画受け取り（メルマガ登録）→ 興味醸成 → 入塾
```
→ コンバージョン率を上げる（動画受け取りは体験申し込みより心理的ハードルが低い）が、ナーチャリングに工数がかかる設計。LP単体の転換率は高くても、実入塾率はファネル後半次第。

**⑥「3月31日まで限定無料配布！」期限設定**

- カウントダウンタイマー（SHE likes）より弱いが、「3月31日」という具体的な日付で「期限を意識させる」
- 「限定」という言葉が希少性を演出
- CTAの理由付け（今すぐ動画を受け取るべき理由）として機能

**FVスコア: 15/20**

## ⑤ 参考パターン・知見抽出

### 参考パターン1: 「会員継続率」による品質証明（GEN-006初発見）

**構造**:
```
[ローレルリースバッジ]
会員継続率 95.1%
※2024年2月時点
```

**機能する理由**:
- 「入ったら辞めない」= コンテンツ品質・コミュニティの居心地良さの間接証明
- スクール選びで潜在する「続けられるか」「途中で辞めてしまうのでは」という離脱懸念を数字で解消
- 転職成功率や合格率と異なり「運用中の事業のリアルな数字」として信頼性が高い

**活用場面**:
```
スクール系全般 / サブスクリプション型サービス / オンラインコミュニティ
```

**組み合わせで最強になるパターン**:
```
累計会員数（入口の多さ） ×  会員継続率（品質の高さ）
= 「多くの人が入って、みんな辞めない」という強力な二重証明
```

### 参考パターン2: リードジェネレーション型CTA（動画受け取り）

**構造**:
```
[CVアクション] ロードマップ動画を受け取る
[CTAファネル] LP → 動画登録（メルマガ）→ ナーチャリング → 入塾
```

**通常CTAとの比較**:
| CTA種別 | 心理的ハードル | 転換率 | 入塾までのステップ |
|--------|-------------|--------|-----------------|
| 「入塾申し込み」 | 高 | 低 | 1ステップ |
| 「無料体験」 | 中 | 中 | 2ステップ |
| **「動画受け取り」** | **低** | **高** | **3ステップ（要ナーチャリング）** |

→ LP単体のCV率は高いが入塾まで時間がかかる設計。認知・検討期の女性ユーザーを獲得する戦略として有効。

### 参考パターン3: 「女性専用」vs「女性向け」の差別化

**2つの表現の違い**:
```
「女性向け」（SHE likes）: 対象を示す → 男性も入れる（だが女性がターゲット）
「女性専用」（リモラボ）: 完全限定 → 男性は入れない（安全・安心空間を保証）
```

**「女性専用」が刺さるシチュエーション**:
- 在宅ワーク希望者（家族のいる環境でのオンライン学習）
- 育児中・産休中の女性
- 「男性がいるとやりにくい」という悩みを持つ女性
- コミュニティ参加をためらっている内向型の女性

**実装ポイント**: 「女性専用」を使う場合は、ロゴ直前に「女性専用○○スクール」タグを配置し、最も目立つ位置に持ってくること。

## Learning Reason

### 保存理由
GEN-006-010として、リモラボ（女性専用オンラインスクール）のFV設計を記録。「会員継続率95.1%」という本シリーズ初の継続率指標と、「動画受け取り型リードジェネCTA」の2つは新パターンとして辞典統合候補。特に「会員継続率」はスクール系LPにおいて「量（受講者数）× 質（継続率）」の二重証明を実現するパターンとして参照価値が高い。

### 再利用価値
- 女性専用スクール・コミュニティ全般
- サブスクリプション型サービス（継続率指標）
- リードナーチャリング型ファネルを持つスクール・サービス
- 「入口を低くして後でナーチャリング」設計のLP全般

### 特に優れている点
「会員継続率95.1%」という指標は、転職成功率・合格率とは異なり「スクールの現在の顧客満足度のリアルな数字」として機能する。「入った人が辞めない」という事実は、「体験申し込みをするか迷っている人」の最後の不安（続けられるか）を直接解消するため、CVR向上効果が高い。

### 他案件への応用先
- オンラインスクール全般（「卒業生の〇%が継続利用」等）
- ジム・習い事スクール（「継続率○%」「平均在籍期間○ヶ月」）
- サブスク型サービス（「解約率○%未満」の裏返しとして）

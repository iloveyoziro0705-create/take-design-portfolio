## ⑧ 参考パターン・知見抽出

### 参考パターン1（★高優先）: 「悩み共感チェックリスト → 解決宣言」ブリッジ構造

**構造**:
```
「あなたにこんな英語力の悩み 抱えていませんか？」
□ 何をやっても英語が上手くならない
□ お金と時間だけが増えていくだけ
□ 海外旅行で英語が喋れなくて困った
□ TOEIC点数が上がらない
□ 社交的に英語が使えなくて困っている
□ だんだん英語が嫌いになってきた
          ↓
「英語がなかなか身につかなくて悩んだり、諦めてしまったりしていませんか？」
          ↓
「今すぐ解消するように英語力がつくスクール、それが「ZERO ENGLISH」です！」
```

**機能する理由**:
1. チェックリストがユーザーの内部対話を引き出す（「これ、自分のことだ」）
2. 「諦めてしまったりしていませんか？」で「はい」という共感を獲得
3. 「それが〇〇です！」という宣言でソリューションへの期待感を高める

**GEN-006での重要性**: GEN-006-001〜008では悩み共感セクションがある場合でも「ブリッジ（問題→解決宣言）」が弱かった。このブリッジの明確さがユーザーを次のセクションへ引き込む力になる。

### 参考パターン2: 3段階料金プラン設計（目的別選択）

**構造**:
```
プラン① ¥8,640/月  → 英語で話しながら慣れる（入門）
プラン② ¥12,960/月 → 英語のOUTがスムーズに（中級）
プラン③ ¥15,120/月 → 合格・TOEIC等の試験対策（上級）
```

**機能する理由**:
- 3段階プランは「最安を選ぶ安心感」と「目的に合わせた選択肢」を同時に提供
- 月額¥8,640という最安値の提示で「まず試せる」という低ハードルを演出
- 「どれを選べばいいか」という迷いを目的別ラベルで解消

**GEN-006への教訓**: 料金は「1種類の月額」より「3段階の選択肢」を提示する方が意思決定を促進する。最安プランがアンカーになり全体の価格感覚が下がる。

### 参考パターン3: 「FVに2種類の数字バッジ」信頼性先行設計

**バッジ種別**:
```
スクール生数バッジ: 3,600名突破 → 「多くの人が選んでいる」社会的証明
満足度バッジ: 93%           → 「既存生徒が評価している」品質証明
```

2種類の異なる視点（量と質）の数字をFVに並置することで、「どのくらい使われているか（量）」「どのくらい良いか（質）」の両面を一度に解消する。GEN-006で最も数字設計が洗練されたFV。

## Learning Reason

### 保存理由
GEN-006-009として、GEN-006シリーズ初のBランク達成LP（ZERO ENGLISH）を記録。スクール系FAILの3大要因（料金不明示/FAQ欠如/FVCTAなし）をすべて解消した構成は、今後のスクール系LP制作のベンチマークとなる。特に「悩み共感→解決宣言ブリッジ」と「3段階料金プラン」の組み合わせはスクール系LP設計の標準パターンとして辞典化すべき水準。

### 再利用価値
- 英会話スクール全般（このLPは英会話に特化した悩みリストを持つ）
- 転用: プログラミング / Webデザイン / 資格スクール全般（悩みリストと料金プランを業種別に差し替え）

### 特に優れている点
「GEN-006シリーズで唯一、スクール系特有のFAIL3点（料金不明示・FAQ欠如・最終CTAなし）をすべて解消している」という点。個別の要素の質だけでなく「何も抜け落ちていない構成の完全性」がBランクの本質的な理由。

### 他案件への応用先
- 英会話スクール（そのまま）
- 語学系・コミュニケーション系スクール全般
- 「短期間で成果を出す」系のスクール（プログラミング / デザイン / 営業研修）

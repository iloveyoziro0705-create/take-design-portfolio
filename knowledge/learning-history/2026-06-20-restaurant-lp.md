## ① 構造分析

### セクション構成（実測）
```
1. Hero（FV）
   - フルスクリーン背景画像（Unsplash）
   - ダークオーバーレイ rgba(0,0,0,.45〜.75)
   - 英字ラベル + H1（serif）+ サブコピー
   - 予約CTAボタン

2. コンセプト
   - 横幅広めテキストブロック
   - シェフのこだわりメッセージ

3. メニュー
   - 3カラムグリッド
   - 料理写真 + 料理名 + 説明

4. 空間・雰囲気
   - ギャラリーグリッド

5. アクセス
   - Googleマップ風エリア + テキスト情報

6. 予約CTA
   - ボタン2択（電話 / ネット）
```

## ② 心理学分析

| トリガー | 実装箇所 | 詳細 |
|---------|---------|------|
| 視覚的食欲喚起 | FV・メニュー写真 | 高品質な料理写真が主役 |
| 希少性 | オープン告知 | 「GRAND OPEN」で期待感 |
| 権威性 | シェフメッセージ | 経歴・こだわり |

---

## ③ 導線分析

```
FV（期待感）→ コンセプト（共感）→ メニュー（欲求）→ 
空間（没入）→ アクセス（来店イメージ）→ 予約CTA（行動）

最短化された導線。飲食系は「見て→食べたくなる→予約」が最適。
```

## ④ デザイン分析

### カラーパレット
```css
background: #0f0f0f;          /* ほぼ黒 */
color: #fff;                  /* ホワイトテキスト */
--gold: #d8a76a;              /* 料理系ゴールド */
--title: #f5e6c8;             /* クリーム見出し */
```

### タイポグラフィ
- 見出し: serif（総称フォント）
- H1サイズ: `clamp(46px, 8vw, 92px)`（大胆）

### デザイン特性
- ダーク背景 × ゴールドアクセント = ラグジュアリー飲食の定番
- フルスクリーンヒーローで没入感最大化

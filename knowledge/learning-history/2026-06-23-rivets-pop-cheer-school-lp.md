## ⑧ 参考パターン・知見抽出

### 参考パターン1: 習い事系スクールの「3点差別化」数字構造

**構造**:
```
01: こどもの成長（定性価値 / 指導法の根拠「EBC」）
02: 発表の場 150本以上（活動量 / 機会の多さ）
03: 累計 1万人（社会的証明 / 仲間の多さ）
```

**機能する理由**: 習い事スクールの保護者が持つ3つの関心（「子どもが成長できるか」「発表する場はあるか」「続けられる環境か」）を3点が順番に解消している。「定性価値 → 活動量 → 仲間」の順番が意思決定フローと一致。

**転職型スクールとの対比**:
```
転職型（GEN-006-001）: 転職成功率 / 給料 / カリキュラム
習い事型（GEN-006-007）: 成長 / 発表の場 / 仲間
```
→ スクールの「目的」によって信頼性訴求の軸が大きく変わることをGEN-006で初確認。

### 参考パターン2: 「まずは一度ためしてみよう！」習い事系低摩擦CTA文言

**機能する理由**: 「申し込む」「始める」より「まずは一度ためしてみよう」の方が、コミットメント感が低く、習い事スクールに慎重な保護者の心理的障壁を下げる。「一度」という限定が「嫌なら辞められる」という逃げ道を示唆している。

**IND-006追加候補**: 習い事系・体験型スクール全般のCTA文言として転用可能。

### 参考パターン3: エリア別スクール検索型の展開設計

多校展開スクールの場合、LP内に全スクール情報を詰め込むのではなく「エリア別に探す」で下層ページへ誘導する構造は設計として合理的。ただしLP内に料金や体験の費用が全く表示されない問題は別途解消が必要。

## Learning Reason

### 保存理由
GEN-006-007として、チアリーディングスクール（習い事型）の初事例を記録。「キャリア転職型」と「習い事・情緒価値型」でスクールLPの設計軸が大きく異なることをGEN-006で初めて確認した。特に信頼性訴求の軸（転職率 vs 成長・発表機会・仲間）の違いはIND-006サブカテゴリ設計に反映すべき重要知見。

### 再利用価値
- 子ども向け習い事スクール全般（ダンス / 体操 / 音楽 / 絵画 / プログラミング（子ども向け））
- 多校展開型スクール（エリア別スクール検索設計の参考）

### 特に優れている点
「発表の場150本以上」という数字は習い事スクールとして独自の差別化軸。「学ぶだけでなく活躍の場がある」という訴求はスポーツ・パフォーマンス系スクール全般で転用できる。

### 他案件への応用先
- バレエ / ダンス / 体操 / 武道スクール（「大会出場〇回以上」等の発表・活躍機会訴求）

## ★「お客様を想う心が身に付く仕事。」— スキル獲得型FVコピー

### GEN-002初確認: 「この仕事で〇〇が身に付く」という成長訴求型

```
「お客様を想う心が身に付く仕事。」
```

GEN-002で確認したFVコピーの類型にこれまでなかった「スキル獲得型」:

| 類型 | 例 | 訴求軸 |
|-----|---|------|
| 内発動機型 | 「好きを 仕事に。」（LAVA）| 好き・情熱 |
| ビジョン対称型 | 「ひらけ、世界。ひらけ、自分。」（日本生命）| 世界と自己成長 |
| 仕事哲学型 | 「仕入れの極意」（八百鮮）| 職人的奥深さ |
| **スキル獲得型** | **「お客様を想う心が身に付く仕事。」（吉帆楼）** | **仕事を通じた人間的成長** |

**「心が身に付く」の表現分析**:

| 要素 | 内容 |
|-----|------|
| 「お客様を想う心」 | 高級和食・おもてなし業界の最高価値「心」を訴求軸に選択 |
| 「身に付く」 | 技術ではなく「人格・人間性の成長」への訴求。長期の変化を示唆 |
| 「仕事。」（句点で断言）| 「この仕事だから身に付く」という断言。他の仕事との差別化 |

「技術が身に付く」ではなく「**心が身に付く**」という表現は、高級和食・ホスピタリティ業界特有の  
「技術より精神性・おもてなしの心」を価値の核心に据えた訴求。

応用テンプレート:
```
「〇〇の心が身に付く仕事。」（おもてなし・接客業全般）
「プロとしての技が身に付く仕事。」（職人系）
「この仕事で、人として大きくなれる。」（成長訴求の展開形）
```

## ⚠️ テクニカル問題: 2020年旧採用バナーの残存

```
「新卒採用情報 2020年3月卒業予定の方の募集を開始しました 募集要項はこちら→」
```

2026年6月時点でこのバナーが残存しているならば、**6年前の採用情報**が表示されている状態。

**問題の深刻度**:

| 問題 | 影響 |
|-----|------|
| 「2020年3月卒」という古いターゲット表示 | 「現在は採用していない」という印象 |
| 更新管理の放棄サイン | 「この会社は今も採用活動をしているか」という疑問 |
| 採用LPとしての信頼性損失 | 最初の情報で「古い」と感じると応募意欲が下がる |

飲食系OINOSのNewsStale問題（2013年 = 13年放置）と同様の構造的問題。  
ただし採用サイト全体の設計は優れており、バナーの更新だけで大幅にスコア改善できる。

**テクニカル改善 → 予測スコアアップ**: バナー削除/更新 → +3点（合計56点/C+に上昇）

## 問題点

### 問題1: LP内にスタッフの声が不在

「先輩インタビュー」ページへのリンクはあるが、  
LP内（1ページ内）に顔写真+実名+入社前後の変化が表示されていない。  
高級和食店の採用において「どんな先輩と働くか」「未経験でも育ててもらえるか」は  
応募決定の最大要因。スタッフの声をLP内に3名分配置するだけで+7〜10点。

### 問題2: 給与の具体額不在

「働くメリット」ページへの誘導はあるが、LP内で給与の具体額が見えない。  
「高級和食 = 修行期間は薄給」というイメージがある飲食・料理業界では、  
「安定生活にすることで生涯役立つスキル」という表現よりも  
「月収〇〇万円〜（経験・資格不問）」という数字の方が応募意欲を高める。

# CV改善辞典

## 概要
コンバージョン率（CVR）を改善するための手法・心理学・実装パターンを体系化する。

---

## Learning Reason（各CVパターン追加時に必須記録）

新規CV改善パターンを追加する際は、パターン末尾に以下を記録すること。

```markdown
## Learning Reason
### 保存理由
[なぜこの心理学・CV手法をナレッジ化したか（効果の根拠・発見経緯）]
### 再利用価値
[この手法が有効な業種・LP場面・ターゲット層の範囲]
### 特に優れている点
[心理的効果・CVR改善幅・実装容易性の観点で優れている点]
### 他案件への応用先
[適用推奨の業種・セクション・CVアクションタイプ]
```

## 心理学トリガー一覧

### PSY-001: 社会的証明（Social Proof）

**定義**: 他者の行動・評価を見て、自分も同じ行動を取ろうとする心理

**実装パターン**
```html
<!-- 数字型 -->
<p>累計 <strong>1,500名</strong> が体験済み</p>

<!-- 星評価型 -->
<div class="rating">★★★★★ <span>4.9 (口コミ128件)</span></div>

<!-- ロゴ型（メディア掲載） -->
<div class="media-logos">
  <img src="nikkei.svg" alt="日経新聞掲載">
  <!-- 複数ロゴ -->
</div>

<!-- SNS型 -->
<p>Instagramフォロワー <strong>2.1万人</strong></p>
```

**使用場所**: Hero下・口コミセクション・フッター近く  
**効果**: 不安解消・比較検討の短縮

### PSY-002: 希少性（Scarcity）

**定義**: 手に入りにくいものに高い価値を感じる心理

**実装パターン**
```html
<!-- 残り枠型 -->
<p class="scarcity">⚡ 今月の無料相談枠 残り<strong>3名</strong></p>

<!-- 期間限定型 -->
<p class="deadline">🎁 6月30日まで 初回30%OFF</p>

<!-- カウントダウン型 -->
<div class="countdown" id="timer">23:14:05</div>
```

**注意**: 虚偽の希少性は信頼を失う。実際の限定性のみ使用する。

### PSY-003: 権威性（Authority）

**定義**: 専門家・権威者の意見を信頼する心理

**実装パターン**

```html
</div>

<!-- 受賞・掲載 -->
<p>〇〇アワード2024 最優秀賞受賞</p>
```

### PSY-004: 損失回避（Loss Aversion）

**定義**: 得るより失う痛みを2倍強く感じる心理

**実装パターン**
```html
<!-- 損失提示型 -->
<h2>この3つに当てはまる方は<br>毎月<strong>〇万円</strong>損しています</h2>

<!-- 機会損失型 -->
<p>今のまま続けると、5年後には差が<strong>〇〇万円</strong>に</p>

<!-- 比較型 -->
<div class="compare">
  <div class="no-service">サービスなし → ○○○のリスク</div>
  <div class="with-service">サービスあり → ○○○の安心</div>
</div>
```

### PSY-005: 返報性（Reciprocity）

**定義**: 何かをもらったらお返しをしたくなる心理

**実装パターン**
```html
<!-- 無料コンテンツ配布 -->
<div class="freebie">
  📖 今すぐDL無料 「〇〇完全ガイド」
</div>

<!-- 無料相談 -->
<p>まずは無料でご相談ください。費用は一切かかりません。</p>

<!-- 無料体験 -->
<p>7日間 無料でお試し。クレジットカード不要。</p>
```

### PSY-006: コミットメント一貫性（Commitment & Consistency）

**定義**: 一度行動すると、それに一貫した行動を取り続けたくなる心理

**実装パターン**
```html
<!-- 小さなコミットメント型 -->
<p>まずは「見学だけ」でもOKです →</p>

<!-- チェックリスト型（自己確認） -->
<p>以下に当てはまる方は、ぜひご相談ください</p>
<ul>
  <li>✓ 現在の収入に不安を感じている</li>
  <li>✓ 老後の資金が心配</li>
</ul>
<!-- チェックを入れた瞬間に「自分ごと」になる -->
```

### PSY-007: アンカリング（Anchoring）

**定義**: 最初に見た数字が基準（アンカー）になる心理

**実装パターン**
```html
<!-- 価格比較型 -->
<div class="price-anchor">
  <span class="before">通常 ¥30,000</span>
  <span class="after">今なら ¥18,000</span>
  <span class="save">40% OFF</span>
</div>

<!-- 比較表型（高い選択肢を先に見せる） -->
<table>
  <tr><th>プレミアム ¥50,000</th></tr>
  <tr><th>スタンダード ¥30,000（おすすめ）</th></tr>
  <tr><th>ライト ¥10,000</th></tr>
</table>
```

## CV改善チェックリスト

### FV（ファーストビュー）
- [ ] FVだけでサービス内容が伝わる
- [ ] CTAボタンがスクロールなしで見える
- [ ] キャッチコピーにベネフィットが含まれる
- [ ] 視覚的に「信頼できる」と感じる要素がある

### CTA設計
- [ ] ボタンテキストが「行動後のベネフィット」を伝えている
- [ ] 「無料」「〇秒」等のハードル下げ表現がある
- [ ] ボタンの色がページ内で目立つ（コントラスト比 4.5:1 以上）
- [ ] スマホで固定フッターCTAが設置されている

### 信頼性
- [ ] 実績数字が3個以上表示されている
- [ ] お客様の声に「名前・年代・地域」が含まれる
- [ ] スタッフ顔写真または会社概要がある
- [ ] プライバシーポリシーリンクがある

### 摩擦の除去
- [ ] 応募フォームの項目が5個以下
- [ ] 「しつこい営業なし」等の不安解消文がある
- [ ] FAQ で上位5つの不安を先回り解消している
- [ ] 決済・個人情報セキュリティの表示がある

### モバイル最適化
- [ ] スマホでのタップ領域が44px以上
- [ ] フォントサイズが16px以上
- [ ] 横スクロールが発生しない
- [ ] LCP（最大コンテンツ描画）が2.5秒以内

## 離脱防止パターン

### EXIT-001: スクロール深度によるCTA変化
```javascript
window.addEventListener('scroll', () => {
  const scrolled = window.scrollY / (document.body.scrollHeight - window.innerHeight);
  if (scrolled > 0.7) {
    document.querySelector('.cta-final').classList.add('highlight');
  }
});
```

### EXIT-002: 離脱意図検知（デスクトップ）
```javascript
document.addEventListener('mouseleave', (e) => {
  if (e.clientY <= 0 && !sessionStorage.getItem('exitShown')) {
    sessionStorage.setItem('exitShown', 'true');
    document.querySelector('.exit-popup').classList.add('visible');
  }
});
```
